ToolsGroup JustEnough Functionality Comparison Chart

The following functionality comparison chart includes versions 2022.1 to 2024.1 inclusive.

Microsoft Installer (MSI)

Analysis Installer

ToolsGroup JustEnough v2022.3 introduced a new Analysis Installer. This new installer is required to install the Analysis API and is required to be run on its own compute node. These compute nodes can be configured remotely using dialog windows in which users configure the audience, instance name, and the base application server address. This removes the need to follow a manual configuration process for each compute node reducing the chances of errors.

This installer is for customers who will be taking advantage of ToolsGroup JustEnough’s machine learning capabilities. For more information, see the Forecasting section.

MSI enhancement

A new dialog window has been added to all 4 MSI (Services, Compute Endpoint, Client, and the new Analysis) that takes in the required configured parameters such as the Authority URL. This new window displays when the installer is run and reduces the need for manual edits and the chances of misconfigurations.

Wholesale Planning (v2023.4)

As the presence of online marketplaces expands, traditional retailers are identifying a need to create plans for these non-traditional sites. This problem statement could begin with a retailer allocating stock (and sales expectations) to be fulfilled by Amazon, or Walmart, or the like; or it could expand into a broader "wholesale" strategy. In order to support this growing need for retailers to plan in the wholesale space, the traditional TG JE retail functionality has been enhanced for v2023.4 to better support wholesale functions.

The new functionality relating to wholesale planning impacts multiple modules. For convenience and ease of understanding, the wholesale planning content can be selected using the link from any impacted module or selecting the following: Wholesale Planning
