Rounding rules

Rounding rules can be configured using two views:

  • Rounding Rules - Price Ending

  • Rounding Rules - Standard

Rounding rules allow users to create grouped rounding criteria which are applied to a promotion calculation to conform to corporate price rounding standards. rounding criteria determine which prices a rule should act upon and what should be done to the price. The Promotion module can be configured with both standard rounding as well as rounding with price endings.

Price ending rounding refers to the selection of a numeric position and rounding it to the configured price ending value based on the preceding numeric position value.

Create a rounding rule – standard or price ending

Within the relevant Rounding Rule tab, click New to create a new rounding rule, enter the required fields and click Save.

Standard tab: fields and descriptions

Field Description
Name A descriptive label summarizing the purpose of the rounding rule. This is used to identify the rounding rule when creating promotion calculations.
Description An optional summary of the rounding rule.
Active  (Optional) Leave selected to indicate that this rounding rule is active and is available when configuring promotion calculations.

Modifying rounding rule or rounding rule criteria

For both standard and price ending rounding rules, modifications can be made directly to the Rounding Rule and Rounding Rule Criteria grids.

Deleting rounding rule or rounding rule criteria

For both standard and price ending rounding rules, selecting a rounding rule or rounding rule criteria enables the Delete button above the applicable grid.

Rounding rules cannot be deleted if associated to a Calculation and once deleted cannot be restored.

Create rounding criteria – price ending

With the rounding rule selected, click New above the Rounding Criteria grid to add criteria to the rounding rule. Enter the required fields and click Save above the grid.

Field Description
Currency The available currencies specified by each site. Select a currency for the criteria configuration which applies to the applicable sites when price actions are calculated.
Price Ending The right most decimal value searched for in price fields.
Position Select the numeric position that is modified based on the criteria.
Direction (Optional) Whether the calculated value rounds up, down or to the closest based on the calculated mid-point of the price ending configured value.
  • Closest - The configured position determines whether the calculated value rounds up or down to the desired price ending.
  • Down - The configured position and calculated mid-point are evaluated and the calculated value rounds down to the desired price ending.
  • Up - The configured position and calculated mid-point are evaluated and the calculated value rounds up to the desired price ending.
Active (Optional) Leave selected to indicate that the rounding criteria is active within the rounding rule.

Create rounding criteria – standard

With the rounding rule selected, click New above the Rounding Criteria grid to add criteria to the rounding rule. Enter the required fields and click Save above the grid.

Field Description
Currency The available currencies specified by each site. Select a currency for the criteria configuration which applies to the applicable sites when price actions are calculated.
Position The numeric position to be modified based on the criteria.
Mid Point The decimal value that is considered the mid-point for the rounding criteria. If the numeric value in the position is smaller than the mid-point, no changes are made. However, if the numeric value in the position is equal to or greater than the mid-point, the numeric value rounds up.
Active (Optional) Leave selected to indicate that the rounding criteria is active within the rounding rule.
