Bidding for events

Bids for Events are configured using the Bids tab which lists all Events that the Promotion could bid for. Only Events with sale start and end dates that overlap with the Promotion start and end dates display.

The Bid Status column indicates if the promotion already has a bid has for the event. The possible statuses are:

  • No Bid – The Promotion has not made a bid for the event.
  • Submitted – A bid has been submitted.
  • Re-Submitted – A bid has been resubmitted for the event.

To place a bid

  1. Select the event.
  2. Click the Edit button.
  3. The Edit Bid Details dialog displays.

  4. Click the Preferred Bid Event Spaces button.

    The Select Preferred Spaces dialog displays.

  5. Select the preferred spaces and click Ok.

  6. Complete the fields as required in the Edit Bid Details window.

  7. Click Ok.

Field descriptions

Field Description
Preferred Bid Event Spaces (Optional) The preferred spaces.
Bid Action (Required) Select an action from the dropdown list:
  • Submitted – Indicates a new bid.
  • Re-Submitted – Indicates a previously placed bid has been updated.
  • Cancelled – Indicates that the bid is no longer active.
Financial Forecast (Optional) The projected financial sales for the bid.
Unit Forecast (Optional) The projected units sold for the bid.
Vendor Supported (Required) If the Vendor is providing support for the bid.
Comments (Optional) Any supporting information to help ensure the bid is successful.
