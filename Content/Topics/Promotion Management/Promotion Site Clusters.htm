Sites with more than one currency used in a promotion

If a site cluster containing sites with more than one currency is used in a promotion, the site cluster is split to display an instance of the site cluster for each currency. For example:

  • Site Cluster A consists of Site Alpha (USD), Site Beta (USD), Site Gamma (CAD), and Site Delta (USD).
  • Site Cluster A is added to a site cluster set which is used in a promotion.
  • The top area of the promotion displays rows for Site Cluster A in the list of promotion site clusters: A (USD) and A (CAD), each with their respective sites that match the currency appended to the site cluster name.

Promotion strategies can be modified at the site cluster level.

  1. Select the site cluster to be updated.

  2. Modify the strategy as needed.

    The promotion saves automatically.
