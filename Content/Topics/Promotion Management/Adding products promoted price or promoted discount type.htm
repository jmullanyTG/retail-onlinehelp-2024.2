Adding products to Promoted price or promoted discount promotion strategies

For Promoted Price and Promoted Discount type Promotion Strategies there are three ways to add products. They are not mutually exclusive and all three methods can be used within a promotion.

Once all the products have been added to the promotion, they display in a group called Item Group below the Promotion Group.

Enter default pricing information for all products and all site clusters in the fields for Promoted Price and Promoted discount type strategies in the left pane. The information to be entered differs based on the promotion strategy selected and can be modified further at the site cluster level after the new promotion is saved.

Adding a variant

  1. Click Add Variant.

  2. In the Add Variants dialog, enter a product code (for Variant or Item Master, depending on the global policy) in the Products field or select the Add Variants From Clipboard button.

     Variant codes can be copied using the right-click menu option Copy Column Cell(s).

  3. Click Ok.

Using a filter to add products

  1. Click Select to Add.

  2. In the Variants dialog, click the dropdown arrow in the Filter field and select a filter.

    If an appropriate filter does not exist click, click on the black hammer icon to create a new filter.

  3. If required, select the checkbox Select only some items from the current filter for individual product selection from the selected filter.
  4. Once the appropriate Filter is selected, click Ok.

Adding a variant group

  1. Click Add Variant Group.
  2. If required, click Filter to change which filter is being used.
  3. In the Select Variant Groups to Add dialog, select the checkbox for each group to be added to the Promotion.
  4. Click Ok.

Variant group notes

  • If the group to add is not available, first ensure the filter selected in the pop-up is for all Variants or Item Masters. If still unavailable, navigate to Content Manager to create the group required for the promotion.
  • These groups are dynamic meaning the associated products included in the group are automatically updated depending on the filter criteria used to create the group. This update only occurs as long as the promotion is in an unlocked state.
  • When adding a group to a promotion, a copy of the original group is automatically created, including the filter criteria used to define the group. Changes can be made at the promotion level without affecting the originating product group.

Additional information
