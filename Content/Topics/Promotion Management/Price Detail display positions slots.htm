Price detail – display positions (slots)

The Price Detail – Display Positions (Slots) tab on the Price Details tab is used to assign Promotions directly into an open Slot in an Event Slot Set. Ensure the Parameters button is enabled. To load all available Slots, select All from the Available Events View Option dropdown menu and click Execute.

Event Slots that do not display signifies that either a different Promotion has been placed the Slot or the user does not have access to the Space that the Slot is in. A Slot can be displayed across more than one row if the Promotion in the Slot is not associated with all Geographies in the Event.

The Available Events View Option dropdown list determines which accessible Slots from overlapping Events display in the grid. Select one of the options below and click Execute to alter the Slots displayed in the grid:

  • All – Show Slots that are available or assigned to the current Promotion.
  • Available – Show Slots that do not have Promotions assigned to all Event Geographies.
  • Assigned – Show Slots that the current Promotion has been assigned to.

The grid displays a row for every Slot contained in an Event that has overlapping Sale Dates and some Sites in common with the current Promotion.

A check mark in the Assigned column signifies that the current Promotion has already been placed in that Slot.

Rows that are formatted red are read only meaning that the user can't change that Promotion in the event for that row.

Assigning Promotions to available Event Slots

To assign the Promotion to an available Event Slot, select the Assigned column in the applicable row in the grid, click Save and click Ok in the Event Slot Links dialog.

Once a Promotion has been assigned to a Slot, the pane on the right displays all Products and Product Groups associated with the Promotion.

Selecting the Featured checkbox beside one Product (Item or Group) applies the selected Product’s Master Content and the Pricing to the Event’s Slot.

Click Save above the right pane to apply the Content and Pricing selections to the Event Slot. The ability to push a specific Product’s Content and pricing is not available for BOGO AB type Promotions.
