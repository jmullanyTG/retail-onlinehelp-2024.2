Calculation sets

Calculation sets are a grouping of calculations that are performed in a specified order. Calculation sets can be assigned to a promotion strategy.

For more information, see Promotion Strategy.

Creating calculation sets

Within the Calculations tab, click New on the Calculation Sets toolbar to create a new calculation set. Enter a name and description and click Save.

Field Description
Name A descriptive label to summarize the purpose of the Calculation Set. This identifies the calculation set when configuring promotion strategies.
Description An optional summary of the calculation set.

Modifying calculation sets

Modifications are made directly in the Calculation Sets grid. Click Save to save modifications. Click Cancel to discard modifications. Existing promotions referencing the calculation set must either be recalculated manually or by an overnight process.

Modifications can also be made to the Calculation Set Calculations grid. For more information, see Calculations assigned to a calculation set .

Cloning calculation sets

Within the Calculation Sets grid,

  1. Select the calculation set to be cloned.

  2. Click the Clone button on the toolbar.

    A new calculation set is added to the Calculation Sets grid with the suffix (Clone).

  3. Modify the copy as desired.

  4. Click Save.

Deleting calculation sets

Click the Delete button in the Calculation Sets section of the top toolbar. Calculation sets cannot be deleted if associated to promotions not already exported and in that state.

Assigning calculations to calculation sets

Within the Calculations tab, select a Calculation Set and then click Add in the Calculation Set Calculations section of the toolbar. Complete the required fields and click Save to assign the Calculation to the Calculation Set.

Field Description
Order A numeric value identifying the sequence order for the new calculation being added.
Calculation A dropdown list populated with calculations defined in the Calculation Repository grid.
Description (read-only) Defaults to the calculation description after changes are applied.
Active Selected by default and indicates that the calculation is active in the calculation set.

Modifying assigned calculations

Modifications are made directly to the Calculation Set Calculations grid. Existing promotions referencing the calculation set must either be recalculated manually or by an overnight process.

Calculations can be temporarily removed from a calculation set by unchecking the Active checkbox.

Deleting assigned calculations

Click the Remove button and select Yes to confirm. Existing promotions referencing the calculation set must either be recalculated manually or by an overnight process.
