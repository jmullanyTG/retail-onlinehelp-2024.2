Creating promotions

New promotions are created using the Create New Promotion wizard. The wizard is comprised of fields to enter values at the top, a display grid on the left that is dynamically generated based on the selected promotion strategy, and a products grid on the right that is available after a promotion strategy is selected.

A second customized tab displays in the products grid area if global policy New Promotion View Collection Set is set.

Promotion strategies available for selection

Roles and users only see those Promotion Strategies for which they have access. This is to ensure compliance with corporate promotional pricing for all advertising channels.

Creating a new promotion

  1. Click New to display the Create New Promotion wizard.
  2. Complete the required fields.

  3. Selected the desired privacy option (Private, Public, Shared).

    For shared promotions, click the horizontal ellipse button to select the users and roles to share the promotion with.

  4. Click Ok.
Field Description
Promotion Name The name of the promotion.
Start Date Modify the default date/time to the promotion start date/time. If the promotion is being created from an Event, the Event Ad Start Date is used and the field is restricted from modification.
End Date Modify the default date/time to the promotion end date/time. If the promotion is being created from an Event, the Event Ad End Date and the field is restricted from modification.
Workflow Modify the default Workflow if desired for the promotion.
Price Cluster Set Select the appropriate Site Cluster Set for the promotion. The Site Cluster Set loaded to the promotion is determined based on the Site Cluster Set start date and promotion Start Date. The promotion can be customized for each Site Cluster in the Promotion Detail screen.
Promotion Strategy (Optional) Select a default promotion strategy for the promotion from the dropdown list. Populated with available promotion strategies defined in the Promotion Configuration Promotion Strategy area. When modifying the promotion, the promotion strategy can be changed for one or more Site Clusters.
Private, Public, Shared The default is Public. Change the default to designate certain promotions as sensitive and restrict the ability for other users to view the promotion. Selecting Shared restricts viewing the promotion to certain users and roles.

Display grid fields

The display grid on the left pane shows fields that have been configured for the selected promotion strategy. If an editable field is available, this allows for a promotion level value to apply across all site clusters. This can be customized at the site cluster level in the Promotion Detail screen.

Products (optional)

If a promotion strategy has been specified, products may be added to the promotion. These products apply to all site clusters in the promotion. They may be customized per site cluster in the Promotion Detail screen. How products are added differs depending on the type of strategy selected.

Once all of the required information has been entered in the promotion, click Ok. The Promotional Detail screen displays allowing further detailed modification of the promotion.
