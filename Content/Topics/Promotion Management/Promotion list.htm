List tab

The List tab provides a filtered list of promotions. Depending on assigned permissions users can view, edit, and create new promotions. In the grid, the first checkbox column allows for selecting multiple promotions for deletion.

Promotions marked as Private to other users do not display. The promotions that do display are those marked Public and those promotions shared with the logged-in user or the roles that the logged-in user belongs to.

Filtering promotions

To filter the list of promotions, click Filter. Click the dropdown arrow to select an existing filter or click on the black hammer icon to create a custom filter.

Opening Promotions

Double-clicking a promotion in the grid displays the promotion as a separate tab.

The available strategies displayed are filtered based on the promotion strategy’s restricted user and role access.

If a previously selected promotion strategy is no longer valid, then the now restricted strategy is retained. However, once the user selects a different promotion strategy the retained promotion strategy is no longer available.

On the promotion details screen, there are several buttons and icons available.

List grid field descriptions

Columns include:

Column Description
Lifecycle Indicates if the promotion occurred in the past, future, or is currently being executed. The field displays:
  • PAST - The current system date is after the sale end date time.
  • LIVE - The current system date is between the sale start date time and the sale end date time.
  • FUTURE - The current system date is before the sale start date time.

This column does not display if system policy Allow Multiple Dates is set to True.
Price Cluster Set Name The Site Cluster Set Name that was used during creation. Modification of the Price Clusters and Cluster Sites after creation does not modify this value.
Price Cluster Set ID The Site Cluster Set ID that was used during creation. This is the actual Site Cluster Set, not its Snapshot ID.

Toolbar functionality

Action Description
Save Saves the current promotion.
Save As  See, Creating promotions based on existing promotions.
Clone
Recalculate (Available if changes have been made to the promotion) Refreshes all promotion calculations for all Site Clusters.

  • Change Status
  • Change Multiple Statuses
Price Clusters Manages the Promotion Site Clusters.
Sites Manages the sites in the selected Promotion Site Cluster. The total number of sites where the promotion applies displays next to the Details button.

For more information, see Sync Promotion screen.
Details Displays detailed editable promotion information.
Date Range The promotion’s start and end dates.
Status The icon is green if there is a message available for the promotion at the current workflow status.
