Change promotion status

Workflows determine the number of statuses, what transitions are available at each status, and what access users have at each status. The Change Status button displays if users have permission to change a promotion's status.

Promotion workflows are setup in Workflow Manager.

Click Change Status to display a dialog to change the current status to one of the available statuses for this transition.

Field Description
Status A dropdown list of all available statuses to select from.
Message An optional area to enter a message for the next user to review. A green message icon displays beside a status if a message has been entered.
