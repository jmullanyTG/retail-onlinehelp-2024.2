Images

By clicking the Images tab in the Content Details window, images associated with the Promotion Content Version can be viewed, new image associations can be made or requested as well as removed from future use.

Associating view types

Each new image added should be associated with a View Type so that it displays properly in an Item Display Format. This association can be made directly in the Images grid by selecting the View Type dropdown and choosing a value for each image. Each View Type can only be associated to one image.

Toolbar Functionality

The three methods of adding images are not mutually exclusive and all three methods can be used within a promotion.

Action Description
Save It is beneficial to save changes if a several images have been newly associated to a promotion. If the promotion has been placed on a future Event or the product in the promotion has Master Content, the Content Update window displays providing the ability to update images for the other instances as well. For more information, see Content Update window.
Add Existing Opens the Repository Explorer window allowing selection of an existing image.
Add Opens the local computer’s Explorer window for navigation to the image on the computer or network.
Remove Selecting images and clicking this button removes the selected images.
Remove All Removes all images for the Content Version.
View Clicking View or double-clicking an image displays a larger preview of the image in a window.

Attributes Displays the Attributes window for the selected image. If more than one image is selected, attributes display for the last selected image in the grid.

Image Attributes are not available for new image requests.
Image Request: New If the required image doesn’t exist, it can be requested by clicking this button. A form can be completed to initiate a request. Click Ok to save the request.

Image Request: Edit The existing new image request opens to be modified as needed. Click Ok to save the updates.
