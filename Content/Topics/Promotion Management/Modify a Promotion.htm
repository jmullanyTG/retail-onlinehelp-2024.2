Modifying promotions

New promotions when saved and existing promotions when opened display in a separate tab. With the appropriate rights, all aspects of a promotion can be modified.

The Promotion screen is divided into two main sections.

The upper portion displays the site clusters with their assigned strategies allowing promotions to be tailored for specific sets of sites as needed. It also includes functionality to modify the sites assigned to the site clusters for the promotion.

The lower portion displays the selected site cluster’s products, pricing details, bids (if configured), companion items (if configured), and analysis. The promotional products and pricing details can be modified to be specific for each site cluster.

Promotional pricing for a selected site cluster is viewed and edited in the left pane on the Price Cluster Items and Price Details tabs. When applied, changes affect the pricing data in the right pane of the Price Details tab.

Click the Details button on the top right of the upper portion to display the Promotion detail pane. All required fields are completed during promotion creation. However, depending on permissions and workflow status they can be modified. There are also several additional fields that can be completed. Use this pane to review and modify a promotion’s details including workflow and privacy setting.

For example, a promotion with a status of Private can be modified to a status of Public so that other users can see it after its information is no longer considered sensitive.

The ability to modify a promotion’s privacy configuration is based on the ability to edit the promotion (rights for a promotion's status and validation state in conjunction with blacklisting).

Field Description
Promotion Name The name of the promotion.
Date Range The start and end dates of the promotion.
Description A summary of the promotion.
Priority Select a Priority value from the configurable dropdown list.
Effect Select an Effect value from the configurable dropdown list.
Tags Click Tags to associate tags with the promotion. New tags can be created with the appropriate rights.
Requirements Click on … and select one or more requirements in the window.
Price Cluster Set The site cluster set used to set up the promotion initially. This can be modified in the promotion.
Workflow The promotion workflow.
Created By Who created the promotion and the creation date and time.
Last Changed By Who last modified the promotion and the date and time when that change was made.
