Adding or removing site clusters

Site clusters can be created or deleted from promotions.

To create site clusters:

  1. Click on New.
  2. Enter the name, select a strategy, and select the currency.
  3. Click Ok.
  4. Add sites to the new site cluster.

To delete site clusters:

  1. Select the site cluster.
  2. Click on Delete.
  3. Click Yes.

Any sites assigned to the deleted site cluster are removed from the promotion.
