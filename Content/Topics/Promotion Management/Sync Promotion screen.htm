Syncing promotions

Demand forecasting accuracy can be improved by defining a planned lift factor by cluster set and having those values pushed to the Causals import API. This is achieved by selecting the button labeled Sync located on the toolbar. Selecting this button displays the Sync Promotion dialog in which the override lift value as a percentage can be entered.

Right-click in the Override Lift column and choose the option labeled Multiple Changes to update multiple rows simultaneously.

The user right Allow access to Sync (Right Group: Promotions) is required to access this functionality.
