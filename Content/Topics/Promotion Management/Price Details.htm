Entering price exceptions for site clusters

The Price Details tab provides the ability to enter any pricing exceptions for the Site Cluster as well as provides the ability to add the Promotion directly to an Event Slot. The ability to add a Promotion to a Slot is only supported for implementations using Item Master products not Variants.

After selecting a Site Cluster from the Promotion Header, the lower left pane displays promotion pricing details that apply to all products associated with the Site Cluster.

The lower right pane displays the Price Detail tab and the Price Detail – Display Position (Slots) tab.
