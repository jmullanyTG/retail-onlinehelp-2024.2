Creating an export configuration for Promotion export

After an export process is defined, create an export configuration to define the export columns to be included in the Export Columns grid. The columns configured are associated to the specific row selected in the Export Definition grid.

For more information on defining an export process, see Defining an export process for Promotion export.

To create an export configuration:

  1. Click New in the Export Columns grid.

  2. Fill in the required columns.

  3. Click Save.

Export Columns field definitions

Field Description
Level Select the level of data to export from the dropdown list (e.g. Group, Item, etc.)
Column The list refreshes based on the selection in the Level column. Select the column to be exported.
Include In Export (Optional) Select this column to indicate that the column should be exported.

To indicate an Export Column is temporarily inactive, clear the Include in Export column and click Save.
Description (Optional) Enter a summary of the data to be exported.

For Promotions exports, Header columns 'ExportBatchID', 'PromotionID', 'PriceClusterID', 'PromotionName', 'PromotionStartDate' and 'PromotionEndDate' are always included.
