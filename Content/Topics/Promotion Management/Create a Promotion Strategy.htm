Creating and managing promotion strategies

Promotion strategies are created using the Strategies tab in the Promotion Configuration screen.

Removing access to a new promotion strategy

By default, all roles and users have access to all new promotion strategies. To maintain corporate compliance across advertising channels, click on the Edit Users field to restrict access to roles and users as required. This does not affect any current promotions. Once roles and users have been selected, click Save.

Only active roles and users that have the User Right Allow Creation of Promotion display in the Remove User Rights dialog.

Creating promotion strategies

  1. Click New to add a blank row to the Strategies grid.

  2. Complete the fields as required.
  3. Click Edit Users to define which Roles and Users have access.
  4. Select the checkboxes as required in the bottom grid.
  5. Click Save.

Modifying promotion strategies

Modifications can be made directly to the grids.

Existing promotions referencing the promotion strategy must either be recalculated manually or by an overnight process.

Deleting promotion strategies

Promotion strategies cannot be permanently deleted from the system. However, they can be inactivated. Selecting a promotion strategy and unchecking the Active checkbox makes the strategy unavailable in the system.

Promotion Strategy Cluster Inputs field descriptions

Field Description
Field Names of all available pricing fields.
Is Buy Whether the field will be used as a Buy field. For each of the following Promotion Strategy types, select Is Buy for pricing and limit fields:
  • Promoted Price
  • Promoted Discount
  • Spend Get
  • Rebate
  • Buy Get:
    • for BOGO AA, select Is Buy for pricing and limit fields, and
    • for BOGO AB, select Is Buy for all buy pricing and limit fields.
Is Get Whether the field will be used as a Get field. For each of the following Promotion Strategy types, select Is Get for pricing and limit fields:
  • Promoted Price
  • Promoted Discount
  • Spend Get
  • Rebate
  • Buy Get:
    • for BOGO AA, select Is Get for pricing and limit fields, and
    • for BOGO AB, select Is Get for all get pricing and limit fields.
Is Visible Whether the field displays in the display grid on the left of a New Promotion Wizard and Promotion.
Is Mandatory Whether the field must have a value before the Promotion can be saved.
