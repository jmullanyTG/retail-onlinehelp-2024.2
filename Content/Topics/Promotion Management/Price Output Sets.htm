Price Output Sets

The Price Output Sets screen is used to add or remove fields, choose whether the fields are write-enabled, and at what level. To access this screen, click Configuration in the Promotion screen.

Promotion pricing details should only be modified with the assistance of ToolsGroup JustEnough Support or a Professional Service resource.

Creating price output sets

Click New in the Price Output Sets section of the toolbar to create a new price output set. Complete the required fields and click Save.

Modifying price output sets

Modifications are made directly to the Price Output Sets grid.

Deleting price output sets

  1. Select a price output set.
  2. Click Delete on the toolbar.

  3. Select Yes to confirm.

Field descriptions

Field Description
Name A descriptive label for the promotion pricing detail. This identifies the price output set when creating promotion strategies.
Description A optional summary of the price output set.

Assigning fields to a Price Output Set

Fields must be assigned to a price output set using the Price Output Set Fields grid.

  1. Click Add above the Price Output Set Fields grid to add a new line.
  2. Complete the required fields.
  3. Click Save.

Modifying assigned fields

Modifications are made directly to the Price Output Set Fields grid. Click Save to save modifications. Existing promotions referencing the price output set must either be recalculated manually or by an overnight process.

Deleting assigned fields

Select a price output set and click the Remove button. Existing promotions must either be recalculated manually or by an overnight process.
