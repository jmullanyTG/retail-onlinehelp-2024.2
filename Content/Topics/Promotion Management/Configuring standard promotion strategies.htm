Configuring standard promotion strategies

Users with the relevant User Rights, can click the Configuration button and use the following table to configure the standard promotion strategies.

The promotion strategy configuration may vary from these defaults due to implementation requirements.

  • Price Output Set – EDLP
  • Price View Collection Set – Price Detail – EDLP/Regular Retail

  • Is Group Synced – No

  • RetailHigh – Is Get, Is Visible
