Text

Content in the text fields of the Text tab can be updated as desired.

Button Description
Save As copy text is modified, it is beneficial to save changes if there are a large number of fields being modified or if changes are being made over an extended time frame.

If the promotion has been placed on a future Event or the product in the promotion has Master Content, the Content Update window displays to provide the ability to update text for the other instances as well. For more information, see Content Update window.
Special Character Map Launches the local computer’s Character Map window to allow copying and pasting of special characters into the text fields.
Content Link Used to identify one product from the promotion to use as the master source for text and images.

  • Only one product can be selected as the master source for text and one for images. This could be the same product or two different products.
  • Any text or images that were manually entered prior to linking are overwritten.
  • The inheritance is one time only and the information can be manipulated as needed after it has been copied over.
