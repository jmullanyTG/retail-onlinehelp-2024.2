Add sites to or removing sites from site clusters

Sites to be added must not already be part of any other site cluster associated with the promotion. Also, the sites must match the currency of the site cluster.

To add sites:

  1. Select the site cluster.

  2. Click on Add.

    The Sites selection box displays.

  3. Select the sites to be added.

  4. Click Ok.

To remove sites:

  1. Select the site cluster to display the associated sites in the pane on the right.
  2. Select the checkbox next to the sites to be removed.
  3. Click on Remove.
  4. Click Yes.
