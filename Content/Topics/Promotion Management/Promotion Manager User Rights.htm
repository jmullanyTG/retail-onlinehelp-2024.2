Promotion manager user rights

There are two sets of user rights within Promotion Management. One set not affected by the Promotion Workflow Status (i.e. global), the other affected by Status. Below is a list of global user rights and the purpose of each. These permissions can be configured by clicking Users from the Maintenance tab and using the arrow buttons to add or remove rights for a user or role.

User Right Functionality
Allow Access to Analysis Provides access to the Promotion Analysis functionality.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses
Allow Access to Bids Provides access to the Promotion Bidding functionality.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses
Allow Access to Companion Items Provides access to the Promotion Companion Items functionality.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses
Allow Access to Content Provides access to the Content button in the Promotion Detail header that allows for viewing copy and images associated with the Promotion.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses
Allow Access to Promotion Provides access to the Promotion solution.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses
Allow Access to Sync Provides access to the Sync button on the toolbar. For more information, see Syncing promotions.
Allow Creation of Price Cluster Provides access to create Site Clusters in the promotion.
Allow Creation of Promotion Provides access to create promotions.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses
Allow Creation of Tags Provide access to create new tags in the promotion header.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses
Allow Deleting of Price Cluster Provides access to delete Site Clusters in the promotion.
Allow Deleting of Promotion Provides access to delete promotions.
Allow Editing of Content Provides access to edit copy and images associated with the promotion. The Allow Access to Content right must also be granted for this to work properly.
Allow Editing of Price Cluster Provides access to edit Site Clusters in the promotion.
Allow Editing of Promotion Provides access to edit promotions.
Allow Editing of Promotion After Sale Start Date Provide access to edit a promotion after the Sale Start Date has occurred.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses but rather as a Super User privilege.
Allow Editing of Promotion Columns Provides access to edit promotion fields. User right Config - Allow Access to Promotion Config is also required.
Allow Override of Validation Failures Provides access to mark the Override checkbox, for a failed promotion Validation (Information, Warning, Error), in the Promotion Validation Log. Overriding a Validation error replaces the Validation failure severity icon with the Failure Reviewed icon identifying that a failure exists but has been overridden.
Config - Allow Access to Promotion Config Provides access to the Promotion Configuration window:Calculations, Validations, Price Output Sets, Promotion Fields, and Strategies.

The Rounding Rules and the Export Maintenance screens do not display by default. These screens are available under System Utilities > Views.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses
Modify of Promotion Status Provide access to edit Promotion Status.
