Site cluster validation statuses

The first column shows the validation status of the promotion for each site cluster. The possible statuses are (most severe to least):

Status Description
An error that requires investigation has been found. Review the promotion to resolve the problem.
A validation warning was encountered.
Validation ran successfully but additional information is available to review.
A calculation is required before a validation status can be assigned. The promotion must be recalculated manually or by an overnight process.
The promotion is successfully validated.

The most severe validation status at the site cluster level rolls up to display the validation status at the promotion level.
