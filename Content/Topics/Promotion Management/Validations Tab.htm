Validations

Validations may be configured as a mathematical formula using SQL or as an installed stored procedure for more complex requirements. To change the severity level of a validation, select the Severity column in the Validation Repository section and choose a new level from the dropdown list.

For more information on the Validation Repository section, see Validations Repository.

Users who are unsure how to best implement a new validation should contact Support or the Professional Service team.

To access this screen, click Configuration in the Promotion screen.

Validation Sets section

Validation sets are a grouping of validations that are performed in a specified order and can be assigned to a promotion strategy. Validation sets are created and maintained within the Validations tab of the Promotion Configuration window.

For more information, see Promotion Strategy.

Creating validation sets

Click New in the Validation Sets section of the toolbar. Enter the required fields and click Save.

Field Description
Name A descriptive label to summarize the purpose of the validation set. This identifies the validation set when configuring promotion strategies.
Description A summary of the validation set.

Modifying validation sets

Modifications can be made directly to the grids. Existing promotions referencing the validation set must either be recalculated manually or by an overnight process.

Cloning validation sets

Within the Validation Sets grid:

  1. Select the validation set to clone.
  2. Click the Clone button.
  3. Click Save.

A copy of the selected validation set is created with the suffix - (Clone). Modify the copy as desired.

Deleting validation sets

Validation sets cannot be deleted if associated with promotions that are not already exported and in that state.

Select a validation set and click the Delete button.

Validation Set Validations section

Any Validations from the Validation Repository can be assigned to a Validation Set. A Promotion Strategy referencing the Validation Set runs in the order that they are listed in the Validation Set.

Assigning validations to validation sets

Within the Validation Set Validations section of the toolbar, click Add to add a new validation to the Validation Set. Enter the required fields and click Save.

Field Description
Order A numeric value identifying the sequence order for the new Validation being added.
Name A Validation from the dropdown list. Populated with Validations defined in the Promotion Validation Configuration grid.
Description A read-only field that defaults to the Validation Description after changes are applied.
Active (Optional) Selected by default to identify that the Validation added is active in the Validation Set.

Modifying assigned validations

Modifications can be made directly to the Validation Set Validations grid. Existing Promotions referencing this Validation Set must either be recalculated manually or by an overnight process.

Validations can be temporarily removed from a Validation Set by unchecking the Active checkbox.

Deleting assigned validations

Select a validation and click the Delete button. Existing Promotions referencing the Validation Set must either be recalculated manually or by an overnight process.
