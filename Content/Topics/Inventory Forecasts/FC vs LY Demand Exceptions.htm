FC vs LY demand variance exceptions

The FC vs LY Demand Variance exception flags level forecasts when the forecast (FC) for the next 3 periods is exceedingly above or below the item's sales history for the same periods last year (LY).

Formula

This exception is based on the variance in demand (calculated as a percentage) between the upcoming and last year periods. The formula for determining the percentage of demand variance is: 

(Forecast - LY Demand History) / LY Demand History

Where:

  • Forecast
    The sum of Effective Forecast quantities for the upcoming number of periods (default is 3 periods).
  • LY Demand History
    The sum of Total Sales quantities for the same number of periods in the previous year (counted as 52 weeks prior to the upcoming periods).

A positive value resulting from this calculation occurs when the forecast for the upcoming periods is greater than the demand for the same periods last year. A negative value occurs when the forecast for the upcoming periods is less than the demand for the same periods last year.

Whether the resulting value constitutes an exception depends on the variance threshold defined in the exception filters.

  • The default number of periods examined for this exception is 3 and includes the current period.
  • If either the 
    Forecast
    or 
    LY Demand History
    value is equal to zero, the variance percentage will equal 100%.

Filter conditions

To filter for forecasts with FC vs LY Demand over-forecast exceptions

Select the following from the Level Forecast Exceptions field group:

  • Level Forecast Forecast LY Sales Ratio is more or equal to 3 (or whatever positive percentage value is to be the threshold).
  • Level Forecast Reviewed is False.

To filter for forecasts with FC vs LY Demand under-forecast exceptions

  • Level Forecast Forecast LY Sales Ratio is smaller or equal to -0.75 (or whatever negative percentage value is to be the threshold).
  • Level Forecast Reviewed is False.
