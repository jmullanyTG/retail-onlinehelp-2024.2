Manually adjusting a forecast

A forecast is manually adjusted by entering new values in the grid or by dragging the Manual Level Forecast data series on the graph.

  • Level forecasts have two Manual Forecast fields: Manual Forecast and Manual Level Forecast.

    The Manual Forecast field reflects manual forecast quantities captured at the Inventory Forecast level. This field is read-only in the Level Forecast view.

  • An item must be re-forecast to push the Level Manual Forecast quantities to the Inventory level.

Manually adjusting forecasts via the grid

  1. In the grid, enter a new value in the Manual Level Forecast field for the desired time period.

    The graph updates automatically to reflect the newly entered value.

  2. Click Save and Forecast to save the changes and re-generate the forecast using the manual adjustments.

    The Manual Level Forecast data series in the graph reflects the manual adjustment.

Manually adjusting forecasts via the graph

  1. In the grid, make sure the Manual Level Forecast field checkbox is selected.

    If not checked, the Manual Level Forecast data series does not appear in the graph.

  2. In the graph, click on the Manual Level Forecast value to be adjusted and drag it to the desired value.

    In the grid, the Manual Level Forecast field updates automatically to reflect the manually adjusted value.

  3. Click Save and Forecast to save the changes and re-generate the forecast using the manual adjustments.

    The Manual Level Forecast data series in the graph reflects the manual adjustment.
