Generating demand forecasts

Demand forecasts help replenish items while taking into account holidays, seasons, promotions, and so on.

To generate a demand forecast

  1. Navigate to Execute > Inventory Forecast.
  2. Click either Inventory View or Level View.

  3. Select a filter using the dropdown list in the Items dialog.

  4. Click Ok.

  5. Select an item in the Item field using the dropdown list.

  6. In the Forecast toolbar, click Forecast.

    The Forecast Task window displays.

  7. In the Forecast Task window, select the desired forecast methods and, if required, a method set from the Method Set dropdown list.

  8. Click Ok.

    The completed forecast displays.
