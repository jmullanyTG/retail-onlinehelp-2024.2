Viewing sales history and forecasted demand

Clicking on the Graph button in the Forecast viewer displays a graph. The graph contains various information series related to sales history, forecasted demand, etc in a graphical format. In the graph, certain series are displayed as a line while others may be displayed as a bar.

When hovering the mouse over the forecast graph, an informational box displays the values for the period in focus.

To decide what information to display in the graph, click the Visible checkbox next to a data series in the grid to toggle displaying or hiding that data in the graph.

The ability to configure a series’ color requires user right Allow Series Configuration.

Within the graph viewer, the dot series representing the Manual Forecast can be manipulated to affect a manual forecast value.

Series

The graphical representation of the different series related to the forecast can give insight into the meaning of the data.

Series Description
Manual Forecast Dark red if a forecast has been manually changed.
Effective Forecast The values passed along to the replenishment engine via the Inventory Manager. Should align with the system forecast (blue series) before any changes are made.
System Forecast The system generated forecast produced by the Forecasting engine. The values past the current date represent the values produced by the winning forecast model. Any series values preceding the current date may be viewed as the “model fit”, a combination of the simulation and backtrack.
  • Backtrack – The backtrack series is considered to be optimistic in that it corrects itself to be closer to the historical series for each history period.
  • Simulation – The simulation is considered to be pessimistic in that it will not correct itself in any way over the historical periods.
Original Forecast The yellow bars (not the yellow dotted line) represent the original system generated forecast. They remain for comparison as changes are made.
