Forecast screen toolbar

The toolbar contains the following functionality.

Button Description
Save Saves changes made to the forecast (re-forecast, manual forecast adjustments, etc).
Undo Enabled when changes are made to the forecast (re-forecast, manual forecast adjustments, etc). Click this button to undo the changes.
Change All Change multiple lines (either rows or columns) within the grid. They can be set to a specified value, increased or decreased, copied from another column or row, or split across other columns based on a percent contribution.
Forecast Launches the Forecast Task window to select method sets and forecast methods. Click Ok to generate a forecast.
Lock, Unlock
  • Lock. Allows users to include or exclude certain forecasting methods from being run for the inventory item. A lock expiration must be set to determine the period of time for which the lock is intended to cover.

  • Unlock. If a forecast has had a method lock applied, the Unlock button displays allowing users to remove the method lock.
Reviewed As part of a business flow process, the Reviewed button shows that a particular forecast for an item has been approved. Clicking Reviewed populates the current date in the Review Date field.
Notes Launches the Notes Viewer in which users communicate any information related to the particular inventory item as it relates to the forecast.

Causals Launches the Maintain Item Causal Links window letting users include or exclude any causals which have been created in the Causals module.
Inventory Manager Launches an Inventory Manager view for the inventory item to bring additional pertinent information to the user’s attention.
Forecast All Allow users to run a re-forecast for all items within the filter. This may take some time for very large filters.
Settings Launches the Visible Bucket Period Limits dialog. This determines the number of periods which display in both the graph and grid sections of the forecast viewer. Depending on the period type (daily, weekly, or monthly) the visible bucket period limits can be adjusted to take a bigger picture look at the inventory item, or narrow the focus if so desired.
Replace Displays if there is a need to replace a method by choosing a different method from the dropdown list. This replaces the effective forecast.
