Selecting a different forecast method

If the forecast method used for a given forecast has a low method fit or just does not make sense (for example, the item is not seasonal but a seasonal method was used), choose a different method to be used for the forecast.

To select a different forecast method

  1. In the Forecasts screen, click Lock on the toolbar.

  2. In the Forecast Task dialog box, select only the methods to be considered when forecasting demand for the item.

    For example, do not select their checkboxes if seasonal forecast methods shouldn't be used.

  3. Click Ok to close the dialog box.
  4. Click Forecast in the toolbar to re-generate the forecast using only the selected forecast methods.

Validity of selections

The selections made in the Forecast Task dialog box remain valid until new selections are made or the date defined in the Lock Expiration field is reached.

Unlocking selections

Click Unlock on the toolbar to re-open the Forecast Task dialog to select different methods after a forecast has been locked.
