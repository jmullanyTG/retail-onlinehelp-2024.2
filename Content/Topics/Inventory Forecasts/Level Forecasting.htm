Level forecasting

Level forecasting is used to define a single level above inventory using standard filter hierarchies. This higher level forecast mode is useful for forecasting by an arbitrary item group or site group (e.g. category – channel). The functionality in level forecasting is similar to functionality in aggregate forecasting.

To create a level forecast view, click the Level View button, then select an appropriate filter. A concatenated description of the product dimension level and location dimension level displays in the Item field.

To define levels, a specific inventory hierarchy for level forecasts must be created. The hierarchy is used by system policy Level Forecast Hierarchy (Engine: Forecasts). Inventory hierarchies are created using the Hierarchy Filtering view.
