Defining the curve used for shape forecasting

Shape forecasting applies a pre-selected curve to an item's moving average sales history to generate a moving-average based, season-affected demand forecast for that item.

Select the Forecast toolbar button to display the Forecast Task dialog in which Shape Forecasting methods are selected to be used in the forecast tournament.

Shape forecasting requires the following setup tasks:

  • Define appropriate curves (see Curves overview.Refer to the Curves User Guide.)
  • Identify the desired curve for each item in the Shape Forecast Maintenance view.

Shape Forecast Maintenance View

Use the Shape Forecast Maintenance view to identify the curve that generates an item's shape forecast.

Assigning or removing curves for multiple items

Select multiple items and right-click within the Curve field to display a dropdown menu. Select Multiple Changes to display the Multi Data Change Form For Column Curve window.

Selecting (None) in the New value field removes curves from all selected items.

Defining curves for items

  1. In the Views module, open the Shape Forecast Maintenance view.

  2. In the Filter list, choose the filter containing the desired items.

    The items appear in two grids. The top grid lists items at sites. The bottom grid lists items by channel. (The same items appear in both grids.)

  3. Select an item.

  4. In the Curve field, click the ... (ellipsis).

  5. In the Curve Selection dialog box, choose the curve to apply by selecting its row in the grid.

  6. Click Ok.

  7. Click Save above the grid to save the change.
