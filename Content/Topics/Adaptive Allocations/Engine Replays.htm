Debugging the draft engine

The Save Replay button can be added to the Scenario Plan screen UI through a User Right if it becomes necessary to debug the draft engine.

Replay recipients are configured in the Replay Email Recipients system policy. Email addresses are separated by either comma separated email addresses or a file path destination (located on the services tier machine).

System Policy Description
Replay Email Recipients (Engine.ReplayDestinations) The email addresses which will receive the engine execution input data.

This should only be used for debugging purposes as it has a severe performance impact.
