Creating product sets

Creation of product sets is accomplished by using a filter of variants as a starting template. Product set details (variant and attribute details) can be updated by saved changes within the product set configuration.

Product sets can be created and configured for consumption during the scenario plan building phase within the product set grid.

Click New on the toolbar to create new sets.

All fields must be completed and changes saved before the variant detail configuration sections become available. The access level defaults to Public unless otherwise specified.

Selection sets (even linked selection sets) can be deleted for users with appropriate permissions by highlighting a row and selecting Delete.

Changes made in the Variant Details sections must be saved prior to navigating away. This is represented by graying out the product set view.
