Additional data requirements

Price and cost

For customers using Price Management, the price and cost conversion calculations read information from tbl_InventoryPrice and tbl_InventoryCost (Where PriceType/CostType = system policy values, default to 1) if there is data available. This may also be imported.

If this data is unavailable, the conversion calculations will be based on the imported values for DefaultCostPrice and DefaultSellingPrice.

Location planning

To use location plans as a source for demand plan generation, there must be either existing location plans, or imported financial plan data.

Using strategic financial plans

To use strategic financial plans as a source for demand plan generation, there must be either existing strategic financial plans, or imported financial plan data.
