Working with scenario plans

The Scenario Plan screen displays demand plan results, choice count results, and scenario allocation volumes. Scenario plans can be saved at any time and changes reversed by using the Undo button.

The Configuration Status and Plan Status fields display information which can be used to complete the construction of the scenario plan. Specific details on configuration exceptions may be viewed by clicking on the ellipses (…) in the Configuration Status field. This launches the Configuration Exceptions pop-up.
