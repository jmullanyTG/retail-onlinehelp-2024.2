Variant-Size range relationships

After a size range has been configured to cover the range of sizes for a variant (a configured data entity, typically relating to style-color), a variant-size range relationship should be configured.

These size ranges will likely be built in reference to a specific level of the product hierarchy (e.g. Mens Apparel) selected in Variant Manager. The Size Range Name field returns available size ranges.

For more information, see Variant Manager overview. For more information, refer to the Variant Manager User Guide.
