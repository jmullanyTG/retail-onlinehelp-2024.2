Size ranges and default curves

Configuration of size ranges is required to correctly sample supporting data and create selection options. A size range should be configured to support the full breadth of possible sizes for a variant across all locations. As a new size range is generated, default curves are created (system generated) to cover size curve option values for all sites.

A default curve is generated by specifying size curve creation parameters (sales types, etc.), then populating the Default Size Model column within the Size Range Manager blade in Variant Manager.

For more information, see Variant Manager overview. For more information, refer to the Variant Manager User Guide.
