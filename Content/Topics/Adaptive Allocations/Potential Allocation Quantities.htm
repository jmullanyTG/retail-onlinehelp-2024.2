Potential allocation quantities

The Allocation Volumes blade displays the potential allocation quantities. After quantities are verified as a desirable result, the scenarios can be converted to orders. Additionally, all attribute families included in the draft process display along with the associated attribute value for each item master. 

Viewing allocation results in a pivot grid

To view allocation results in a pivot grid, toggle the view using the Pivot button within the Scenario Details menu section. This displays the total volumes by store, by variant, or by attribute value. It is highly recommended to use the pivot grid view to verify potential allocation quantities before proceeding.

Attribute descriptions

  • Site Code – The unique code for the site.
  • Site Description – The description for the site.
  • Variant Code – The unique code for the variant.
  • Variant Description – The description for the variant.
  • Item Master Code – The unique code for the item master. This provides size level detail.
  • Item Master Description – The description for the item master.
  • Size Name – The unique code for the size. This is unique by item master and can be removed from the view if sufficient information exists to describe size in the item master code or item master description.
  • Size Description – The description for the size.
  • Allocation Quantity Value – The value in base currency of the allocation quantity units.
  • Allocation Quantity Units – The calculated quantity of units to be allocated as the result of the scenario plan.
  • Store On Hand Value – The value in base currency of the store on-hand units.
  • Store On Hand Units – The quantity of units on-hand at the store at the time of scenario calculation.
