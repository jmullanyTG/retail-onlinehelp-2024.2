Viewing choice counts by store

The Choice Counts blade displays the calculated choice count by store, alongside the historical choice count for the same store as used in the sampled time frame. This can be used for reference when validating choice count results. All stores within the location strategy display.

Viewing choice counts as a graph

To view the results as a graph, toggle the view using the Graph button within the Scenario Details menu section.

Fields and descriptions

  • Site Code – The unique code for the site.
  • Site Description – The description for the site.
  • Choice Count – The calculated choice count result.
  • Historical Choice Count – The historical choice count used within the calculation.
