Generating orders from calculated scenario plans

Once a scenario plan has been calculated it will be in a plan status state of Ready to Execute. A scenario plan in that plan status state can be converted to an order which finalizes the plan.

Once finalized, scenario plans become read-only to prevent duplication of orders.

The Generate Order button is enabled for calculated scenario plans.

Once selected, users must approve the finalization of the scenario plan.

Upon approval, the orders are opened within an Order Manager document.

Using Order Manager to generate orders from scenario plans

Selecting From Scenario within Order Manager displays the Add From Scenario Allocation Order wizard in which an order document based on a particular scenario plan can be generated.
