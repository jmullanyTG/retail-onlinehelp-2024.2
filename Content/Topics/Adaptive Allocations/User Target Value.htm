User target value as demand source

If User Target Value is selected as the demand source, the following fields display in the Detail Configuration window:

  • Product Hierarchy – (read-only) Displays the system policy defined product hierarchy. It is purely informational.
  • Product Selection – A mapping of the applicable levels within the product hierarchy which will be sampled to create the demand plan.
  • Target Value – A direct entry of the financial target for the demand plan. Expected to be entered in base currency (as specified in the system policy value). 

  • Product Model – The product model is a calculated index of sites which will function to appropriately dis-aggregate need target values across stores.

    For more information on Product Models, see Product models. For more information on Product Models, refer to the Allocations User Guide.
