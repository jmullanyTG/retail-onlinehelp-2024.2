Per store demand plan values

After a demand plan has been tied to a scenario plan (and, therefore, a location strategy) the per store demand plan values can be calculated and displayed on the Demand Plan Results blade.

Fields and descriptions

  • Site Code – The unique code for the site.
  • Site Description – The description for the site.
  • Need Target – The calculated need target for the store. Displays in base currency and is representative of total cost of goods sold (COGS) or retail sales target based on the specified accounting method.
  • Site Groups 1-20 – Additional site grouping information (i.e. channel) may be returned to provide reporting options.
