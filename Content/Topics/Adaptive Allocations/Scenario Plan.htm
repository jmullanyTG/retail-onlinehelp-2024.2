Scenario plans

Scenario plans produce potential allocation quantities once the dependent configuration requirements have been met. Depending on the scenario outcome, the potential allocation quantities can be converted to orders.

The Scenario Plan screen is used to build and execute on the combined demand plans, product sets, and choice count models to generate allocation volumes to stores. Configuration occurs in the Scenario Plan grid and results are available in the Scenario Details grid.

Like all management screens, the screen can be undocked and realigned or moved to a secondary monitor by clicking and dragging the tab to the desired location.

Creating new scenario plans

  1. Click New to create new scenario plans.

  2. Fill in all fields before saving.

    The default access level is Public unless otherwise specified.

If any configuration details are deleted, previously executed plan results will not be altered. However, any plans which include a deleted configuration element will not recalculate (plan status updates to Configuration Changed).
