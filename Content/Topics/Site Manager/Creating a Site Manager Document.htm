Creating and reviewing site manager documents

From Site Manager, create and review site manager documents to manage sites.

To create a site manager document

  1. From the Management tab, click Site Manager.

    The Items window displays.

  2. Select the filter that contains the sites you want to manage and click Ok.

    The Site Manager Document view displays.

  3. In the Columns toolbar, click Choose.

  4. Select additional fields to display and click Ok.

    The Site Manager Document displays.

Filtering sites to match specific criteria

Metadata is used to filter sites matching specific criteria. This metadata appears as fields in the grid in the Site Manager Document view.

Exception filters

Metadata can be used to create exception filters that identify sites in need of additional configuration, such as sites that:

  • have not yet been assigned to a site cluster.

  • do not appear in any Allocation location strategies.

    For more information on filtering sites in location strategies, see Allocations overview. For more information on filtering sites in location strategies, refer to the Allocations User Guide.

  • have not been added to a supply chain.

Reviewing site manager documents

Click the Details button to review individual sites in the grid.

For more information, see Site manager document detail pane.

Field descriptions

Field Description
Site Code The numeric site code of the selected site.
Site Name The name of the selected site.
Priority The priority of an item (days, years, etc.).
Priority Group Associated priority group used to resolve shortfall situations. These groups can be used to protect a specific set of sites before referring to global priority to resolve shortfall situations. Takes precedence over global priority.
Currency The currency applied to the site.
Channel Code The channel code where an item is sold, such as retail, outlet, e-commerce, no channel.
Channel Description The channel description.
Current Site Comparative Syte The status of each site (comparative or non-comparative) for the current period.

If a site does not have a comparative status for this period, this value is blank.
