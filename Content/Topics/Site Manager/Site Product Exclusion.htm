Creating restrictions for site products

Site product exclusion helps manage inventory restrictions at a site level. To create restrictions, add item master filters to the exclusion list or select segment of the product hierarchy.

Only one product exclusion can be set per site at a time. Users must either create an item master filter to fulfill the requirements or set the exclusion at an applicable level of the hierarchy.

The ItemMaster Hierarchy, ItemMaster Filter Master, and Site.Capacity:Product Hierarchy filters were created as placeholder examples. Filters and system policy filters must set up before applying restrictions for a site product. The product hierarchy used by the site product exclusion is determined by the system policy settings.

Product exclusion sets

Using the Site Product Exclusion tab, users can create reusable product exclusion sets based on combinations of item filters and product hierarchy levels. After creation, product exclusion sets can be quickly assigned to any site in the grid.

The hierarchy used to populate the product dimension on the Site Product Exclusion Set tab is determined by the Site ItemMaster Exclude Filter ItemMaster HierarchyID system policy.

Creating product exclusion sets

  1. Navigate to the Site Product Exclusion tab and click New.

    An empty row displays.

  2. Enter the name and description for the product exclusion set.
  3. Click Save.
  4. In the Levels section, click the ellipsis in the Levels column and choose a level.
  5. Click Save.
  6. In the Filters section, click New.
  7. In the Filter column, click the ellipsis to open the Items window.
  8. From the Filter dropdown menu, select a filter and click Ok.
  9. Click Save.

Applying product exclusion sets to sites

  1. Click Choose on the Site Manager Document toolbar and include the Site Exclusion group.

  2. Click Ok.

    The Site Manager Document displays the Product Exclusion Set and Product Exclusion Exists columns.

    Column Description
    Product Exclusion Exists If a product exclusion set has been applied to the selected site.
    Product Exclusion Set Specifies the product exclusion set applied to the site.

  3. Click in the Product Exclusion Set field.

  4. Click the dropdown arrow and select a product exclusion set from the list.

  5. Click Save on the toolbar.

    The Site Manager Document refreshes and the checkbox in the Product Exclusion Exists field is checked.

ItemMaster Hierarchy Level

Selecting ItemMaster Hierarchy Level from the Exclude Filter Type displays a tree wizard with the selected hierarchy as set by the system policy. Levels to be excluded must be selected.

ItemMaster FilterMaster

If ItemMaster Filter Master is selected from the Exclude Filter Type, then select a filter from the filter chooser to be applied.

Field descriptions

Field Description
Site Code The numeric site code of the selected site.
Site Description Description of the selected site.
Exclude Filter Type Determines the method for product exclusion. The eligibility does not overlap.
  • ItemMaster Hierarchy Level: if selected, the Exclude Levels field becomes active letting users select segments of the product hierarchy for exclusion.
  • ItemMaster Filter Master: if selected, the Filter Master ID field becomes active letting users select products for exclusion based on an item master filter.
Field Master ID If selected, select products for exclusion based on an item master filter.
Exclude Levels If selected, select segments of the product hierarchy for exclusion.
