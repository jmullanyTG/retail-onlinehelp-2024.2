Viewing source priority for a store site

The Site Source Priority tab lets users view source site priorities for other sites which are classified as stores. This is a read-only view that is populated by the Supply Chain module. All active store sites which are not included in the siteâ€™s priority are struck-through.

A system policy determines the supply chain that populates this view.

To view the source priority for a store site

  1. Display the Site Manager Document containing the site.

    For details, see Creating a site manager document.

  2. Select the desired site in the Site Manager Document.

  3. Click the Site Source Priority tab.

    The Site Source Priority section displays.

Field Description
Source Site Code Site code for the store.
Source Site Type The site type (store).
Source Site The name of the store.
Source Site Priority The priority ranking as established by the supply chain.
Default Site Lead Time Days The default lead time (set at site level. Product agnostic).
