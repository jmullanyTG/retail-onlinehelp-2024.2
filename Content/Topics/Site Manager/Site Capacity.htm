Setting site capacity

Site capacity information can be viewed and updated using the Site Capacity tab. This data is used for reporting and filtering only. Site capacities can be defined at a desired level in the product hierarchy (total store, department, style, etc.). The aggregated capacity displays at the higher levels.

The hierarchy used to populate the product dimension on the Site Capacity tab in the Site Manager Document is determined by the Site Capacity:Product Hierarchy system policy.

To set up site capacity

  1. Click the Choose button and move the Site Capacity group to the Included column.

    The following columns are added to the top grid: Total Selling Capacity and Total Stockroom Capacity. These columns display the totals for the Selling Capacity and Stockroom Capacity fields respectively.

  2. Click Ok.

    The Site Manager Document updates with the new columns.

  3. Select the Site Capacity blade.

    The parameters for site capacity display.

  4. Click Parameters.
  5. From the Hierarchy Level dropdown menu, enable the hierarchy levels to use, with the lowest level enabled being the level at which site capacity is entered.
  6. Click Save.

To set up site capacity at the system policy level, repeat the procedure for setting up site capacity at the hierarchy level. From the Hierarchy Filtering view, enter the ID of the hierarchy form.

Field Description
Hierarchy Level The specified hierarchy level at which capacity is measured for the site. The hierarchy used is controlled through a system policy setting.
Level The level value in the product dimension. This value associates directly with the hierarchy levels.
Selling Capacity Showroom capacity in quantity of units.
Stockroom Capacity Stockroom capacity in quantity of units.
Minimum Capacity Minimum capacity for the total site.
Maximum Capacity Maximum capacity for the total site.
Active A flag that identifies the active status of the capacity constraint.
Created Date Date of creation for the capacity entry.
Created By Who set the site capacity.
Last Updated Date of the most recent update to the capacity entry.
Last Update By Who last updated the site capacity.
