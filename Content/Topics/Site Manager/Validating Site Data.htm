Validating site data

A Site Manager Document displays all stores added during a particular period of time. This document is used to verify that the information received from the ERP system for each site is valid.

To validate site data

  1. From the Management tab, click Site Manager.

    If the Items dialog box does not automatically open, click + Site Manager Document on the toolbar to open it.

  2. From the Items filter dropdown menu, select and double click a filter.

  3. Click Ok.

    The Site Manager Document displays the selected stores.

  4. In the toolbar, click Choose and select additional fields to display in the site document.

  5. Review the fields for each new site to ensure the values they show are as expected.

  6. Click the tabs in the Details pane and verify the values displayed.
