Setting default contributions (split percentage) for suppliers

The Site Account Source Site Contribution blade displays a cross product of items in the hierarchy and all DCs. The blade displays, as a calculated percentage, the default contribution of each DC with the calculation being performed by new job task Calculate Source Site Percent Contribution for Accounts which looks at an account’s sales history (customer orders). Users can use the Site Account Source Site Contribution blade to refine contribution percentages as an override to set what percentage of an account’s projected stock each DC should hold. Overwritten values take priority over the calculated percentage.

After a user enters a value in the Override field and clicks Save, the Clear Override field for that row is formatted orange. The Clear Override functionality clears overrides for all the DCs for the item. When overrides are cleared the default contribution percentage is used.

If 10,000 units have a 50/50 split between 2 DCs out of a total of 4 DCs, then 5,000 units are sent to each of the 2 DCs.

System policy Site Wholesale Account Distribution: ItemMaster Hierarchy ID defines the hierarchy used.
