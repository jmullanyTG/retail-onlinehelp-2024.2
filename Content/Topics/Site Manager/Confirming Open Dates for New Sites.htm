Confirming open dates for new sites

Opening dates for new stores can be confirmed as being accurately scheduled. Site opening dates help determine when orders need to be placed for the new stores.

To view open dates for a new store

  1. From the Management tab, click Site Manager.

    The Items window displays.

  2. Select a filter using the Filter dropdown menu.

  3. Click Choose and use the arrows to select fields to display in the document.

  4. Click Ok.

    The Site Manager screen displays the required information.

  5. Double-click a site to display its details.

    Click Details on the toolbar to display the Details pane if hidden.

  6. Select the Site Status tab and review the open date.
  7. If required, click New and enter the following information in the respective fields:
    • Site Status: Open
    • Site Status Attribute: Open
    • Status Start Date: Store opening date

  8. Click Save.
  9. In the grid, review the values in the fields to make sure they are correct.
