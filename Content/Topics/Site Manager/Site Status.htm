Managing site status

Site status refers to when a site will be open (such as a new store opening) or closed (such as for remodeling or permanent closure). The system uses site status to generate accurate forecasts and order recommendations. That is why it is important to keep sites updated with status changes as they are planned.

The Site Status tab in the Site Document Manager Document contains a grid that displays when the status of a selected site is scheduled to change. Each upcoming status change is displayed as a row in the grid.

To add a column to the Site Status tab

  1. Display the Site Manager Document containing the desired site.

    For details, see Creating a site manager document.

  2. Select the site in the Site Manager Documen.

    Make sure that the Details pane is displayed.

  3. In the Site Manager Document, click the Site Status tab.

    The Site Status Over Time section displays.

  4. Select the desired site.

  5. Click the Fields button on the Entry toolbar.

    The Customization window displays the available columns.

  6. Search for a column or double-click an existing one.

    The selected column displays in the Site Status Over Time grid.

Field Description
Site Code The numeric site code of the selected site.
Site ID The site identification number.
Site Status If the site is open or closed during this period.
Site Status Attribute The reason for the status change and specifies if the orders are accepted at the site during this period.

Examples:

  • Remodel (accepting orders)
  • Expansion (no orders)
  • Permanently closed
Notes Optional. Any additional information.
Site Status Temporal ID The site status temporal identification number.
Status Start Date The date on which this status is to go into effect.
Status End Date The date on which this status is to end.

To add a new status entry for a site

  1. Display the Site Manager Document containing the desired site.

    For details, see Creating a site manager document.

  2. Select the site in the Site Manager Document.

    Make sure that the Details pane is displayed.

  3. In the Site Manager Document, click the Site Status tab.

    The Site Status Over Time section displays.

  4. From the Site Status tab, locate the Entry toolbar, and click New. A new row is added to the grid below.
  5. In the new row,:
    • enter values in the Site Status and Site Status Attribute columns.
    • select a start date and any end date for the status in the Status Start Date and Status End Date columns.
    • enter any optional notes for the status change in the Notes field.
  6. Click Save to save the changes and populate the entry's ID values.

To modify a status entry for a site

  1. Display the Site Manager Document containing the site to be modified.

    For details, see Creating a site manager document.

  2. Select the site in the Site Manager Document.

  3. Click the Site Status tab.

    The Site Status Over Time section displays.

  4. Select a row in the Site Status column and make the necessary changes.
  5. Click Save.

To delete a status entry from a site

  1. Display the Site Manager Document containing the site to be deleted.

    For details, see Creating a site manager document.

  2. Select the site in the Site Manager Document.

  3. Click the Site Status tab.

    The Site Status Over Time section displays.

  4. Select the row that contains the site status to be deleted.
  5. Click Delete to remove the entry from the grid.
  6. Click Save.
