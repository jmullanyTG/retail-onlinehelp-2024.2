Creating a view

To reference metadata, the metadata must either be contained in a specific view or referenced in an existing table. An initialization procedure (INIT) must also be created to populate this new view. A primary key must be included in the view in order for functions in the tbl_Common_Filter_MetaViewData table to work.

Example for uvw_Site_SiteCapacity

SETANSI_NULLSON GO SETQUOTED_IDENTIFIERON GO CREATEVIEW [dbo].[uvw_Site_SiteCapacity] AS SELECT si.SiteID ,SUM(c.SellingCapacity)AS TotalSellingCapacity ,SUM(c.StockroomCapacity)AS TotalStockroomCapacity FROM tbl_Site si LEFTOUTERJOIN tbl_Site_Capacity c ON c.SiteID = si.SiteID GROUPBY si.SiteID; GO

After the view is created and populated, it can be referenced in the tbl_FilterType table.

Initialization

Initialization procedures must be created and/or run to populate the tbl_Common_Filter_MetaViewData table.
