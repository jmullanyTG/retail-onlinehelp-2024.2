Viewing site location strategies

In Allocations, store type sites are grouped into location strategies to determine allocation quantities. The membership of a specific site is not exclusive to a single location strategy. The membership of a site in active location strategies can be reviewed using the Site Location Strategies blade in Site Manager.

To view the strategy location of a site

  1. Display the Site Manager Document containing the site to be modified.

    For details, see Creating a site manager document.

  2. Select the site in the Site Manager Document.
  3. If not already displayed, click the Details button.

  4. Click the Site Location Strategies blade.

    The strategy location details display.

Field Description
Is Included? Indicates if a site is included in a location strategy. A site’s membership can be modified in this view. Distribution Centers cannot be included in location strategies.

Any updates to location strategies require calculation of all linked allocation strategies.

Strategy Location Name The name of the location strategy.
Strategy Location Description The description of the location strategy.
