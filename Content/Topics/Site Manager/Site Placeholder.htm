Creating a site placeholder

Placeholder sites are created and managed using the Site Placeholder blade in the Site Manager’s Details pane. The Placeholder sites are accessible through site filters in Assortment Planning, Merchandise Financial Planning, Allocations, and Adaptive Allocations. These virtual sites can facilitate planning for stores that do not yet have master data in the ERP system.

For example, in Allocations, placeholder stores can be used to create initial orders for stores that will open soon and then reserve stock for the new stores at the DC. In Assortment Planning and Merchandise Financial Planning, placeholder stores can be used to represent stores that will open soon.

Placeholder sites can only be created for stores, not distribution centers (DC) or DC/Stores.

When master data for a store is imported, the placeholder store maps itself to the real store’s site code and all data related to the placeholder site references the new permanent site.

Placeholders and order management

Orders for real and placeholder sites are treated in exactly the same way. The workflow for these are the same. Authorized orders to placeholder sites are sent to the ERP system. If required, set up a validation in the export process to block orders to placeholder sites from being sent to the ERP.

Prerequisites

The following user rights are required to add, modify, and delete placeholder sites.

Right Group: Site Manager

  • Allow Deletion of Placeholder Sites
  • Allow Updating of Placeholder Sites

For more information, see Right group: site manager. For more information, refer to the Users User Guide.

To create a site placeholder

  1. Display the Site Manager Document containing the desired site.

    For details, see Creating a site manager document.

  2. Select the site in the Site Manager Document.

  3. With the Details button selected, click the Site Placeholder tab.

    The details to the selected site display.

  4. Click New on the Entry toolbar.

    A new row displays in the Site Placeholder grid.

  5. In the new row, enter details for the placeholder site.

  6. Click Save.

    The new placeholder site is saved.
