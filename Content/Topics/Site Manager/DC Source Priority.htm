Viewing source priority for a distribution center (DC)

The DC Source Priority blade displays the site specific sourcing priority for distribution centers. This is a read-only view that is populated by the Supply Chain solution. All active DC sites that are not included in the site’s priority display struck-through.

A system policy determines the supply chain that populates this view.

To view the source priority for a distribution center

  1. Display the Site Manager Document containing the desired distribution center.

    For details, see Creating a site manager document.

  2. Select the desired distribution center in the Site Manager Document.

  3. In the Site Manager Document, click the DC Source Priority tab.

    The DC Source Priority blade displays.

Field Description
Source Site Code Site code for the distribution center.
Source Site Type The site type (DC, DC/Store).
Source Type The name of the distribution center site.
Source Site Priority The priority ranking as established by the supply chain.
Default Site Lead Time Days The default lead time (set at site level; product agnostic).
