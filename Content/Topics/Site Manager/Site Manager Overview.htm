Site Manager Overview

Use the Site Manager module to view and modify information about store sites.

Site Manager documents

Site data is managed in tabs called Site Manager documents. The collection of sites displayed in a Site Manager document is controlled by a filter which limits the sites to only those that need attention. Multiple documents can be open at a time and each document can list a separate set of sites.

Site modeling

When opening a new store or Distribution Center (DC), select an existing store or DC to model forecasts after. This creates a store supersession which uses the product level sales history from the model site to forecast demand for the new site. Orders for the new site can then be generated based on the modeled forecasts.

For more information, see Planning demand for a new store.

Global priority

Global priority sets the default order in which sites should be sourced when any site experiences a shortfall in stock.
