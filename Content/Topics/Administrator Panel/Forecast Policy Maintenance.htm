Forecast policy maintenance

Forecast policy maintenance views are used to manage settings related to forecasting that are specified by inventory. The functionality in these views are incorporated into the Forecast solution on the System Utilities tab.

Forecast Effective Priority Maintenance view

The Forecast Effective Priority Maintenance view is used to specify the priority and resolution when there are multiple potential answers for which demand signal source should be in effect.

Forecast View Config Details section

  • Database View: the name of the database object that controls how forecast data is returned and consumed.
  • BOM: the BOM (Bill of Materials) field must be checked if the BOM forecasts should be considered. If selected, BOM forecasts display in the Forecast Type dropdown menu in the Forecast View Config Priorities section.

Forecast View Config Priorities section

The values displayed in this section are based on the parameters selected in the Forecast View Config Details section. New entry lines can be added for each forecast type to be considered.

  • Forecast Type: the forecast types to be considered.
  • Priority: priority ranking of the selected forecast type.

Any changes made within the view must be saved before updates are made.

Forecast Phase Time Frame Maintenance view

An effective forecast series can be linked to a time frame to create a very specific phase period or periods. This allows for non-contiguous (Multi-Phasic) forecast series through configuration of schedule based time frames. An inventory filter can be selected and a time frame applied where applicable.
