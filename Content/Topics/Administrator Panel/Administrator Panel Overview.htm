Administrator Panel Overview

The Administrator Panel serves as a container for configuration and maintenance of specific views and is configured based on the specific needs and capabilities of client IT administrators and super users. The panel can include some or all of the listed views and varies depending on the scope of solutions and service or support agreements in place.

The panel is configured using Views and Generic View Collection Set Module under System Utilities.

To configure the Administrator panel

  1. Select Views.

    The Navigation Menu displays.

  2. Click the View Collection Sets button.

    The Maintain View Collection Set window displays.

  3. Select the group where the new collection set will be added.

  4. Click the New button next to View Collection Set.

    The New View Collection Set window displays.

  5. Label the new view collection set Administrator Panel, and click Ok.

    The Maintain View Collection Set window reappears and displays the new collection set.

  6. Select the new collection set.

  7. Click the New button next to View Collection.

    The New View Collection window displays.

  8. Enter the details of the view collection.
  9. In the User Access Rights section, assign user access rights to the respective roles.

  10. Click Ok.

    The Maintain View Collection Set window displays the new collection under the respective collection set.

  11. Select Generic View Collection Set Module.

    The Configuration screen displays.

  12. Click the Add button.

    A new row is added to the grid.

  13. Enter the necessary information to configure the view collection set.
  14. Click Save.
