Finding and replacing filter criteria

The Find and Replace Filter Criteria view, available from the Maintenance folder in the Navigation Menu, is used to find any filters created with a particular criteria, and to replace that criteria with a new one.  A common use of this view may be after a hierarchy reclassification.

For example, finding any filters that include the condition “items in Company > Department Womens > Subdepartment Tops > Category Tees” and replace that condition with “Company > Department Womens > Subdepartment Shirts > Category Short Sleeve.”

To find and replace filters

  1. From the System Utilities tab, click Views.

    The Navigation Menu displays.

  2. Select Find and Replace Filter Criteria.

    The Find and Replace Filter Criteria view displays.

  3. Select the entity type from the Entity Type dropdown menu.

  4. Next to the Find Filter field, select the Maintain Filters icon.

    The Maintain Filters window displays.

  5. Select Update The Following Filter and choose a filter.
  6. Click Ok.

  7. Update the filter as required.
  8. Click Ok.
