Modes and methods in forecasting

Modes and methods together define the forecast base. It is sometimes referred to as a forecast method, but ignoring the mode misses a large part of how a forecast is generated for a particular series.

There are three different types of modes: two for quantitative methods and one for the naive approach.

The classification of forecasting methods can be broken down into two distinct groupings. The group called averaging methods conform to the conventional understanding of what an average is, namely, equally weighted observations.

The second group of methods applies an unequal set of weights to past data and because the weights typically decay in an exponential manner from the most recent to the most distant data point, the methods are known as exponential smoothing methods.

In addition to these two types of forecasting methods the system uses linear regression and intermittent demand.

There is a system default forecasting method labeled system default. The system default method is a moving average that takes into account as many points as are available. It is not user configurable and is designed to return a result regardless.
