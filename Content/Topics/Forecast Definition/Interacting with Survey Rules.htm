Interacting with survey rules

Interaction with the survey rules occurs in one of the two following places:

  • The forecast viewer displays a specific trend rule if it has disqualified the method from the forecasting tournament. This functionality is described in the Forecasts documentation. For more information, see Inventory forecasting overview. refer to the Inventory Forecasts User Guide.

  • The Edit Method Definition window within the Forecast module.

    The Edit Method Definition window shows several tabs across the top of the window. Each tab is specific to a set of rules.
