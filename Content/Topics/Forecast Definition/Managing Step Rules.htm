Managing step rules

The step rules are managed within the Edit Method Definition window.

A step rule may be added to a definition by clicking on the Add button and Delete to remove a step rule from the definition.

Clicking on the lower Add button populates the step rules stages and default parameter values windows. A step rule may appear in more than one stage by clicking the Add button multiple times.

Step rule information is stored in the database as a “forecast modifier”. It has a modifier type of 4. Any details may be found in tbl_Forecast_Modifier.

A default value for the parameter must be chosen for any series rule which requires it.
