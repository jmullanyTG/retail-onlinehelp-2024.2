Managing method series rules

To work with Definitions and method series rules, select the Add, Edit, or Delete buttons on the Definitions tab toolbar as required. Method series rules are added or edited using the Add Method Definition and Edit Method Definition windows.

To delete a definition, click Delete and select Yes in the pop-up window.

Series rule information is stored in the database as a forecast modifier. It has a modifier type of 3. Any details may be found in tbl_Forecast_Modifier.

A default value for the parameter must be chosen for any series rule which requires it.
