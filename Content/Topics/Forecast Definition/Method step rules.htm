Method step rules

A step rule modifies a step, which is a single data point. This is used to disregard any outliers or anomalous data points when calculating forecasts.

The available step rules are identical to the series rules (with the exception of Grubbs and Fill In and the addition of the Period Trend Rate Limiter). However, they are applied to any single point in the selected stage (Backtrack, Simulation, Forecast).

Step Rule Configuration Parameter
Trend Limiter The explicit limiting of the data trend. StepTrendLimit
Trend Rate Limiter The dampening of the trend over time. This step rule must be coupled with the Trend processor, which itself calculates the Slope as an independent process. StepTrendPercentRateLimit
Percent Bias Adds or removes percent bias to the data series. StepPercentBias
Min The absolute minimum for the data series. StepMin
Max The absolute maximum for the data series. StepMax
Period Trend Rate Limiter Applies a period-on-period rate limit on trends. This step rule must be used with an exponential smoothing method where the method itself calculates the Slope. StepTrendPercentRateLimit
