Editing forecasting methods

To edit a forecasting method, highlight the row and click the Edit button or double click on the row.  The Edit Method Definition window displays.

Within the Method Fundamentals tab, complete the following related to the forecast base:

  • Method Name – The name for the forecasting method. Commonly matches the description in the Base Method field.
  • Method Description – Any additional description for the forecasting method.
  • Method Group – Used for sorting and organization of multiple forecasting methods.
  • Method Active – If checked, the method is active and may be entered into the forecasting tournament.
  • Forecast Mode – The three possible mode types.
  • Base Method – The possible methods for use in forecasting.
  • Included Parameters – The mathematical parameters associated with the method displayed in the Base Method dropdown.
