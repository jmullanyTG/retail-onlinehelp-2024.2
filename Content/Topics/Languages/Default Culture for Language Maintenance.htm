Default culture for language maintenance

A single language can have multiple cultures, each with its own set of translations. The culture setting allows users to further localize their settings within a specific language.

  • Language – The language which has been selected from the dropdown menu.
  • Culture – The culture setting specific to the language.
  • Default Flag – The default culture if the language is selected.

Selecting languages

Languages are selected from a dropdown menu.

For languages with available culture settings, the cultures appear once the language has been selected.

Saving changes

Changes made to the culture settings must be saved at the end of the session.
