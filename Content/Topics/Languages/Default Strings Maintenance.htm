Default strings maintenance

The Default Strings Maintenance screen displays current string names and their translated string value, along with the date of change. This is used to add default strings to the selection available for translation.  Strings added need to be valid. 

  • String Name – The string name as used by the application itself. Without a translation, this is the value which displays in the User Interface.
  • String Value – The translated value.
  • Date Time – The date on which the translation was created or edited.
  • Note – Any applicable notes.

Locating individual strings

Users can use the search bar to locate individual strings (or use a contextual keyword). New entries can be made by clicking on the Click here to add a new row section in the grid.

String values can be edited by clicking on the String Value field.

Importing or exporting defaults language files

The defaults languages files can be imported or exported as a ToolsGroup JustEnough Defaults Language (.dlan) file type.
