Setting apply and ordering rules

The Maintain Inventory Profile Set Apply Rules view is used to manage conflict resolution between profile sets. Application order can be set as well as start and end dates for the application of the set. These can be changed by clicking on the Edit Apply Rule button within each profile set.

  1. Click Manage on the Profile Set toolbar.

  2. Select a profile set and click Set Apply And Ordering Rules on the Configuration toolbar.

    The Maintain Inventory Profile Set Apply Rules view displays.

  3. Click Edit Apply Rule to display the Profile Set Apply Rule window.

  4. Update fields as required and click Ok.

Fields and descriptions

  • Model Name – The name created and assigned to the profile set.
  • Order – The specified execution order in which the rules are applied.
  • Apply Type – The manner in which the model is applied to inventories.
    • Do Not Apply – Will not apply the profile set to inventories.
    • Automatic – Will automatically apply the profile changes to inventories.
    • Manual – Will manually apply profile changes to inventories when users request a manual push.
  • Apply From, Apply To – A start and end date to the application rule.
