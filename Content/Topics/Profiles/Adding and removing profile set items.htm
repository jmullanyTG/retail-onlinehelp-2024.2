Adding and removing profile set items

Profile sets are managed using the Profile Set Management window.

To add a profile set item

  1. Click Profiles and then Manage.

    The Profile Set Management window displays.

  2. Select the profile set and click Ok.

    The profile set details display.

  3. Click Add.

    The Add Profile Set Items window displays.

  4. Enter the date ranges for the profile start, profile sample, profile effective, and value for the profile evaluation period

  5. Click Next.

  6. Enter the Group Types for the profile item in the Group Types field.

  7. In the Group Conditions section, set the included or excluded groups for each group type.

  8. Click Ok and then Next.

  9. Add the individual profile set items.

  10. Click Finish to populate the profile set items.

    11. Each item line can be modified for dates and volumes. The effective dates are the time period during which the profile curve is applied to the projected demand.

To remove a profile set item

  1. Select the profile set item to be removed.

  2. Click Delete.

    A message prompts for confirmation of the deletion.

  3. Click Yes.

    The profile set item is removed.
