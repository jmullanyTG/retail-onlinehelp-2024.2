Applying profile sets to inventories

Select the Inventory Apply Rules button to determine the inventories to which the profile sets are applied. This allows for changing how the profiles are applied at the most granular level.

To apply profile sets to an inventory

  1. In the Profile Set screen, click Manage. The Profile Set Management screen displays.

  2. Select a profile set.
  3. Click the Inventory Apply Rules button.

  4. Use the Items filter to select a filter and click Ok.

    The Maintain Inventory Profile Set Apply Rules window displays.

  5. Click the Change Model button to display the Select Profile Set window.
  6. Use the dropdown list to select a profile set.
  7. Click Ok.
  8. Complete the fields in the Profile Set Apply Rule dialog as required.
  9. Click Ok to exit the windows.

To edit apply rules

  1. Click the Edit Apply Rule field in the Maintain Inventory Profile Set Apply Rules window.

    The Profile Set Apply Rule window displays.

  2. Select a rule in the Apply Type field.
  3. Complete the date fields as required.
  4. Click Ok.
