Editing hierarchy settings

Profiles are created at a location level and a product level. This makes it possible to apply a profile to the projected demand for dozens or hundreds of inventories in a single step. This is useful when introducing a new product category or creating profiles for new locations.

The hierarchy has to have at least one product level and one location level. This gives users granularity options.

To edit hierarchy settings

  1. From the Execute tab, click Profiles.

    The Profile Management view displays.

  2. Click Manage.

    The Profile Set Management window displays.

  3. Select a profile set from the list and click Edit.

    The Update Profile Set window displays.

  4. Click Edit Hierarchy.

    The Maintain Hierarchy window displays.

  5. Do one of the following:
    • To add hierarchy levels, click the Product Level and Location Level dropdown lists using the Available Levels dropdown menu.
    • To reorder the levels, select a level and click the up and down arrows on the right side of the window.

  6. When finished, click Ok.

Fields and descriptions

  • Available Levels – Which levels (from various entities) to include.
  • Description – Level descriptions.
  • Level Order – In the same entity type, the levels must be ordered to infer a hierarchical relationship (e.g. itemgroup1, itemgroup2, etc.)
  • Product Level – The explicitly defined product level (must be at least one).
  • Location Level – The explicitly defined location level (must be at least one).
