Profile Set Item Details graph data reference

The Profile Set Item Details grid displays a graphical representation of the data in six standard data series.

Series Description
Effective Profile Volume The volume per period using the profile curve listed (effective periods only)
Sample Profile Volume The Sales volume per period using the profile curve listed (sample periods only)
Sales History Quantity Actual sales history
Profile Volume Override (%) The Sales volume per period using the override adjustments
Profile Value The Sales volume per period using the profile curve
Applied Profile Value with Causal The Profile Value with Manual causal affect applied. The Manual causal affect is a percentage multiplier.

This value matches the Profile Forecast series found in the Level View within the Inventory Forecast module.
