Creating profile sets

A profile set is a collection of profiles and profiled items. Building a profile set is an important first step in creating and applying profiles.

  1. From the Execute tab, click Profiles.

  2. Click Manage on the Profile Set toolbar.

    The Profile Set Management window displays.

  3. Select a profile set group where a new profile is to be added and click New.

    The Add Profile Set window displays.

  4. Enter the name and description for the new profile set.

  5. Configure or change fields as required.

  6. Click Ok.

The new profile set is created.

Fields and descriptions

  • Name – The name for the profile set.
  • Description – The optional description for the profile set.
  • Group – The parent folder used to assist with organization of profile sets.

  • Filter – The filter which is applied to determine the items within the profile set (items to be profiled).

    This must be a static filter as any changes made to the filter causes the Profiles Module to exclude it from the sets (this is intended to maintain the integrity of the profile).

  • Security – The security settings will affect which users may or may not edit the profile.
