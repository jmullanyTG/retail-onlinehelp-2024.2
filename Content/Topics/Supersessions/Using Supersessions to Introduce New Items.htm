Using supersessions to introduce new items into stores

The following end-to-end process begins with creating supersessions for new items and proceeds all the way through to generating orders optimized for distribution centers and stores.

Step Task Example
1. Identify new items and create supersessions for them Supersessions overview
2. Define phase-in and phase-out dates for the new items Defining item lifecycle dates
3. Generate demand forecasts for the new items Example - Demand Forecast
4. Define the stocking strategy for the new items Example - Defining Stocking Strategy and Store Eligibility
5. Generate supplier and distribution orders for the new items Order manager overview

Replenishment

When the new items are in stores, their replenishment strategies are managed through the Replenishment process. For more information, see Replenishment Overview.
