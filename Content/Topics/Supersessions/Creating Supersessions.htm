Creating supersessions

Supersessions are used to create accurate demand forecasts for items with little or no history.

To create a supersession

Before a supersession is created, a filter must be added that includes the new and old items that are superseded.

  1. From the Execute tab, click Supersession.

  2. Click New.

  3. From the Filter dropdown menu, select the filter to use for the supersession.

  4. Click Ok.

    The Add supersession wizard dialog displays.

  5. Select the existing items to supersede.

  6. Click Next to continue.

  7. Select the new item that will replace the old item.

  8. Click Next.

    The wizard refreshes to display the Supersession Parameters window.

  9. Review or edit the supersession parameters.
  10. Click Next and select the sites where the supersession will be applied.

  11. Click Next to continue.
  12. Click Finish.

The supersession is created and displays on the Supersession Grid screen.

Demand forecasts can be manually generated for the new items or users can wait for the forecasts to be generated by the Daily Job Manager.

For more information, see Generating Demand Forecasts. For more information, refer to the Demand Forecasting User Guide.
