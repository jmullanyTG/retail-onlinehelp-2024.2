Currency exchange rates

The Currency Exchange Rates view is used to maintain both global exchange rates and time-based exchange rates from and to all currencies defined within ToolsGroup JustEnough. At a minimum, exchange rates should be maintained globally for a single base currency, being the home currency used in the assortment plans and as many other currencies as are assigned to the sites within the cluster sets in those plans.

This screen can be accessed by navigating to System Utilities > Views > Currency > Currency Exchange Rates.

Doing business in one currency only

If the company only does business in a single currency, then set a single conversion entry in the Currency From and Currency To fields with a Conversion Factor of 1 to indicate no change.

Parameters

Field Description
Currency From Sets the original currencies to be worked with.
Currency To Sets the currencies to be converted to.

Toolbar buttons

Button Description
Parameters Shows or hides the Parameters area of the view.
Execute Updates the view with a conversion factor grid for the Currency From and Currency To selections.

Default Currency Exchange Rates tab

This tab allows the user to manipulate the field layout by dragging and dropping fields in the Filter, Row, Column, and Data areas of the grid.

Functionality

  • Export: Exports and saves the Pivot Grid in different formats including Microsoft Excel and Adobe PDF.
  • Open: Opens the Pivot Grid in different formats including Microsoft Excel and Adobe PDF (this does not require saving prior to viewing in the desired format).
  • Graph: Displays a graph and any selected data series.

  • Fields: launches the Fields Customization dialog with two tabs. Each tab contains a search bar to more quickly locate fields. The fields that display is limited to only those containing the entered text.

    • Simple tab: Fields can be dragged and dropped between Filter, Row, Column, and Data areas of the Pivot Grid.

      Dragging fields to the None area hides them in the layout.

    • Advanced tab: Manipulate the layout by selecting the desired location of each field from the Pivot Area dropdowns next to each. An Index can be specified within each area to define the order in which the fields display.

Restoring layouts

Restore a custom layout by right-clicking anywhere in the Pivot Grid, choosing Restore Layout, and selecting the layout to restore.

  • Save: Saves changes made to conversion factors in the current view.
  • Cancel: Reverts all changes to conversion factors in the current view.
  • From Currency Description: Description of the From Currency.
  • From Currency Code: Unique system code for the From Currency.
  • To Currency Description: Description of the To Currency.
  • To Currency Code: Unique system code for the To Currency.

  • Conversion Factor: Sets the exchange rate from the currencies.

    Setting a Conversion Factor of 1 indicates no change.

Temporal Currency Exchange Rates tab

This tab allows users to enter exchange rate changes with specific start- and end dates. This is useful for planning for future economic fluctuations without having to alter the default exchange rates. Temporal Currency Exchange Rates supersede rates entered on the Default tab for the time frames in which they are active.

  • New: Adds a new line for entering a temporal exchange rate.
  • Delete: Deletes the selected temporal exchange rate lines.

    Users are not asked to confirm deletion of temporal exchange rates.

  • Fields: Opens the Customization dialog in which users can drag fields to or from the tab’s header area and this dialog to customize the view.

    • Custom layouts can be saved or restored by right-clicking in the Pivot Grid and selecting Save Layout or Restore Layout, respectively.

  • Save: Saves all new temporal exchange rate lines and changes made to existing lines.
  • Cancel: Removes all new temporal exchange rate lines and reverts all changes since the last save.
  • Currency From: The currency to convert from for the line.
  • Currency To: The currency to convert to for the line.
  • Start Date: The start date for the temporal currency conversion.

    Creating temporal currency conversions in the past does not have any effect on actualized periods.

  • End Date: (not required) The end date for the temporal currency conversion.

    Not setting an end date for a conversion means it will persist into the future until one is set or another conversion is entered in the future. This effectively makes the end date for the conversion the day prior to the start date of the new conversion

  • Conversion Factor: Sets the exchange rate from the From Currency to the To Currency.

    A Conversion Factor of .88 from USD (US Dollars) to GBP (UK Pounds) means a sale of $1,000 USD buys £888.
