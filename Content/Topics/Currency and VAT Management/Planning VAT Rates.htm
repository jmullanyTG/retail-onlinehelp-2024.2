Planning VAT rates

Some countries include a sales tax in the actual ticket prices of products sold. Others calculate it as a percent of the selling price during sales transactions. This tax is known as value added tax (VAT). In this view, VAT rates are managed at any level of any defined product hierarchy and for Site Groups 1-20, as defined during implementation.

All home ticket prices (Home AUT) within Assortment Planning are assumed to be inclusive of the Home Site’s VAT, and all blended ticket prices (Blended AUT) are assumed to be inclusive of the VAT of each site in the cluster, unless otherwise noted by an ex VAT descriptor in the metric name.

Even if VAT does not apply, a VAT rate of zero must be assigned to the highest level of all product hierarchies to be used in assortment plans. Because Site Groups are not explicitly tied to site hierarchies, this zero rate may need to be entered for more than the highest level Site Group, depending on how the groups are defined.

This screen can be accessed by navigating to System Utilities > Views > Maintenance > Planning VAT Rate.

Parameters

Field Description
Product Hierarchy The product hierarchy for which VAT rates will be managed.
Hierarchy Level The specific level of the selected Product Hierarchy for which VAT rates will be managed.

Changes entered for higher levels of the product hierarchy push down to the levels beneath.

For higher hierarchy levels with multiple VAT rates assigned to the levels beneath, VAT rate displays a straight average of the rates for those levels. Bold italic formatting as shown below is applied to make cases where this is true easily identifiable to users.
Site Group The Site Group level (1-20) for which to display and work with VAT rates in the view.

Changes entered for site groups corresponding to higher levels of the site hierarchy will be pushed down to lower level site groups.

For higher level site groups with multiple VAT Rates assigned to the levels beneath, VAT Rate displays a straight average of the rates for those levels. Bold italic formatting is used to indicate this.

Start Date The start of the time frame for which to display VAT rates in the view.
End Date The end of the time frame for which to display VAT rates in the view.

Toolbar buttons

Button Description
Parameters Toggles between displaying and hiding the Parameters area of the view
Execute Updates the view with all defined VAT rate entries for the intersection of the selected Hierarchy Level, Site Group, and time frame.

All entries overlapping any portion of the time frame specified by the start- and end dates display in the view.

Planning VAT Rates tab

Saving or restoring custom layouts

Custom layouts can be saved or restored by right-clicking in the Pivot Grid and selecting Save Layout or Restore Layout, respectively.

Start dates and end dates

The latest line for a given product level’s VAT rate always defaults to the end of the system time line (currently Dec 31, 2999) to always provide an active VAT rate for system calculations. When adding a new VAT rate line and setting a new VAT rate, the previous line changes its End Date automatically to just before the new line’s time frame begins.

The End Date for a line is currently Dec. 31, 2999. A new line has a start date of June 1, 2023 and an end date of Dec 31, 2023. The previous line’s End Date changes from Dec. 31, 2999 to May 31, 2023. The new line begins on June 1, 2023 as entered but its End Date changes to Dec 31, 2999 to provide VAT rates through the future.

Site group columns

The columns shown depends on the levels that exist within the selected Site Group.

Column Description
VAT Rate User editable decimal (0.0001) (non-editable for saved lines). Sets the VAT rate for the specified product level and time frame.
Site Count

(read-only). Displays the number of sites to which the VAT rate is applied. Updates after Save is clicked for new lines.

VAT rates cannot be set for newly entered lines until they have first been saved.

Toolbar buttons

Button Description

Delete

Removes the selected VAT rate lines from the current view.

There is no pop-up message requesting confirmation of deletion of VAT Rate lines.
Fields Opens the Customization dialog in which users can drag fields to or from the tab’s header area and the dialog to customize the view.
