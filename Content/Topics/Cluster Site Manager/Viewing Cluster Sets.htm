Viewing cluster sets

From the Cluster Set screen in the Site Clustering solution, users can calculate and view specific cluster sets in both detail and chart modes, make manual modifications to site cluster assignments, and arrange multiple cluster sets in the view for easy comparison.

Cluster sets selected to display appear as tabs with the cluster set name just below the toolbar.

To view a cluster set

  1. From the Management tab, click Site Clustering.

    The first time Site Clustering is opened, the Cluster Set Management window displays. Otherwise, the Cluster Set window displays.

  2. To view a cluster set document for a given cluster set, double-click the cluster site in the grid.

    The cluster set details display.

Header toolbar functionality

Button Description
Manage Launches the Cluster Set Management screen.
Save Saves changes.
Undo Undoes changes.
Calculate Calculates the cluster assignments for all stores included in the cluster set (based on the parameters and criteria specified during cluster set configuration) and populates the Cluster field with these assignments.

Multiple cluster sets can be calculated simultaneously using the Cluster Set Management screen. Click Allow Multiple and select the desired cluster sets or cluster set partitions. For more information, see Viewing a cluster set snapshot or template.
Configuration Launches the cluster set Configuration window.
Refresh Refreshes the current view.

Header fields

Field Description
Name The name of the cluster set being viewed.
Description The description of the cluster set being viewed.
Cluster Type The cluster set type. For more information on cluster set types, see Cluster Set Configuration.
Last Computed The date the cluster set was last calculated.
Header right-click menu options
Close Closes the cluster set document opened.
Close All But This Closes all open cluster set documents except the one the user opened. Only available when multiple documents are open.
Float Converts the document tab to a window that can be sized and positioned on the screen (or multiple screens).

Dragging the document header

The document header can be dragged to reposition it on the screen. After floating, click the document header and drag the document to the desired location. Right-click the document header and click Dock as Tabbed Document to return the document to its tabbed position. Hover over the directional buttons to see possible docked locations for the floated window.

New Vertical Tab Group: arranges all open documents vertically side by side. After selecting this option, right-click the document header again and select Move to Previous Tab Group to return to the last screen configuration.

New Horizontal Tab Group: arranges all open documents horizontally stacked. After selecting this option, right-click the document header again and select Move to Previous Tab Group to return to the last screen configuration.

Detail tab

The Detail tab displays the site codes and descriptions of all stores in the cluster set and their cluster assignments. Fields include:

Field Description
Name The site name and number.
Description The site description.
Cluster The system calculated cluster assignment for the site.
User Cluster Optional editable dropdown field. Sets the user override cluster assignment for the store.
Apply User Value Editable checkbox. Indicates if the system uses User Cluster in place of Cluster as the Effective Cluster. Default is checked.
Effective Cluster The effective cluster assignment for the store.
Financial Plan The MFP financial plan selected as the Financial Plan Attribute.

Chart tab

The Chart tab displays all stores in the cluster set as data points against X and Y axes for all Attributes defined in Cluster Set Configuration.

Field Description
X-Axis Attribute Selects an attribute from those defined in Cluster Set Configuration to plot on the X (horizontal) axis of the graph.
Y-Axis Attribute Selects an attribute from those defined in Cluster Set Configuration to plot on the Y (vertical) axis of the graph.
Legend Displays one color coded checkbox per cluster, corresponding to the colored data points that are assigned to that cluster on the chart. If unchecked, hides the related data points.
Data Points Colored dots and stars in the chart area: dots correspond to clusters of the same color in the legend, stars represent the centroids (K-Means types) or medoids (K-Medoids types) for the distance calculations used to assign stores to clusters. Align the cross hairs on the chart to a specific data point to display a pop-up of the store associated with that point, as well as the specific values associated with the X and Y axes at the left and bottom of the chart, respectively.

Show Labels: Toggles the display of data point labels for all points on the chart. Default is unchecked.
