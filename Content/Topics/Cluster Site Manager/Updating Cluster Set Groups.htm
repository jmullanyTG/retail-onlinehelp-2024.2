Updating cluster sets

To update a cluster set:

  1. From the Management tab, click Site Clustering.

    The Cluster Set Management window displays.

  2. Select the cluster set to update.

  3. Click Edit.

  4. Make the necessary changes in the Update Cluster Set window.
  5. Click Ok.
  6. Click Ok.
  7. Click Calculate on the Cluster Set window toolbar.

Toolbar functionality

Button Description
New Add a new cluster set group.
Edit Edit the selected cluster set group.
Delete Delete the selected cluster set group.

Deleting a group does not delete the cluster sets or templates in it.
