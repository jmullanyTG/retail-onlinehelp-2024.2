Cluster Set Snapshot Comparison report

The Cluster Set Snapshot Comparison report compares cluster set snapshots across a specified time frame.

To generate the Cluster Set Snapshot Comparison report

  1. Display a cluster set.

    For details, see Viewing cluster sets .

  2. From the Snapshot window at the bottom of the screen, click the Report button.

    The Time Frame dropdown menu displays.

  3. From the Time Frame dropdown menu, select a period start and end date to display.

  4. Click Execute.

    The Cluster Set Snapshot Comparison Report displays.

Toolbar functionality

Object Description
Time Frame Select the time frames that include the snapshots to compare.
Parameters Toggles displaying or hiding the Parameters area of the report.
Execute Generates the report and compares the snapshots selected in the Time Frame list.

Cluster Set Snapshot Comparison report

For each site, a snapshot classification is highlighted in orange when it is different than the classification for the snapshot to its left.

Column Description
Site Code The unique code for the site in the system.
Site Name The description for the site.
Classification Name The assigned classification for the site during the corresponding snapshot's effective time frame.
