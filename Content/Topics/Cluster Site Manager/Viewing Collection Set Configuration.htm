Viewing collection set configuration

Use the Cluster Set Configuration screen to set up the parameters used in the creation of cluster templates and sets, including the cluster type, the sites to be included, the number and descriptions of the actual clusters, the range of data to be used, the specific criteria to be considered in the clustering algorithm, and any partitioning to be used in the generation of multiple cluster sets.

Adding cluster collection sets

  1. From the Management tab, click Site Clustering.

    The Cluster Set window displays.

  2. Click the Manage button on the toolbar.

  3. Click the View Collection Set Configuration button.

    The Cluster Site View Collection Sets Maintenance window displays.

  4. Click Add to add a new row.
  5. Select the Cluster Entity Type and Collection Set from the respective dropdown menus.
  6. Click Ok.

Deleting cluster collection sets

  1. Click the View Collection Set Configuration button.

    The Cluster Site View Collection Sets Maintenance window displays.

  2. Select a cluster set and click Remove.

    The cluster collection set is deleted.

  3. Click Ok.

Undoing the last action

If a cluster collection set was added or deleted by mistake, click the Undo button in the Cluster Site View Collection Sets Maintenance window to undo the last action.
