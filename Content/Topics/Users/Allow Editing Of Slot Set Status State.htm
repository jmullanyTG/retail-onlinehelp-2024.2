Allow Editing Of Slot Set Status State

Displays the Page Status field providing access to modify the Slot Set Status State. When all Slot Sets have a Slot Set Status State of Complete, a Workflow transition is triggered. This functionality and its placement on the screen is the same whether accessed from the List or Layout.

This permission can be configured to be enabled or disabled during specific Event Workflow Status’ States.

Modifying the Slot Set Status State can modify access depending on the Event Workflow Slot Set Status State Blacklist configuration.

The User Right Allow Editing of Slot Set Header must also be granted.

Events Management with the user right

Without this user right, the Page Status field does not display.
