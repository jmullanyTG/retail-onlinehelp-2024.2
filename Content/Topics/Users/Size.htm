Size

The Size Similarity Manager view allows users to identify sizes that should be grouped together as one, specify a single name under which to unify and display them, and ultimately work with them as a single entity without the need to manually unify the relevant data points.

For more information, see Identifying sizes that must be grouped. refer to the Allocations User Guide.

The following User Rights are available:

  • Allow a user the ability to associate sizes and mark a similarity set active or not

  • Allow a user the ability to delete a size similarity set

  • Allow a user the ability to run the procedure that migrates size similarity sets
