Content Manager user rights

Content Manager user rights provide the ability to grant permissions to update text and images for products and product groups at the Master and Event levels. If content is updated for a product at any level, it can be pushed to other levels for the same product or where the product is included in a grouping of products.

For more information regarding individual functionality, see Content Manager overview.

From the Maintenance tab, click Users. Select a user and then an option in the left pane. This displays the User Information window in which permissions are configured.

User Right Functionality
Allow Access To Content Manager Access to the Content Manager.
Allow Editing Of Content Edit product content in Content Manager.
Allow Updating of Copy from Master to Promotions Save copy text from the product Master copy text to the copy text for one or more Promotions for the same product.
Allow Updating of Copy from Master to Events Save copy text from the product Master copy text to one or more Events with Promotions for the same product.
Allow Updating of Copy from Promotion to Master Save copy text from a Promotion to the Master copy text for the product.
Allow Updating of Copy from Promotion to Events Save copy text from a Promotion to the copy text for one or more other Events with Promotions for the same product.
Allow Updating of Copy from Event to Master Save copy text from an Event to the Master copy text for the same product.
Allow Updating of Copy from Event to Promotions Save copy text from an Event instance to the copy text for one or more Promotion’s for the same product.
Allow Updating of Copy from Event to Events Save copy text from one Event instance to the copy text for one or more other Events with Promotions for the same product.
Allow Updating of images from Master to Promotions Save images from the product Master images to the images for one or more Promotions for the same product.
Allow Updating of images from Master to Events Save images from the product Master images to one or more Events with Promotions for the same product.
Allow Updating of images from Promotion to Master Save images from a Promotion to the Master images for the product.
Allow Updating of images from Promotion to Events Save images from a Promotion to the images for one or more other Events with Promotions for the same product.
Allow Updating of images from Event to Master Save images from an Event to the Master images for the same product.
Allow Updating of images from Event to Promotions Save images from an Event instance to the images for one or more Promotion’s for the same product.
Allow Updating of images from Event to Events Save images from one Event instance to the images for one or more other Events with Promotions for the same product.
Config - Allow Access To Content Config Without this access the Configuration button does not display. Provides access to the Content Manager Configuration screen.
Config - Allow Creation Of Content Version Allows users to create, edit (add, remove fields) and delete content versions within the Content Manager Configuration screen.
Config - Allow Deleting Of Content Version
Config - Allow Editing Of Content Version
Config - Allow Editing Of Content Field Edit content fields within the Content Manager Configuration screen.
