User details

Users are created, cloned, modified, and deleted within the User Information window. Select Users in the Navigation menu and then choose the desired functionality to display the User Information window.

User Details tab

The User Details tab contains most of the general information common to user accounts such as login credentials and assigned filters. Assigned filters give Administrators the ability to segment sections of the company data to restrict users or streamline their individual workflows. Users may be assigned filters built on specific inventories, sites, item masters and many other filter types.

Cloning users

Cloning a user is a shortcut used when setting up new users. It is advisable to do this instead of creating a new user, as similar rights, roles and filters (the most time consuming part of user account setup) may be carried over.

User Details tab field descriptions

  • User Name – The user name as it appears in the main users window. This user name is the sign in credential for users.
  • User Description – A description of the user account. If it is a cloned user, the description reads as Cloned from _____ until it is changed (the blank is filled by the user account from which it was cloned).
  • Full Name – The actual name of the user. This is an optional field.
  • User Domain Name – If the user is a domain account and has been imported from active directory, the associated domain appears here.
  • Cube Role Name – If there is a SQL Server Analysis Services (SSAS) OLAP cube deployed on the database, ToolsGroup JustEnough will need to pass a cube role name to the OLAP cube to access information via the OLAP pivot grid widget. This role name will be stored by user.
  • Account is Disabled – A disabled account cannot log into the system, but it is still visible.
  • Inventory Assigned Filter – The default inventory level filter assigned to the specified user.
  • Site Assigned Filter - The default site filter assigned to the specified user.
  • Promotion Assigned Filter – The default promotion filter assigned to the specified user.
  • Event Assigned Filter – The default event filter assigned to the specified user.
  • Item Master Assigned Filter – The default item master level filter assigned to the specified user.
  • Supplier Assigned Filter – The default supplier filter assigned to the specified user.
  • Customer Assigned Filter – The default customer filter assigned to the specified user.
  • Customer Inventory Assigned Filter – The default filter for customer inventory (to support custom implementations of customer level forecasting only).
  • Item Master Group Assigned Filter – A default filter for item master groups, a sub-set of item master filters which can be used to further refine item master filtering within content manager.
  • Variant Assigned Filter – A default variant filter assigned to the specified user.
  • Variant Group Assigned Filter – A default filter for variant groups, a sub-set of variant filters which can be used to further refine variant filtering within content manager.

Contact Details tab

The Contact Details tab contains optional information related to users.

  • Telephone Number – The landline phone number for the user.
  • Cellphone Number – The cellular or mobile phone number for the user. May be used as a second phone number.
  • Fax Number – Fax number for the user.
  • Email Address – The email address for the user.
  • Address – Postal address for the user.
  • Postal Code – Postal code related to the physical address for the user.
