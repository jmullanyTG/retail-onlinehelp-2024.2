Workload Queue user rights

Access to functionality in Workload Queue depends on user rights.

User Right Functionality
Allow Access to Workload Queue Module Enables the Workload Queue icon on the Maintenance tab.
Allow Job Reprioritization and Removal Change a batch’s prioritization and removal from the Active list to be processed.
Allow Workload Queue Configuration Enables the Configuration button on the Workload Queue toolbar allowing users to configure threshold overrides and optimizations.
