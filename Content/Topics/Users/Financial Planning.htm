Merchandise Financial Planning User Rights

For more information regarding individual functionality, see The Financial Plan Maintenance window. refer to the Merchandise Financial Planning User Guide.

User Right Functionality
Access To Financial Planning The Financial Planning module.
Add Plan The New button on the Financial Plan Maintenance window.
Add Plan Branch The Branch button on the Financial Plan Maintenance window.
Add Plan Level Access to product and channel hierarchy selection in the Maintain Financial Plan window.
Add Plan Snapshot The New button on the Snapshots tab in the Financial Plan Maintenance and Maintain Financial Plan windows.
Approve Plan (All) The Approve button on the Financial Plan Maintenance window (when the user is not the owner of the selected branch plan).
Approve Plan (Owner) The Approve button on the Financial Plan Maintenance window (when the user is the owner of the selected branch plan).
Close Branch Plan (All) The Close button on the Financial Plan Maintenance window (when the user is not the owner of the selected branch plan).
Close Branch Plan (Owner) The Close button on the Financial Plan Maintenance window (when the user is the owner of the selected branch plan).
Close Master Plan (All) The Close button on the Financial Plan Maintenance window (when the user is not the owner of the selected master plan).
Close Master Plan (Owner) The Close button on the Financial Plan Maintenance window (when the user is the owner of the selected master plan).
Delete Plan Snapshot (All) The Delete button on the Snapshots tab in the Financial Plan Maintenance and Maintain Financial Plan windows (when the selected snapshot was not created by the user).
Delete Plan Snapshot (Owner) The Delete button on the Snapshots tab in the Financial Plan Maintenance and Maintain Financial Plan windows (when the selected snapshot was created by the user).
Edit Branch Plan (All) The Edit button on the Financial Plan Maintenance window (when the user is not the owner of the selected branch plan).
Edit Branch Plan (Owner) The Edit button on the Financial Plan Maintenance window (when the user is the owner of the selected branch plan).
Edit Master Plan (All) The Edit button on the Financial Plan Maintenance window (when the user is not the owner of the selected master plan).
Edit Master Plan (Owner) The Edit button on the Financial Plan Maintenance window (when the user is the owner of the selected master plan).
Initialize Branch Plan (All) The Initialize button on the Financial Plan Maintenance window (when the user is not the owner of the selected branch plan).
Initialize Branch Plan (Owner) The Initialize button on the Financial Plan Maintenance window (when the user is the owner of the selected branch plan).
Initialize Master Plan (All) The Initialize button on the Financial Plan Maintenance window (when the user is not the owner of the selected master plan).
Initialize Master Plan (Owner) The Initialize button on the Financial Plan Maintenance window (when the user is the owner of the selected master plan).
Maintain Merge Measure Sets The Merge Measure Sets button on the Configuration sub-menu on the Financial Plan Maintenance window.
Maintain Plan Filters Add and edit filters.
Manage Global Groups The Global Groups button on the Configuration sub-menu on the Financial Plan Maintenance window.
Manage Plan User Access The User Access button on the Configuration sub-menu on the Financial Plan Maintenance window and the User Access tab in the Maintain Financial Plan window.
Merge Plan Branch To Branch (All) The Merge button on the Financial Plan Maintenance window (when the user is the owner of the selected branch plan).
Merge Plan Branch To Branch (Owner) The Merge button on the Financial Plan Maintenance window (when the user is the owner of the selected branch plan).
Merge Plan Branch To Master The Merge button on the Financial Plan Maintenance window (when the user is the owner of the selected branch plan).
Modify View Collection Sets The View Collection tab in the Maintain Financial Plan window.
Modify Active Flag The ability to edit the Active Flag in the product and channel hierarchies in the Maintain Financial Plan window.
Rebalance LP or Hybrid Plan Any to Branch (All) The Rebalance button on the Financial Plan Maintenance window (when the user is not the owner of the selected branch plan).
Rebalance LP or Hybrid Plan Any To Branch (Owner) The Rebalance button on the Financial Plan Maintenance window (when the user is not the owner of the selected branch plan).
Re-Initialize Plan (All) The Re-Initialize button on the Financial Plan Maintenance window (when the user is not the owner of the selected plan).
Re-Initialize Plan (Owner) The Re-Initialize button on the Financial Plan Maintenance window (when the user is the owner of the selected plan).
Reject Plan (All) The Reject button on the Financial Plan Maintenance window (when the user is not the owner of the selected branch plan).
Reject Plan (Owner) The Reject button on the Financial Plan Maintenance window (when the user is the owner of the selected branch plan).
Re-Seed Branch Plan (All) The Re-Seed button on the Financial Plan Maintenance window (when the user is not the owner of the selected branch plan).
Re-Seed Branch Plan (Owner) The Re-Seed button on the Financial Plan Maintenance window (when the user is the owner of the selected branch plan).
Re-Seed Master Plan (All) The Re-Seed button on the Financial Plan Maintenance window (when the user is not the owner of the selected branch plan).
Re-Seed Master Plan (Owner) The Re-Seed button on the Financial Plan Maintenance window (when the user is the owner of the selected branch plan).
Submit Plan for Approval (All) The Submit button on the Financial Plan Maintenance window (when the user is not the owner of the selected branch plan).
Submit Plan for Approval (Owner) The Submit button on the Financial Plan Maintenance window (when the user is the owner of the selected branch plan).
