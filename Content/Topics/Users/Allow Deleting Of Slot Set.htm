Allow Deleting Of Slot Set

Provides access to delete a Slot Set from an Event. All Promotions that were placed on the Slot Set are removed but not deleted. Event specific content associated with those Promotions when they were on the Slot Set are deleted.

This permission can be configured to be enabled or disabled during specific Event Workflow Status’ States

This functionality and its placement on the screen is the same whether accessed from the List or Layout views.

Events Management with the user right

The Delete button does not display without this user right.
