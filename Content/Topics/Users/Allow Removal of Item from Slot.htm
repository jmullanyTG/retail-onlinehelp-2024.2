Allow Removal Of Item From Slot

Provides access to remove Promotions from an Event Slot or the ability to unassign a Promotion, moving it to Slot Position 0.

This permission may be configured to be enabled or disabled during specific Event Workflow Status’ States

This functionality and its placement on the screen is the same whether accessed from the List or Layout

Events Management with the user right

Without this user right, the options within the red square do not display.
