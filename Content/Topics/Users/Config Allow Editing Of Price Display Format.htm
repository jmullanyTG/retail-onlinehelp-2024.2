Config - Allow Editing Of Price Display Format

Price Display Formats are created automatically when a new Promotion Strategy is configured. As such, Price Display Formats cannot be created or deleted manually. This User Right provides access to edit a Price Display Format including its dimensions, and placement of text and Promotion fields.

Without this right, the Edit button does not display.

For more information, see Strategies. refer to the Promotion User Guide.

This User Right should only be granted to trusted Super Users
