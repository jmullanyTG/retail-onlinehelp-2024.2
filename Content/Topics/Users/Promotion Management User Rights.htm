Promotion Management

Promotion Management User Rights can be configured by clicking Users from the Maintenance tab, selecting a user or role and clicking Assign Rights. The > and < buttons can be used to add or remove rights for the user or role.

For more information regarding individual functionality, see Promotion Management overview.

User Rights can be configured for a Role by selecting a user or role, and clicking Modify.

Some User Rights may be incorporated into Promotion Workflow Status States, while others are not affected by Promotion Status. User Rights need to be granted at a global level, but can be temporarily taken away by Promotion Workflow Status State using the User Rights Blacklist.

For more information to understand blacklisting User Rights during specific Promotion Status States, see Status state permission blacklist.
