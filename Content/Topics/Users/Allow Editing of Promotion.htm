Allow Editing Of Promotion

Provides access to edit the Promotion Price Cluster Sites, Price Cluster Promotion Strategies, clone the Promotion, modify Price details, create and modify Bids, add and modify Companion Items and view the Analysis screen.

Without this User Right, it is still possible to view Promotion information in the Price Cluster Items, Price Details, Bids, Companion Items and Analysis screens by double clicking on the Promotion Name from the Promotion List screen

Some fields may appear to be editable but are not, if this User Right has not been granted
