Allow Add Promotion To Event

Provides access to add an existing Promotion that has effective dates that overlap the Event’s Sale Dates. If this User Right is not granted, users can still create a new Promotion from the Event if they have the Allow Creation of Promotion from Event User Right.

This permission can be configured to be enabled or disabled during specific Event Workflow Status’ States

Events Management with the user right

Without this user right, the buttons do not display.
