Creating and maintaining users

Selecting Create launches the User Information window. Editing user accounts is performed in the User Information window The functions are exactly the same as they are when creating a user.

Administrators must populate the user name and password fields on the first tab at minimum. It is suggested that filters be assigned (if applicable) at the time of account creation.

Adding the assigned filters is only possible if those filters have already been created using the Filters module. Contact details are optional.
