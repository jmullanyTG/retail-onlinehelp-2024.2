Allow Creation Of Price Cluster

Provides access to create Site Clusters in the Promotion allowing users to specify a different Promotion Strategy or pricing at the Site Cluster level. The Allow Editing of Price Cluster User Right must also be granted to provide the ability to add and remove Sites to newly created Site Clusters.
