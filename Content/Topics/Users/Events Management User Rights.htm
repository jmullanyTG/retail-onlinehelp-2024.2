Events management user rights

For more information regarding individual functionality, see Events overview.

Events Management User Rights are configured by clicking Users from the Maintenance tab, selecting a user in the right pane and clicking Assign Rights. This displays the User Information window in which the > and < buttons are used to add or remove rights.

User Rights are configured for a Role, by selecting Roles > Modify. This opens the Role Information window used to modify the Rights related to a Role.

Some User Rights may be incorporated into Event Workflow Status States, while others are not affected by Event Status. User Rights need to be granted at a global level, but can be temporarily taken away by Event Workflow Status State using the User Rights Blacklist.

For more information regarding blacklisting, see Status state permission blacklist. refer to the Workflow Manager User Guide.
