Allow Access To Slot Set Layout View

Provides access to the Layout and Versions screens to view the selected Slot Set as a visual representation of Promotions and Content. From the Layout screen, edit the Slot layout, add, create, delete and move Promotions, view and edit Bids and modify Content. Access to these tasks depends on assigned User Rights and the User Rights Blacklist for the Event Workflow Status State.

The Versions screen provides a preview of the Slot Set color versions which are determined by unique combinations of promotional content and pricing assigned to distribution points (e.g. geographies, demographics, segmentations, etc.)

Events Management with the user right

Without this user right, the Layout and Versions buttons do not display.
