Variant Manager

The User Rights for Variant Manager are:

  • Allow Access to Rate Of Sale Bulk Copy Variants

  • Allow Access To Variant Manager

  • Allow Maintaining Of Prepacks

  • Allow Maintaining Of Views For Variant Manager

  • Allow Rate Of Sale Configurations for Variants

  • Allow Save Changes In Variant Manager Data Grid

  • Allow Update Access to Rate Of Sale Curve Type and Curve

  • Allow Update Access to Rate Of Sale Inventory Phase Dates

  • Allow Update Access to Rate Of Sale Method Sets

  • Allow Update Access to Rate Of Sale Product Models

  • Allow Update Access to Rate Of Sale Value
