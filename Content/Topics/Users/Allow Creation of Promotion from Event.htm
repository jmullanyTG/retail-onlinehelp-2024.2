Allow Creation Of Promotion from Event

Provides access to create a Promotion from an Event Slot regardless of whether users were granted the User Right to create Promotions in the Promotion solution.

This permission can be configured to be enabled or disabled during specific Event Workflow Status’ States

This functionality and its placement on the screen is the same whether accessed from the List or Layout views.

Events Management with the user right

The New button does not display without this user right.
