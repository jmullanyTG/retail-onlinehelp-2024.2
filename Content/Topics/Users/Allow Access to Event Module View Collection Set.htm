Allow Access To Event Workbench Tab

Provides access to the Event Workbench tab in the Events solution which may or may not be available. Functionality within this screen varies depending on the configuration. Without this access, the Event Workbench tab does not display.

System policy

The following system policy needs to be configured:

System policy Description
Event Module View Collection Set Assign the View Collection Set ID to be displayed in the third tab of Event Manager by its ID. If this is not set to a valid ID, a blank screen would be displayed for this customized tab.
