Choosing items and data to display

The items and data to display in Inventory Manager can be chosen and changed. The configurable views can be used to create custom reports. Views are created and opened in separate tabs within the Inventory Manager module.

Choosing items to review

  1. Navigate to Management > Inventory Manager or click Inventory Manager Document.

    The Items dialog displays.

  2. Click the dropdown arrow in the Filter field to select items for review.

  3. Click Ok.
  4. In the Choose Fields dialog window:
    1. Select a view from the Views pane.

    2. Select fields in the Fields in View pane.

      The search box near the top of the Choose Fields window can be used to filter the fields and find specific measures.

      Fields are organized by category and may be included or excluded by category or individually using the arrows.

  5. Click Ok.

Choosing additional data to review

To review additional data:

  1. Within an Inventory Manager Document, click Choose.

  2. Select fields in the Fields in View pane.

    The search box near the top of the Choose Fields window can be used to filter the fields and find specific measures.

  3. Click Ok.

    The view automatically refreshes.

Identifying site types

Select Choose on the toolbar and include Site Type from the Site Type metadata group.

The Site Type column displays Account for wholesale accounts. Detailed forecast data can be viewed by navigating to the Forecast Viewer blade and reviewing the Account Forecast series.
