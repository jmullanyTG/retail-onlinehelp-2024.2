Setting the Consider Demand Transference option for an item

  1. From the Management tab, click Inventory Manager.
  2. If the Items dialog box does not appear, click Inventory Manager Document on the toolbar to open it.
  3. In the Items dialog box, select the filter containing the items to manage.
  4. In the Choose Fields dialog box, add the Consider Demand Transference field.
  5. In each product row of the inventory manager document, select the Consider Demand Transference checkbox to toggle including or ignoring demand transfers when calculating replenishment for that item.
  6. Click Save to save changes.

Additional demand transfer configuration

To include demand transfers in an item’s replenishment strategy, perform the following configuration tasks:

Module Task
Job Manager Ensure the (DF) Calculate Fulfillment Split Factor task is added to Standardized jobs. Standardized jobs.

This task must run after DF tasks but before RP tasks.
Inventory Policy If the item is sold at a showroom, configure the showroom Demand Transference options for the item's related inventory policy.

See, Configuring demand transference for showroom items.

Once configured, demand transfers are included in order recommendations run by the Replenishment engine.

After orders have been generated, review the split factors for an item-site combination in the Forecast Fulfillment Split Factors tab in the Inventory Manager's Detail pane. For more information, see Forecast Fulfillment Split Factors.
