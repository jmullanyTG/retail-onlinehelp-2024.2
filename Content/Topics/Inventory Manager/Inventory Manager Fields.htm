Inventory metadata by filter category

Users are prompted to select a filter upon entering the Inventory Manager. This can be a static filter for a select set of items or a dynamic filter capable of returning items which match the specified criteria.

After selecting filter, users are prompted to choose the fields to return. These are organized by category and can be included or excluded as needed. Also, the search bar can be used to search for applicable fields.

Once the view is created, fields can be added or removed by clicking the Choose button.
