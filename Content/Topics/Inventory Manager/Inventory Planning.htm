Inventory planning

Field Description
Age Age in days of an item from first stocked date
Back Order Ratio Used in replenishment logic. Estimation of number of customers who will place a back order.
Bin Items In Use Items being used. For example, in a manufacturing plant.  This is used for Bin Level calculations together with Bin Mean Time Between Failure values.  These values need to be imported from the transaction system.
Bin Meantime Between Failure Days The (mean) time in days before failure is likely to occur.  This is used for Bin Level calculations together with Items in Use values. 
Creation Date Imported from transaction system date or today – imported date
Criticality Description Criticalities are user-applied indicators that are used to differentiate the functional importance of products such as a low cost item that is critical to the sale of a high cost item, but does not have a significant investment value.  Criticalities can be used to set bin levels by service level per criticality and minimum safety stock values per criticality.  Update during data import or by making the field editable.
