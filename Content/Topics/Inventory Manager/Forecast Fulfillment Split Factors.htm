Forecast Fulfillment Split Factors

The Forecast Fulfillment Split Factors tab displays the calculated fulfillment split factors for the row selected in the Inventory Manager document.

For more information, see split factors.

For split factor data to be visible:

  • the item must have its Consider Demand Transference option selected.
  • replenishment orders for this item must have been generated by Order Manager after its Consider Demand Transference option was selected.

Parameters

Select the Parameters button to display the following options:

Parameter Description
Show Zero Values When checked, Inventory Manager includes destination sites missing a split factor. Uncheck to exclude destination sites missing a split factor.
Is Source Site When checked, all the sites that source from this site are displayed. If unchecked, the site shows as a destination site.
