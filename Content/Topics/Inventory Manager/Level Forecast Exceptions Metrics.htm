Level forecast exceptions

The following metrics are available in the Level Forecast Exceptions group. These metrics can be selected or removed through the Choose Fields dialog.

The metrics prefixed with Level are the same calculation as for Inventory Forecast Exceptions, except that they are calculated using the inventory level results generated by the level forecast, using the pre-configured Level Forecast Hierarchy.

System configuration must be done before creating level forecasts.

Metric Definition
Forecast Level Code An imported text value at the different levels and concatenated to create a unique code at the level
Forecast Level Description An imported text value at the different levels and concatenated to create a description
Forecast Level ID A system generated unique integer value attached to the level
Level Forecast Current Consumption Ratio (Current Forecast Quantity - Current Rate Sales) / Current Rate Sales
Level Forecast Current Forecast Quantity Forecast Per Day for current week
Level Forecast Current Rate Sales Sales Per Day for current week
Level Forecast Demand History Total Sales of past [PeriodCount] periods (Excluding this week)
Level Forecast Forecast LY Sales Ratio (Sum of effective forecast for [PeriodCount] future periods(Excluding this period) - DemandHistory ) / DemandHistory
Level Forecast Forecast Sales Ratio (Sum of effective forecast for [PeriodCount] future periods(Excluding this period) - LY DemandHistory ) / LY DemandHistory
Level Forecast Is Current Consumption Ratio Error Current Consumption Larger than threshold
Level Forecast Is Forecast LY Sales Ratio Error Forecast Error LY larger than threshold
Level Forecast Is Forecast Sales Ratio Error Forecast Error larger than threshold
Level Forecast Is MAPE Error Method fit lower than the threshold
Level Forecast LY Demand History Sum of current and [PeriodCount-1] Last Year
Level Forecast MAPE Mean Absolute Percentage Error (Forecast Method Fit)
Level Forecast Periods Forecast Exceptions calculated by how many periods (Configuration possible)
Level Forecast Reviewed Reviewed Button in Forecast Manager
Level Forecast Reviewed Reason Autopopulated by Exception (editable column)
Level Forecast Sales Spike Factor Relative Deviation from the mean for the past year
Level Forecast Total Forecast Total of [PeriodCount] forecast excluding current
Level Forecast Zero Forecast Total Forecast is Zero
