Forecast Viewer

The Forecast Viewer tab displays identical information to that which may be found in the Forecasts module. Users can configure the colors of the various available series within the graph selecting a color palette which is most appealing and accessible for them.

Additional information is found within the Forecasts module online help.

Allocation versus replenishment forecasts

The Forecast Viewer tab displays allocation- or replenishment-related forecast data depending on whether the selected item is managed by the Allocations module or the Demand Forecasting/Inventory Planning/Replenishment suite (Replenishment, for short).

If both the Allocations module and the Replenishment suite are enabled, the Allocation Strategy Selection field in the Allocation Strategy screen sets whether an item is managed by Allocations or Replenishment.

The Execution Process column can be added to the Inventory Manager view to see which process manages each item. Its read-only values are Allocations engine or Replenishment engine.

Generating a forecast for a selected item

  • To generate a forecast for a selected item, click Forecast.

Including or excluding forecasting methods

The locking process performed on items prevents the items from being re-forecasted and from using a different method until the lock expires.

Setting a lock

  1. Click the Lock icon to open the Forecast Task dialog.

  2. Select the checkboxes corresponding to the forecast methods to be locked.
  3. Set the lock expiration date in the Lock Expiration field.
  4. Click Ok.

Status bar

A status bar is available in both the Inventory and Level Forecast views with additional metadata based on the row selection or cells within a row. Certain cells within a row or a whole row can be highlighted to view the sum, average, count, minimum, and maximum values.

To display the status bar, right-click in the grid and select the Status Bar menu option.

Toolbar functionality

The toolbar contains the following functionality.

Button Description
Save Saves changes made to the forecast.
Undo Enabled when changes are made to the forecast. Click this button to undo the changes.
Change All Changes multiple lines (either rows or columns) within the grid. They can be set to a specified value, increased or decreased, copied from another column or row, or split across other columns based on a percent contribution.
Forecast Generates a forecast for a selected item.
Lock, Unlock
  • Lock. Excludes certain forecasting methods from being run for the inventory item. A lock expiration can be set to limit the period of time for which the lock is intended to cover.
  • Unlock. Displays only if a forecast has a lock applied. Allows users to remove a method lock.
Reviewed As part of a business flow process, displays that a particular forecast for an item has been approved. Clicking Reviewed populates the current date in the Review Date field.
Notes Launches the Notes Viewer to add information related to the particular inventory item as it relates to the forecast.

Causals Launches the Maintain Item Causal Links window letting users include or exclude any causals which have been created in the Causals module.
Settings Launches the Visible Bucket Period Limits dialog. This determines the number of periods which display in both the graph and grid sections of the forecast viewer. Depending on the period type (daily, weekly, or monthly) the visible bucket period limits can be adjusted to take a bigger picture look at the inventory item, or narrow the focus if so desired.
