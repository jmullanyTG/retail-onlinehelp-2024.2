Inventory sales history information

The following metadata (columns) available to Inventory Manager and Order Manager provide visibility to Last Day Sale units for each style color size and the sum of Last 7 Rolling Days Sales units for each style color size. This metadata helps ensure the right number of units are shipped.

The metadata can be selected from the metadata group Inventory Sales History Information. The columns display with the following labels:

  • Last 3 Weeks Average Returns Revenue
  • Last 3 Weeks Average Returns Units
  • Last 3 Weeks Average Sales Revenue
  • Last 3 Weeks Average Sales Units
  • Last 7 Days Returns
  • Last 7 Days Sales
  • Last 14 Days Returns
  • Last 14 Days Sales
  • Last 21 Days Returns
  • Last 21 Days Sales
  • Last 28 Days Returns
  • Last 28 Days Sales
  • Last Week Returns Revenue
  • Last Week Sales Revenue
  • Yesterday Returns
  • Yesterday Sales

Additionally, a new view labeled Sales History by Type - Validation can be added to Inventory Manager. For more information, see Views Overview. refer to the Views User Guide.
