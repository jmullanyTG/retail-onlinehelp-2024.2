Status information

Status information provides a high level overview of the pertinent information related to the inventory on a daily basis. Many of the standard exception filters return information requiring review such as potential stock outs, shortfalls, excess stock, or orders which fall outside of the normal cycle.

Allocation versus replenishment status

The Status Information tab displays allocation or replenishment related information depending on whether the selected item is managed by the Allocations module or the Demand Forecasting/Inventory Planning/Replenishment suite ("Replenishment", for short).

If both the Allocations module and the Replenishment suite are enabled, the Allocation Strategy Selection field in the Allocation Strategy screen sets whether an item is managed by Allocations or Replenishment.

The Execution Process column can be added to the Inventory Manager view to see which process manages each item. Its read-only values are Allocations engine or Replenishment engine.
