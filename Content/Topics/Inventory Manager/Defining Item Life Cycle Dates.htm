Defining item life cycle dates

Phase-in and phase-out dates can be defined for each item. These dates must match the dates an item will first and last appear on shelves in store locations. These dates are used to optimize demand forecasts, purchase orders, and other related features for the item.

To define the life cycle date for an item

  1. From the Management tab, click Inventory Manager.

    The Items window displays.

  2. From the Filter dropdown menu, select the filter that contains the items to manage, and click Ok.

    The Choose Fields window displays.

    If the Items dialog box does not automatically appear, click Inventory Item Management on the toolbar.

  3. Select or create a view that includes the following:

    • Inventory Planning: Phase In Date
    • Inventory Planning: Phase Out Date

    • fields to identify the items to manage.

  4. Click Ok.
  5. In the Inventory Manager Document window, filter the columns to locate the items and/or channels in need of phase-in and/or phase-out dates.
  6. Set the dates:
    1. for a single item by clicking in its Phase In Date and Phase Out Date fields and selecting the desired dates.
    1. for multiple items:
      1. filter columns until only items with the same desired dates appear in the grid.
      2. right-click in an item row and select Select All.
      3. right-click in an item's Phase In Date field and select Multiple Changes.
      4. select the desired phase-in date for the items.

      5. right-click in an item's Phase Out Date field and select Multiple Changes.
      6. Select the desired phase-out date for the items.
  7. When all phase-in and phase-out dates have been entered, click Save on the toolbar to save the dates to the items.
