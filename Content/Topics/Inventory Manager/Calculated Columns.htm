Adding custom columns

Custom columns can be added to any Inventory Manager view to aggregate, average, or otherwise calculate any appropriate data.

To create a custom column:

  1. Click Calculated on the toolbar and then New.

  2. Name the new column and choose the data type.

  3. (Optional) Select a format string.

    Format strings determine how the data displays. The field can be left blank. Format string entries vary depending on the data type. Examples include:

    • P2 – A percentage with two decimal places
    • F3 – A fixed point decimal with three decimal places
    • C – A currency
    • D6 – A decimal with six decimal places

  4. Click Ok.

  5. Use the Expression Editor to build an Expression as required.

    Expressions can be built using the Expression editor in a similar manner to the function bar in spreadsheet applications.

    Clicking on fields brings up the list of columns in the current view. Advanced functions or mathematical operators can be used to build the column output.

  6. Click Ok.

    7. The newly created column displays in the grid.
