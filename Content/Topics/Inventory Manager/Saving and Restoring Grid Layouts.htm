Saving and restoring grid layouts

Fields and layouts can be saved for future use after constructing a report. Previously saved columns and layouts can be retrieved using the Choose Fields dialog.

  1. First ensure that there are no filters active on the grid by checking that there is no writing in the headers and no items selected in the little funnels.

    If a layout with filters applied is saved, the behavior below repeats each time the layout is opened.

  2. Right-click anywhere on the data and select Save Layout.

  3. Give the Layout a name and then check or uncheck the default box depending on if this layout to be the default layout.

    Most of the time, choose Private unless creating a layout that others will use.
