Order document creation

The Order Document Creation task automatically generates order documents as part of a scheduled job. The task uses the same parameters as for manually creating an order document in Order Manager. As many Inventory Order Document Generation tasks as is needed can be created and scheduled.

Task type

Inventory Order Document Generation Task

Task parameters

Parameters listed in italics are dependent on the value selected in the Source parameter.

Parameter Description
Name The task name.
Description The task description.
Progress Type What to do if an error occurs when processing this task:
  • Report Error And Continue
  • Report Error And Halt
Time Out (Seconds) The timeout period.
Retry Count The number of retry attempts to make if a timeout occurs.
Retry Interval The time to wait in between retry attempts.
Task Enabled Whether this task is active or disabled.
  • When selected, the task is enabled and runs if manually executed or part of a scheduled job.
  • When not selected, this task is disabled and will not run even if manually executed or part of a scheduled job.
Source The source for the order:
  • Direct Supplier
  • Supplier
  • Distribution
  • From Strategy Allocation
  • From Scenario Allocation
Workflow The workflow to follow for orders generated by this task.
Owner The owner of orders generated by this task.
Security Whether this task is public, private, or shared with select others.
  • Private: the task is only visible to the logged-in user.
  • Public: the task is visible to all users.
  • Shared: the task is visible only to selected roles and users.
Filter The item filter to use for orders generated by this task. (Direct Supplier, Supplier, and Distribution-sourced orders only.)
Item Filter The item filter to use for orders generated by this task. (From Strategy Allocation sourced orders only.)
Site Filter The item filter to use for orders generated by this task. (From Strategy Allocation sourced orders only.)
Scenario Plan The scenario plan to use for orders generated by this task. (From Scenario Allocation sourced orders only.)
Supply Chain The supply chain to use for orders generated by this task. (Not used by From Scenario Allocation sources orders.)
Today's Orders This Week's Orders Future Recommended Orders
The order horizon to use for orders generated by this task.
Include Zero Orders Select to enable this option for orders generated by this task.
Exclude Uploaded Orders Select to enable this option for orders generated by this task.
Allow Macro Execution Select to run a macro for orders generated by this task.
Allow Inventory Partition Function Select to split orders generated by this task into separate documents according to the values of a specified field.

When selected, choose the field to split orders by in the Partition By Field parameter.

To split the order by supplier, partition by the Supplier Name field. The system then creates a separate order document for each supplier in the generated orders.
