Inventory supply chain resolution task

The Inventory Supply Chain Resolution task reprocesses supply chains when added to nightly or weekly jobs.

Because supply chain overrides are filter dependent, run this task in Job Manager:

  • after imports and filter-rebuild tasks, and
  • before replenish and order tasks.

Task parameters

Parameter Description
Name The task name.
Description The task description.
Progress Type What to do if an error occurs when processing this task:
  • Report Error And Continue
  • Report Error And Halt
Time Out (Seconds) The timeout period.
Retry Count The number of retry attempts to make if a timeout occurs.
Retry Interval The time to wait in between retry attempts.
Task Enabled Whether this task is active or disabled. If:
  • selected, the task is enabled and runs if manually executed or part of a scheduled job.
  • not selected, this task is disabled and does not run even if manually executed or part of a scheduled job.
Work Item Filter Type The type of supply chains this task processes:
  • All supply chains
  • Only supply chains with exceptions
  • Specified supply chains
Supply Chain(s) Visible only when Specified Supply Chains is selected as the Work Item Filter Type value. Lists all applicable supply chains. Selected supply chains are processed by this task.
