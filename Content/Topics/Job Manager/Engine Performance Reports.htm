Engine performance reports

Job Manager task (Weekly) Generate Engine Reports can be configured to send Engine Performance Reports for any engine task longer than the time specified in the system policy Performance Reporting Min Run Inclusion Time (Module: Reporting). The default is 600 seconds.

The reports include:

  • Progress Graphs
    • Progress
    • Input Queues
    • Concurrent Processing
    • Concurrent Processing Details
  • Hardware Graphs
    • Memory
    • Processor
  • Details
    • Progress Details
    • Input Queues Details
    • Endpoint Metrics
    • System Policies
    • Nodes Configuration
