Assortment Promotions Locations window

The Assortment Promotions Locations window is displayed by clicking in the Location Exceptions field in the Promotion Application section in the Assortment Promotion Definitions tab in the Promotions view.

Grid columns

Column Description
Site Hierarchy Columns The description for the Site Hierarchy level of the sites listed in the corresponding band. The columns shown depend on the structure of the Site Hierarchy assigned to the current assortment plan.
Cluster Set The name of the cluster set containing the sites listed in the corresponding band.
Cluster The name of the cluster containing the sites listed in the corresponding band.
Location ID The unique system identifier for the site.
Location Description The description for the site.
Placeholders Assorted to Site Shows how many placeholders that are part of a promotion are actually assorted to each site.
Include? Whether the site will participate in the promotion application.
Cluster % Contribution The site’s percent contribution to the cluster to which it is assigned.

Used in the calculation of the actual promotional index applied and adjustments to rate of sale (if Hold Sales is not checked). Calculated from the Location % Contribution tables for the relevant product hierarchy levels.

For the purpose of determining the effect of excluding certain sites from promotions, location % contribution is calculated by period rather than in total for the entire promotion to yield the most accurate result possible.

Toolbar buttons

Button Description
Export Exports and saves the pivot grid in many common formats including Microsoft Excel and Adobe PDF.
Open Opens the grid in many common formats including Microsoft Excel and Adobe PDF (does not require saving prior to viewing in the desired format).
Graph Graphs the selected data in the view.
Fields Allows the user to customize the field layout for the view. Custom layouts can be saved and restored by right-clicking in the grid and selecting Save Layout or Restore Layout, respectively.
Save Saves changes made to the Include? field.
Cancel Undoes changes made to the Include? field.
