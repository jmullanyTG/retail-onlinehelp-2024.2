Assortment Plan Defaults By Time tab

The Assortment Plan Defaults By Time tab is used to make adjustments to Minimum Presentation defaults across time by product hierarchy level to account for seasonal fluctuations in store inventory needs. Values entered in this tab override the Minimum Presentation value set in the Assortment Plan Defaults tab for the corresponding level of the product hierarchy for the periods in which overrides are made.

This tab features standard Pivot Grid functionality allowing users to manipulate the field layout by dragging and dropping fields in the Filter, Row, Column, and Data areas of the grid.

Aggregation

Time based fields (Year, Half, Quarter, Month, and Week) can be removed from the grid to allow entry of Minimum Presentation for larger time frames at once. When viewing Minimum Presentation at aggregate time levels, the system will display the average across all periods beneath the level being viewed.

Entering minimum presentation at aggregate time levels

When entering minimum presentation at aggregate time levels, the system will push down the entered value to all periods beneath.

Accessing the Assortment Plan Defaults By Time tab

  • In the Assortment Planning module, click Manage and select Defaults on the Manage Assortment Plans screen.

Tab parameters

Window of Time: Sets the time range to display in the Assortment Plan Defaults By Time tab.

Toolbar options

Graph: Displays the data series selected by the user in a standard Pivot Grid graph.

Fields: Launches the Pivot Grid Fields Customization dialog with two tabs: Simple and Advanced. Each tab contains a search bar to more quickly locate the fields to manipulate. The fields shown in the view is limited to only those containing the text entered.

Dragging fields to the None Area hides them in the layout.

  • Simple: fields can be dragged and dropped between Filter, Row, Column, and Data areas of the Pivot Grid to manipulate the current layout
  • Advanced: Manipulate the layout here by selecting the desired location of each field from the Pivot Area dropdowns next to each. Within each area, an Index can be specified to define the order in which the fields display.

Field Description
Name A name for the new layout, or select an existing layout using the dropdown menu
Description An optional description for the layout.
Default Check to make the layout being saved the default layout that loads when the Manage Assortment Defaults dialog is launched.
Private/Public/Shared Whether the current layout is only available for the current user, for all users, or for specified users only.

When selecting Shared, click the ellipsis (...) button to select the desired users

Ok: saves the current layout with the updated settings.

Cancel: cancels saving the layout without applying any changes to Description, Default, or Privacy settings.

Restore a custom layout by right-clicking anywhere in the Pivot Grid, choosing Restore Layout, and selecting the layout to restore.
