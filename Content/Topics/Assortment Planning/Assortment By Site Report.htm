Assortment By Site Report

The Assortment By Site Report is used to review sales and receipts information by site, including:

  • Cluster Set / Cluster membership
  • Site status
  • Approximated proportion of sales plan
  • Approximated proportion of receipt plan
  • Approximated proportion of EOP units

The right hand section displays the split of a sales plan by site with the data being for each Placeholder, as well as aggregates. This section shows: Selling Period, Sales Units Share, Total Orders Units (Receipts + Committed + On Order), EOP Units Share, and the Contribution Percent used to derive the shared units values.

Contribution Percent uses the same LY contribution values as site exclusions and AUR/AUT weighting values. If there is no contribution percent, then an even split is used.

Parameters

The Assortment By Site Report is controlled by five parameters:

Sites: A site filter for the current view.

Product: A product filter for the current view.

Site Hierarchy: The levels of the current plan’s Site Hierarchy to include in the current view.

Cluster Sets: The cluster sets to include in the current view.

Window of Time: The window of time to display in the current view.
