Extending the markdown date in placeholders

For placeholders with a defined markdown date, a regular ROS is calculated for the period from the phase-in date to the beginning of the markdown period. A markdown ROS is calculated for the period from the start of the markdown to the phase-out date.

If the markdown date is extended, then both the original regular ROS and original markdown ROS hold and apply to the new extended regular period and the new markdown period. Whatever the regular ROS was by week by number of weeks is applied to the new weeks that used to be markdown and are now regular.
