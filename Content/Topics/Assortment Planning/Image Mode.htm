Image mode

Image mode enables full-screen display of placeholder images. Like Image view within the Placeholder view, it allows editing of plans while retaining focus on the visual component of an assortment.

Image mode versus image view within the Placeholder view

Image mode is designed especially for single-monitor users who want a bigger workspace for images than Image view provides. Image view is great for multi-screen users, as the view can be detached from the application and moved to a second monitor.

For more information on Image View, see Image View.

Supporting views

The supporting views available in Placeholder and Collection views are also available in Image mode, with the exception of Style Bank and Image view. If desired, detach these views and move them to different areas of the monitor (or to a different monitor if using multiple monitors).

Toolbar options

View options

Quick Text: sets text display options for image cells within the view. Only one quick text option (attribute or metric), plus the placeholder code, can be selected to show at once.

  • Show Code: toggles the display of the placeholder code at the bottom of each cell. The code displays by default.
  • Attributes: select an Attribute Family defined in the Attribute Management module to display in the upper-right corner of each cell
  • AUC, Home AUT, Blended AUT, ROS $, ROS U, Ttl Sls $, Ttl Sls U, Ttl Buy $, Ttl Buy U: selects the metric to display in the upper-right corner of each cell
  • Receipt Status: displays the receipt statuses an indicator of if the placeholder requires recalculation of its receipt plan
  • Replen Status: displays the replenishment status as an indicator of if the placeholder has been updated with the most recent Replenishment forecast data (valid for placeholders of Replen status only)
  • Status: displays the placeholder Status, Open, Created, Assigned, Committed, or Replen, in the upper-right corner of each cell

Inactive: sets the desired visual behavior when temporarily making items inactive in Image view

By default, inactive images show as a diagonal slash:

This parameter can be set to Dark making temporarily inactive items display with a semi-transparent gray overlay:

Recalculate: recalculates sales plans for placeholders of Status “Replen” with a “Dirty” status in the Replen Status field.

  • The date Replen placeholders or collections were last recalculated is shown in the Last Replen field

  • As Replenishment recalculations require complex calculations, they are sent to the Workload Queue module for scheduling and processing.

    Though replenishment placeholders are recalculated immediately they may be deferred for later calculation based on the number of placeholders requiring recalculation and other jobs currently in the queue. Placeholders awaiting recalculation are locked and conditionally formatted to indicate this.

Parameters

Image mode uses most of the same parameters as Placeholder and Collection modes:

  • Product
  • Site Hierarchy
  • Cluster Sets
  • Window of Time
  • Display Currency
  • Grid Config: See Image view.
  • View Selection

For more information, see Placeholder and collection view parameters.

Toolbar buttons

Save: Save changes made to the associated placeholders.

Cancel: Cancel changes.

Parameters: Shows or hides the parameters bar.

Execute: Refreshes Image mode using the selected parameters.
