Packing Instruction view

Packs are configured in the Placeholder view using the Packing Instruction supporting view. The selected placeholder must have a Size Curve assigned to use this view, otherwise the Packing Instruction view is disabled. When accessing this view for a placeholder with an assigned size curve, the Pack Instruction Parameters portion of the view is enabled.

Use Configuration > View Maintenance to add the Packing Instruction view to the View Selection dropdown menu.

A new size curve for the placeholder can be selected to view available options from a different size curve. The Packing Instruction view refreshes and disables additional pack instruction views. Only the Pack Instruction Parameters view remains editable. The process can be repeated as required.

Once a Packing Instruction multiple has been selected, users can recalculate receipts and create commitments.

The Packing Instruction supporting view does not support multi-SKU prepacks.

Defining a pack

  1. In the Pack Instructions Parameters section, review the default values and adjust these values as required.

  2. If values were adjusted, click Save.

    The Receipt Status changes to Dirty after values are changed.

  3. Click Calculate.

  4. In the Pack Instructions Selector, review the suggested pack instructions.

    Fields with values that exceed default thresholds are formatted in red.

  5. Review the various pack instructions in the Pack Instruction Size Details and Pack Instruction Size Cover sections.
  6. Optionally, revise values in the Pack Instructions Parameters section, click Save and then click Calculate.
  7. Select one of the instructions sets and click Save.

  8. If the Pack Instruction Selector dialog displays, click Yes.

  9. Navigate to the Receipt Plan view and verify if Variant Order Multiple matches the Pack Size. If not, recalculate the receipts using the Recalculate Receipts button in the Receipt Plan view.

    Any time a Packing Instruction is selected, the Order Multiple and Receipt Status is set automatically. Manually changing the Order Multiple value may cause the order quantity to not conform to the Pack Size. Click Calculate again to get it to match.

  10. Click the Create Commitments button to create the Commitments.

    The newly configured pack flows to Order Manager. The packing instructions in Order Manager are generated from the Packing Instruction supporting view.

For more information, Order Document details – Prepack tab For more information, refer to the Order Manager User Guide.

Pack Instruction Parameters section

The Pack Instruction Parameters portion displays default parameters as defined from new metrics added to the Hierarchy Defaults view (Manage Assortment Defaults > Assortment Plan Defaults).

The metrics include:

Metric Definition
Pack Type Pick Cost Used in the Pack Instruction Parameters view and as part of the calculation of Picking Cost Ratio.
Pack Type Max Cost Ratio Used in the Pack Instruction Selector view. The Picking Cost Ratio cell is formatted red if the Picking Cost Ratio goes above this value.
Pack Type Max Weeks Cover The number of Weeks cover that the Pack should be below.
Pack Type Max Error Ratio Used in the Pack Instruction Selector view. The Absolute Error cell is formatted red if the Absolute Error goes above this value.
Pack Type Default Multiplier Used as a tie break. If the system cannot select a Pack then the pack with the Multiplier corresponding to this value is selected. Valid values are 1, 2, 2.5, 3, 3.5.
Pack Type Default ROS A default ROS can be set against the hierarchy. Users can choose to use this value instead of the Placeholder ROS in the calculation. To use this Default ROS, Apply Hierarchy ROS must be checked on the Pack Instruction Parameters view.

Default values are adjusted within the Pack Instruction Parameters view. Clicking Save updates these values and starts the calculation. If no changes are desired, the Calculate action button starts the calculation.

For more information on the defaults, see Assortment Plan Defaults tab.

Pack Instruction Selector section

The calculation returns 5 iterations of packing options within the Pack Instruction Selector view. These options correspond to pack multipliers of 1, 2, 2.5, 3, and 3.5. The calculation returns values for each of these multipliers. Any values which are outside of the defined acceptable limits are highlighted red. An ideal pack multiplier would be below the Max Weeks Cover and have the lowest error ratio.

If an ideal pack is not available, the system selects the pack that corresponds to the Default Multiplier metric.

Fields formatted in blue color are editable. Editable values can be overridden at any time.

Pack Instruction Size Details section

Displays each pack’s details.

Pack Instruction Size Cover

Displays the weeks cover for size for average, fast, and slow rates of sale.
