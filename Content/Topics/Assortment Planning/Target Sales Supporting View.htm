Target Report view

The Target Report view displays existing Target Scenario values. New target values can be input based on the currently selected parameters in the main planning view. The Target Report view makes it easier to reconcile back to the original target when making changes.

The top grid is read-only displaying existing scenarios seen in Target Creation. In the lower grid, add, update, and delete Target values.

For more information and field descriptions, see Target creation.

Adding the Target Report view to the main planning view

  1. Select the dropdown arrow in the View Selection parameter.
  2. Click the Target Report checkbox.

  3. Exit the dropdown menu.

    The Target Report button appears on the right side of the screen.
