Receipt Constraint Comparison Report

The Receipt Constraint Comparison Report displays actual receipt constraint values with cells formatted in orange if their values differ from the default, cluster default, attribute default, or item cube default by placeholder. The report highlights where the parameters used to generate a receipt plan varied from defaults that were setup.

Though defaults can be defined in AP, there is the ability to override those defaults. Use the Receipt Constraint Comparison Report to identify where the defaults were overridden.
