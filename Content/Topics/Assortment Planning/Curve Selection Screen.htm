Curve Selection screen

Click the ellipsis () in the Seasonality Curve field to launch the Curve Selection screen. The Seasonality Curve field is found in the main planning view and in the Cluster Sets supporting view.

The Tags field is used to display or set the tags that filter the curves returned for selection. This makes it easier for users to find the desired curve.

Assigning meaningful tags to curves upon generation helps avoid situations where the desired curve is truncated from the list returned for selection.

The maximum number of curves returned based on the selected tags is set by the system policy Max Selectable Lookup Curves. This is to prevent slowdown in the UI.

Field Description
Name The name of the corresponding curve.
Curve Generation Type The method used to generate the corresponding curve: Manual or Specified Conditions Using Enhanced Filtering.
Bucket Type The bucket type for the curve, usually weekly or monthly.
Bucket Start Date The start date of the first bucket within the curve Start Date and End Date of the corresponding curve.

Technically, this can be before Start Date if Start Date falls in the middle or at the end of the first bucket.
Start Date The start date of the corresponding curve.
End Date The end date of the corresponding curve.
Number of Periods The number of time periods (buckets) encompassed by the curve.
Existing Indicates that the corresponding curve is already assigned to one or more placeholders in the current view. This is provided for ease of assignment of curves that have already been used for other existing placeholders.
Tags Indicates that the corresponding curve has one or more of the selected Tags attached to it.
