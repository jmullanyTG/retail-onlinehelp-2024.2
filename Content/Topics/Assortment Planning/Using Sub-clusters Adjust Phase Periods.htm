Using sub-clusters to adjust Phase Periods based on site attributes

Assorting Plans to sub-clusters

Sub-clusters are selected in the Manage Assortment Defaults screen by clicking in the Additional Allowed Cluster Sets field and selecting them. The Cluster Details side pane allows users to manage the assortment of placeholders and collections to clusters and sub-clusters.

It is possible to have regular cluster sets in the Plan as well, but these will appear as overlapping cluster sets. Placeholders cannot be assigned to both the cluster AND the specific sub-clusters.

For information on configuring sub-clusters, see Configuring sub-clusters.

For information on configuring and using sub-clusters, refer to the Cluster Site Manager User Guide.
