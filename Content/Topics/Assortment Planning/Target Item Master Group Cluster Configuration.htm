Configuring Item Master Group values with Clusters

The Target Item Master Group Cluster Configuration view allows for the configuration of Item Master Group X values with Clusters. Each segment is assigned cluster sets. Comparison data is aggregated for all clusters found within those cluster sets (based on the snapshots specified for use in the scenario by the Use Comparison Cluster Set Snapshots for Comparison Data system policy).

The Target Item Master Group Cluster Configuration view is used in Target Creation in the Assortment Planning module and requires that system policy Target Filter Item Master Group be set to a particular Item Master Group.

The variants used to aggregate history is based on either the:

  • Target Filter, or
  • the hierarchy level (if Segment by Hierarchy was used to create the segment).

By configuring Item Master Group X values with Clusters, the list of variants is further narrowed down by cluster, depending on the definitions set in system policy Target Filter Item Master Group (e.g. Cluster A will only use variants with Item Master X property of 1).

When Comparison data is generated, it is further constrained by this association. Then, when Rationalization is run, recommended choice counts calculate over this narrower set of data.

For more information, regarding the Cluster Containment Configuration and Cluster Set Containment Configuration sections, see Configuring which clusters get more choices than others.

For example, where Store Grade values are: 1, 2, and 3 and possible Clusters are: A, B, C, and D, then users can map:

  • A should only consider history for variants with Item Master Group X value of 1
  • B should only consider the values of 1 and 2
  • C should only consider the values of 1 and 2
  • D should use variants with any value for Item Master Group X (1, 2, and 3)

Configuring Item Master Group X values with Clusters

  1. Select an Assortment Plan using the dropdown list in the Assortment Plan field and then click Execute.

    The grid displays the intersection between Item Master Group X values and Cluster values. By default, all combinations are opted in.

  2. As required, deselect particular combinations by clicking on the relevant checkbox.
  3. Click Save.
