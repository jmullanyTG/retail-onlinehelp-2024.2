Creating new collections

Collections are groups of placeholders that can be planned as a single product. These are often referred to as styles or key items.

Collections are created in Placeholder view or in Variants view of Collection view. In Variants view, ensure the Current Collection field is not selected.

Variants view in Collection view

To create a new collection

  1. Click New in the Variants toolbar.

  2. Enter a value in the # of Placeholders field.

    This is the number of new placeholder lines to be added once Generate is clicked.

  3. Optionally, select Mirror Options.

    For field descriptions, see Adding placeholders to collections.

  4. Ensure the Current Collection checkbox is unchecked.

  5. Click Generate.

  6. Click in the Coll Code field to display the following dialog.

  7. In the Collection Desc field, enter the name of the new collection and click New.

    The Coll Code field automatically populates with a system generated code and the Coll Desc field populates with the provided description. The new collection is added to the assortment plan.
