Assortment Rollup metric definitions

After parameters are selected, click Execute to return the following metrics:

EOP and BOP values include Inter-Branch Transfers (IBT) for sites-in-cluster. Therefore, EOP and BOP values in all views and reports which use these metrics (such as the Sales Plan view and the Receipt Plan view) reflect values inclusive of IBT.

Metric Description
AUC Average unit cost price coming from the Sales Plan view. This value matches the system AUC line unless a user has overridden the AUC in the Receipt Plan view.
AUC % +/- LY Percentage increase or decrease of average unit cost price versus LY average unit cost price.
AUR The average unit retail of all products contained in a given aggregate level.
AUR % +/- LY Percentage increase or decrease of AUR versus last year AUR.
Home AUT The average unit ticket price in Home Site currency of all products contained in a given aggregate level.
Blended AUT The average unit ticket price blended from all sites’ currencies of all products contained in a given aggregate level.
Avg BOP Unit Cost Average cost of inventory per unit coming from the Sales Plan view. This value matches the system Avg BOP Unit cost line unless a user has overridden the sales plan in the Sales Plan view or receipt plan in the Receipt Plan view.
Avg BOP Unit Cost (Comparison) Avg BOP unit cost of inventory coming from the selected comparison snapshot. Includes any user overrides present when the snapshot was taken.
Avg BOP Unit Cost % +/- LY Percentage increase or decrease of Avg BOP Unit Cost versus LY Avg BOP Unit Cost.
Avg EOP Unit Cost Average cost of inventory per unit coming from the Sales Plan view. This value matches the system Avg EOP Unit cost line unless a user has overridden the sales plan in the Sales Plan view or receipt plan in the Receipt Plan view.
Avg EOP Unit Cost % +/- LY Percentage increase or decrease of Avg EOP Unit Cost versus LY Avg EOP Unit Cost.
Avg EOP Unit Cost (Comparison) Avg EOP Unit Cost of inventory coming from the selected comparison snapshot. Includes any user overrides present when the snapshot was taken.
BOP Cost Beginning of Period cost of inventory coming from the Sales Plan view. This value matches the system BOP Cost line unless a user has overridden the sales plan in the Sales Plan view or receipt plan in the Receipt Plan view.
BOP Cost % +/- LY Percentage increase or decrease of BOP Cost versus LY BOP Cost.
BOP Units Beginning of Period units of inventory coming from the Sales Plan view. This value matches the system BOP Units line unless a user has overridden the sales plan in the Sales Plan view or receipt plan in the Receipt Plan view
BOP Units % +/- LY Percentage increase or decrease of BOP Units versus LY BOP Units.
BOP Units (Comparison) Beginning of Period units of inventory coming from the selected comparison snapshot. Includes any overrides from the user when the snapshot was taken.
COGS Cost of sales coming from the Sales Plan view. This value matches the system cost of sales line unless a user has overridden the sales plan in the Sales Plan view.
COGS % +/- LY Percentage increase or decrease of cost of sales versus LY cost of sales.
Comparison AUC Average unit cost price coming from the selected comparison snapshot. Includes any user overrides present when the snapshot was taken.
Comparison AUR The average unit retail of all products contained in a given aggregate level coming from the selected comparison snapshot.
Comparison Home AUT The average unit ticket price in Home Site currency of all products contained in a given aggregate level from the selected comparison snapshot.
Comparison Blended AUT The average unit ticket price blended for all site currencies of all products contained in a given aggregate level from the selected comparison snapshot.
Comparison BOP Cost Beginning of Period Cost of inventory coming from the selected comparison snapshot. Includes any user overrides present when the snapshot was taken.
Comparison COGS Cost of sales coming from the selected comparison snapshot. Includes any user overrides present when the snapshot was taken.
Comparison EOP Cost End of Period Cost of inventory coming from the selected comparison snapshot. Includes any user overrides present when the snapshot was taken
Comparison Sales Sales coming from the selected comparison snapshot. Includes any user overrides present when the snapshot was taken.
Comparison Sales Ex VAT Sales excluding value added tax coming from the selected comparison snapshot. Includes any user overrides present when the snapshot was taken.
Comparison VAT Sales VAT sales coming from the selected comparison snapshot. Includes any user overrides present when the snapshot was taken.
EOP Cost End of period cost of inventory coming from the Sales Plan view. This value matches the system EOP Cost line unless a user has overridden the sales plan in the Sales Plan view or receipt plan in the Receipt Plan view.
EOP Cost % +/- LY Percentage increase or decrease of EOP Cost versus LY EOP Cost.
EOP Units End of Period units of inventory coming from the Sales Plan view. This value matches the system EOP Units line unless a user has overridden the sales plan in the Sales Plan view or receipt plan in the Receipt Plan view.
EOP Units % +/- LY Percentage increase or decrease of EOP Units versus LY EOP Units.
EOP Units (Comparison) End of period units of inventory coming from the selected comparison snapshot. Includes any user overrides present when the snapshot was taken.
Inventory Adjustment Cost Planned/actual inventory adjustment at cost for the period. This value can add to or subtract inventory at cost from EOP Cost
Inventory Adjustment Units Planned/actual inventory adjustment units for the period. This value can add to or subtract inventory units from EOP units.
LY AUC Last year aggregated average unit cost price coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY AUR Last year average unit retail of all products contained in a given aggregate level.
LY Home AUT Last year average unit ticket price in Home Site currency of all products contained in a given aggregate level.
LY Blended AUT Last year average unit ticket price blended from all site currencies of all products contained in a given aggregate level.
LY Avg BOP Unit Cost Last year average BOP unit cost of inventory coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY Avg EOP Unit Cost Last year average EOP unit cost of inventory coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY BOP Cost Last year beginning of period cost of inventory coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY BOP Units Last year beginning of period units of inventory coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY COGS Last year cost of sales coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY EOP Cost Last year end of period cost of inventory coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY EOP Units Last year end of period units of inventory coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY Inventory Adjustment Cost Last year planned/actual inventory adjustment at cost for the period. This value can add to or subtract inventory at cost from EOP Cost.
LY Inventory Adjustment Units Last year planned/actual inventory adjustment units for the period. This value can add to or subtract inventory units from EOP units.
LY Margin Last year margin coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY Margin % Last year margin % coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY On Order AUC Last year average unit cost price of LY On Order Units coming from the Sales Plan view.
LY On Order Cost Last year receipt cost for receipt units for which orders have been submitted to external systems and for which order confirmations have been received. Relevant for planned values only in scenarios where planning is occurring further than 12 months out from the current date. Actual receipts delivered are actualized and displayed as LY Receipt Units.
LY On Order Units Last year receipt units for which orders have been submitted to external systems and for which order confirmations have been received. Relevant for planned values only in scenarios where planning is occurring further than 12 months out from the current date. Actual receipts delivered are actualized and displayed as LY Receipt Units.
LY Receipt AUC Last year aggregated average unit cost price of receipts coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY Receipt Cost Last year receipt cost coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY Receipt Units Last year receipt units coming from the Sales Plan view. This is a planned value or actualized value depending on the comparable time period last year.
LY Sales Last year sales coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY Sales Build Ratio Ratio of last year sales of current period versus last year sales of previous period.
LY Sales Ex VAT Last year sales excluding value added tax coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY Sales Units Last year sales units coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY Sales Units Build Ratio Ratio of last year sales units of current period versus last year sales units of previous period.
LY Sellthrough Last year by period sellthrough percentage.  (LY Sales Units/LY BOP Units).  This is not a rolling calculation and looks at each period individually.
LY Shrink Cost Amount of inventory at cost lost last year through theft, damages, spoilage, etc. that are planned or actual. LY Shrink Cost is subtracted from LY EOP Cost.
LY Shrink Cost % of COGS Percentage of LY total COGS incurred from planned or actual LY shrink at cost.
LY Shrink Units Amount of inventory units lost last year through theft, damages, spoilage, etc. that are planned or actual. LY Shrink Units is subtracted from LY EOP Units.
LY Shrink Units % of Sales Units Percentage of LY total sales units incurred from planned or actual LY shrink units.
LY Stock Cost to COGS Ratio expressing the relationship of Last year Stock Cost to Last Year COGS coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year.
LY Stock Units to Sales Units Ratio expressing the relationship of Last Year Stock Units to Last Year Sales Units coming from the Sales Plan view. This value is a planned value or actualized value depending on the comparable time period last year
LY VAT % Last year’s value added tax rate applied to the revenue generated from an organization for same period last year.
LY VAT Sales Amount of sales revenue generated last year through last year value added tax.
Margin Margin coming from the Sales Plan view. This value matches the system margin line unless a user has overridden the sales plan in the Sales Plan view or AUC in the Receipt Plan view.
Margin % Margin percentage coming from the Sales Plan view. This value matches the system margin % line unless a user has overridden the sales plan in the Sales Plan view or AUC in the Receipt Plan view.
Margin % (Comparison) Margin percentage coming from the selected comparison snapshot. Includes any user overrides present when the snapshot was taken.
Margin % +/- LY Percentage increase or decrease of margin versus LY margin.
Margin (Comparison) Margin coming from the selected comparison snapshot. Includes any user overrides present when the snapshot was taken.
Margin Rate % +/- LY Percentage increase or decrease in margin percent versus last year margin percent.
Minimum Presentation Number of units in aggregate bought for presentation.
On Order AUC Average unit cost price of On Order Units coming from the Sales Plan view.
On Order Cost Cost of receipt units for which orders have been submitted to external systems and for which order confirmations have been received.
On Order Cost % +/- LY Percentage increase or decrease of On Order Cost versus LY On Order Cost.
On Order Cost % +/- Receipt Cost Percentage increase or decrease of On order cost versus the receipt cost coming from Sales Plan view.
On Order Past Due Cost Cost of purchase orders not yet received but scheduled to be received in the recent past.
On Order Past Due Units Purchase order units not yet received but scheduled to be received in the recent past.
On Order Units Receipt units for which orders have been submitted to external systems and for which order confirmations have been received.
On Order Units % +/- LY Percentage increase or decrease of On Order Units versus LY On Order Units.
On Order units var to Receipt Units The difference between planned receipt units from the Receipt Plan view and On Order Units.

Receipt AUC % +/- LY Percentage increase or decrease of average unit cost price of receipts versus LY average unit cost price of receipts.

Receipt Cost Receipt cost coming from the Sales Plan view. This value matches the system Receipt Cost line unless a user has overridden the receipt plan in the Receipt Plan view.
Receipt Cost (Comparison) Receipt cost coming from the selected comparison snapshot. Includes any user overrides present when the snapshot was taken.
Receipt Units Receipt units coming from the Sales Plan view. This value matches the system Receipt Units line unless a user has overridden the receipt plan in the Receipt Plan view.
Receipt Units % +/- LY Percentage increase or decrease of Receipt Cost versus LY Receipt Cost.
Receipt Units % +/- LY Percentage increase or decrease of Receipt Units versus LY Receipt Units.

Sales Sales coming from the Sales Plan view. This value matches the system sales line unless a user has overridden the sales plan in the Sales Plan view.
