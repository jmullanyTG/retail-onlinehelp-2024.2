Image view

Image view is a visual summary of an assortment plan, either by cluster or by period. Quick visual edits can be done dynamically to see if the removal of certain items helps or hinders the visual goal.

Image view notes

  • Image view is not available in Collection view.
  • The filter parameters in this view are linked with those in the main planning view. This view updates and refreshes accordingly when parameters change and are executed in the main planning view. The filter criteria within Image view can be changed.

  • Due to the large data size associated with image files, selecting too much data to display can cause the view to take a long time to refresh.

    The system policy Image View Maximum number of SQL Data Rows (Module: Assortment) can be used to limit the amount of data that can be brought back in Image view and help mitigate such issues. The view displays an error if the data selection exceeds the value set in this policy.

Image mode versus image view

Image mode is designed especially for users with a single monitor who want a bigger workspace for images than Image view provides. Image view is great for multi-screen users, as the view can be detached and moved to a second monitor.

For more information on Image mode, see Image Mode.

Working in the image cells

Click the View Options button in the top left of the view to display a dropdown menu with options to configure the information that displays. The following menu options are available:

Option Description
Quick Text
  • Show Code: On by default. Toggles the display of the placeholder code at the bottom of each cell.
  • Attributes: Selects an Attribute Family defined in the Attribute Management module to display in the upper-right corner of each cell.
  • AUC, Home AUT, Blended AUT, ROS $, ROS U, Ttl Sls $, Ttl Sls U, Ttl Buy $, Ttl Buy U: Selects the metric to display in the upper-right corner of each cell.
  • Receipt Status: Displays the Receipt Status, Clean or Dirty, in the upper-right corner of each cell as an indicator of if the placeholder requires recalculation of its receipt plan.
  • Replen Status: Displays the status (Clean or Dirty) in the upper-right corner of each cell. Indicates whether the placeholder has been updated with the most recent Replenishment forecast data. Valid for placeholders of Replen status only.
  • Status: Displays the placeholder status (Open, Created, Assigned, Committed, or Replen) in the upper-right corner of each cell.

Making cells inactive

Users can make placeholders temporarily inactive by double-clicking any cell to make visual assessments. If a placeholder is present in more than one cell, either by cluster or period, then all cells related to the placeholder are made inactive automatically.

Making placeholders temporarily inactive has no effect on the Active status of placeholders in the main planning view.

Configuring and maintaining images

To configure and maintain images:

  1. Right click on an image in the grid and select Edit to launch the Image Gallery dialog for the selected placeholder (not available in Collection view).

  2. If required, click New to add a new cell.
  3. Select a cell and click Carousel.
  4. In the cell, right-click and select Edit.
  5. Scroll down the list of images displayed in the Image Maintenance screen and select an image.
  6. Click Save.
  7. Click Save in the Image Gallery.
  8. Click Execute to refresh the view.

The Image Gallery screen

  • Classification: Displays the classification of each image.

    The first image assigned to a placeholder is always the primary by default, but more images can be assigned with user defined classifications.

  • Primary?: Whether an image is the primary image to display whenever images are shown for the placeholder.

  • Multiples?: Whether multiple images are allowed for the given classification.

    The Primary and Multiples parameters for variant images can be set in Management > Variant Manager > Image Maintenance.
