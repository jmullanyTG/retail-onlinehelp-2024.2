Ship from Store (SFS) example

For 3 cluster set / clusters: E-Com, Retail A, and Retail B.

E-Com will have some of its sales be fulfilled by the other 2 clusters. Therefore, some of the demand needs to be reassigned to those 2 clusters so that:

  • the receipt quantities are sufficiently high, and

  • E-Com's demand is reduced so that it does not over-order stock.

Cluster Set Cluster Metric wk10 wk11 wk12
E-Com E-Com Sales Units 1250 1140 1140
Retail A Sales Units 570 610 650
Retail B Sales Units 480 440 470

E-Com is assigned a Ship From Store % value for each period:

The Ship From Store metrics are calculated as (SFS% * Sales Units) and the Fulfillment metrics are calculated (Sales Units - SFS Units).

Cluster Set Cluster Metric wk10 wk11 wk12
E-Com E-Com Sales Units 1250 1140 1140
E-Com E-Com Ship From Store % 20% 30% 40%
E-Com E-Com Ship From Store Units 250 342 456
E-Com E-Com Fulfillment Units 1000 798 684

The Ship From Store demand is transferred to the other 2 clusters using a percentage split of their existing sales plans:

Cluster Set Cluster Metric wk10 wk11 wk12
Retail A Sales Units 570 610 650
Retail A Ship From Store Units 136 199 265
Retail B Sales Units 480 440 470
Retail B Ship From Store Units 114 143 191

The Fulfillment metrics are calculated (Sales Units + SFS Units) and Ship from Store % remains unchanged:

Cluster Set Cluster Metric wk10 wk11 wk12
E-Com E-Com Sales Units 1250 1140 1140
E-Com E-Com Ship From Store % 20% 30% 40%
E-Com E-Com Ship From Store Units 250 342 456
E-Com E-Com Fulfillment Units 1000 798 684
Retail A Sales Units 570 610 650
Retail A Ship From Store % 0% 0% 0%
Retail A Ship From Store Units 136 199 265
Retail A Fulfillment Units 706 809 915
Retail B Sales Units 480 440 470
Retail B Ship From Store % 0% 0% 0%
Retail B Ship From Store Units 114 143 191
Retail B Fulfillment Units 594 583 661

When Receipt Plans are calculated for the clusters, the Fulfillment data series is used as the demand signal so that E-Com shows a reduced order quantity and Retail A and B show an increased order quantity.
