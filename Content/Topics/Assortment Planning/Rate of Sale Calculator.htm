Rate of Sale Calculator view

The Rate of Sale Calculator is used to model scenarios to determine how much, in units or revenue, a particular placeholder/collection might sell per site in a given week. All of the functionality essential to Assortment Planning is available in this docked version.

The full version of the Rate of Sale Calculator which can also create scenarios for use in Allocation Strategies is accessed via Execute > Rate of Sale.

There are three main options in determining a rate of sale for a placeholder or collection in planning demand with assortment planning:

  • Smart Start: Leverages a marginal return curve to determine rates of sale for each placeholder required to fulfill a target set in Target Creation.
  • Like Style Modeling: Manually create placeholders and collections and model those choices after historical like items.

  • Rate of Sale Calculator: Select after smart starting an assortment or manually creating a placeholder or collection. Assortment plans can be refined by using historical data to determine an appropriate rate of sale to apply to placeholders or collections within them.

    Modeling in the Rate of Sale Calculator is extremely flexible in terms of the product, cluster set, and time dimensions that can be selected to model demand with confidence.

The Rate of Sale Calculator is divided into three main components.

  1. ROS Scenario Creation
  2. ROS Calculation
  3. ROS Application
