Managing split factors

The Manage Split Factors screen has a pivot grid for fields by distribution center (DC), cluster set, and cluster. The percentage of DC receipts Cluster Set Clusters receive can be defined.

Accessing the Manage Split Factors screen

  1. Navigate to the Assortment Planning module.
  2. Click the Manage button.
  3. Select Split Factors in the Manage Assortment Plans toolbar.

Changing the percentage of receipts a DC receives

  1. Select the appropriate field within the grid.
  2. Click the dropdown arrow to display a calculator.
  3. Choose the new percentage.

  4. Click Save.

    The figures displayed in the grid automatically recalculate.
