Target Worksheet

The Target Worksheet (editing and copying only) is a standard pivot grid which lets users manipulate the field layout by dragging and dropping fields in the Filter, Row, Column, and Data areas of the grid.

Toolbar functionality

  • Graph: Displays a graph describing the data series selected by the user

  • Fields: Launches the Fields Customization dialog. Each tab contains a search bar to more quickly locate fields. The fields shown in the view are limited to only those containing the text entered.

    • Simple tab: Fields can be dragged and dropped between Filter, Row, Column, and Data areas of the Pivot Grid to manipulate the current layout. Dragging fields to the None area hides them in the layout.

    • Advanced tab: Manipulate the layout here by selecting the desired location of each field from the Pivot Area dropdowns next to each. Within each area, an Index can be specified to define the order in which the fields display.

To manually key sales plans for new areas of business, where the TY Act New metrics have defaulted to zero due to lack of sales history, enter them in order based on Sales Basis:

  • Revenue: enter Sales first then AUR to calculate Sales Units
  • Units: enter Sales units first then AUR to calculate Sales
