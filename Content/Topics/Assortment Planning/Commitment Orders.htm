Commitment orders

Commitment orders can be created for one placeholder or collection at a time within the Assortment Planning module by using the Create Commitments button within the Receipt Plan supporting view.

Additionally, the Order Manager module can be used to create commitment orders from assortment plans for entire sections of the product and site hierarchies and windows of time at once. (To initiate the creation of new commitment orders in Order Manager, click the Commitment button in the top toolbar.)

Supplemental commitments

Supplemental commitments can be made during a period within which an existing commitment order already exists. These additional commitment orders are supplemental and are neither an update nor a replacement. The ability to create a supplemental commitment is controlled by the Allow supplemental commitments system policy.
