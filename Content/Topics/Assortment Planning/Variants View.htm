Variants view (only available in collection view)

In Collection view, the Variants supporting view provides access to the individual placeholders that exist within each collection. Variants view automatically updates to show the placeholders within the selected collection.

Header area functionality

The header area functionality in Variants view is essentially the same as those in the main planning view. The exception is the ability to change the data filter parameters since the view is linked to the selected collection in the main planning view.

Planning area and metrics

The planning area in Variants view functions in the same way as the main planning view in Placeholder view including all metrics, the ability to add or remove columns via the column chooser in View Options, and the ability to save and restore layouts.

Style bank

Use the style bank to assign variants to placeholders in Variants view in the exact same manner as in Placeholder view of the main planning view.
