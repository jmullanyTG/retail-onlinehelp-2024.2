Freezing columns in the Placeholder and Collection views

Columns can be grouped and frozen in place in both the Placeholder and Collection views.

  1. Right-click in a column header and select Column/Band Chooser from the menu.

  2. In the Customization dialog, navigate to the Bands tab.

  3. Drag the Grouping option to the header bar.

  4. Drag the columns to freeze to the left into the Grouping band.

    Columns can be dragged into and out of the Grouping section.

  5. Right-click in a row in one of the columns and select an option under Column Freezing.

Unfreezing columns

To unfreeze columns, right-click in a row in one of the columns and select Not Fixed under Column Freezing.
