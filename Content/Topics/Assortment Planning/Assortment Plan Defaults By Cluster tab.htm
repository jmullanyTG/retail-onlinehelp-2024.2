Assortment Plan Defaults By Cluster tab

The Assortment Plan Defaults By Cluster tab provides the ability to apply a different default value to new placeholders at different cluster set clusters so that expected differences in the sales plans can be captured based on how items within those clusters typically behave.

Accessing the Assortment Pan Defaults By Cluster tab

  • In the Assortment Planning module, click Manage and select Defaults on the Manage Assortment Plans screen.

Parameters

Parameter Description
Cluster Sets The Cluster Sets parameter drives this view. Any changes to this parameter filters the results brought back based on the cluster set selection.
Window of Time Selects the window of time to display in the current view.

Setting default parameters

Any of four default parameters can be set for any level of the hierarchy for any cluster set cluster. The four default parameters are:

  • Seasonality Curve
  • Minimum Presentation
  • Periods of Supply
  • Min Stk/Sales (Minimum Stock to Sales)

These defaults are used whenever a new placeholder in the particular product hierarchy level is generated.
