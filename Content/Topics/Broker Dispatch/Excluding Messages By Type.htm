Excluding unwanted messages

Filters can be created that exclude unwanted messages from appearing in the Broker Dispatch Viewer. This removes clutter from the message display.

Filtered messages are not deleted but just suppressed from view in the Broker Dispatch Viewer screen.

Excluding filters

Exclude filters are managed from the Message Log Exclude Filter Management dialog box.

Creating an exclude filter

  1. Click Exclude Filter in the Broker Dispatch Viewer toolbar.

  2. In the Message Log Exclude Filter Management dialog box, click New.

  3. Enter the following values for the new filter in the Message Log Exclude Filter dialog:

    • Exclude Filter Description - A brief name or description for the filter.
    • Exclude Filter Type - Select what type of value the filter will exclude. Options are:
      • Message Regular Expression: Select to filter based on the content of the message.
      • Affable Message Regular Expression:
      • Message Event Log Entry Type: Select to filter messages by severity (Information, Warning, or Error).
      • Message Type: Select to filter messages by specific type (Dispatch, Progress, Application, Database Dispatch Execute Action Parallel, API, or Unknown).
    • Exclude Filter Value - Enter or select the values to exclude from view. Options are based on the Exclude Filter Type selected.
      • For expression filter types, type or paste the value directly into the field.
      • For all other types, select one of the preset values.
    • Security - Whether to make the filter private (available only to the logged-in user), public (available to all roles and users), or shared (available only to selected users and roles).

    • Is Enabled - Enables the filter. When unchecked, the filter is disabled.

  4. If Shared is selected, click Change to select the users and roles to share the filter with.
  5. Click Ok to save the filter

Editing an exclude filter

  1. In the Message Log Exclude Filter Management dialog box, select a filter and click Edit.

  2. Edit the filter values as desired.

  3. Click Ok to save changes.
