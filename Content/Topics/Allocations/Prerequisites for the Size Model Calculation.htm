Prerequisites for the size model calculation

The following are prerequisites that must be met for the size model calculation to run:

Stock On-hand Data by Site/Size: the size model calculation makes use of stock on-hand data to qualify whether the sales that occurred for a given variant/period should be included when calculating a size model for a given site. If this data is not present for the items included in the product filter selected for the size model, the enhanced calculation method cannot be employed and the old sales-only calculation method is used instead

Data Cleansing Thresholds: at least one of the two data cleansing threshold fields must be set to a positive value. For more information, see Data Cleansing Thresholds.
