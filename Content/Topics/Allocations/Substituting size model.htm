Substituting size models

To substitute one site’s curve for another:

  1. Select the size model header to make substitutions in from the list in the top area of the Size Models Screen.
  2. Ensure the Override Allowed? field is checked.

  3. In the bottom grid, select a site from the dropdown menu in the Override Site field of the site whose curve is to be overridden.

  4. Click Save.

    Sites that have an Override Site assigned and saved do not appear for selection in the Override Site dropdown menu. To clear the Override Site prior to saving, select the dropdown entry with a blank Site ID and No Substitution in the Description.

Contribution values of the overridden site’s size model

The contribution values of the overridden site’s size model will match those of the override Site. The Clear Override field is conditionally formatted orange, indicating an override has been made.

Is Eligible? settings for the Override Site are also copied. An error displays when trying to assign an Override Site to a site that is already being used as an Override Site. This is done to prevent circular or difficult to follow logic with the curves being substituted.
