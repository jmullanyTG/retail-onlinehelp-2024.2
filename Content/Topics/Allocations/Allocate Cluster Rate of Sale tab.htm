Rate of sale (ROS) by grade demand type

For allocation strategies that use the rate of sale by grade demand type, the included rate of sale grades must be selected and applied. Using the Allocate Cluster Rate of Sale tab, the need to associate a location strategy with an allocation strategy can be bypassed. This feature further enhances the integrated process of assortment plans carrying into allocation strategies.

In its most simple form, the rate of sale is set to what is achievable for each one for the grades by setting the rate of sale override in the Allocate Cluster Rate of Sale tab for each allocation strategy.

Field descriptions:

  • Included – If selected, the cluster set rate of sale grades is included.
  • Cluster Set – The description of the cluster set.
  • Classification – The value of the classification/grade to be included or excluded.
  • Calculated Rate of Sale – The ROS value for the grade as calculated by the Rate of Sale Calculator. This is the value which has been pushed to the strategy. For more information, see Rate of Sale overview. For more information, refer to the Rate of Sale User Guide.
  • Rate of Sale Override – The by grade user override rate of sale value. If present, this is used instead of the calculated rate of sale.
