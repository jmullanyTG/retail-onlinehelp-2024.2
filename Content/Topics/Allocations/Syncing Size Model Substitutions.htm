Syncing size model substitutions

It is possible to link a site to its override site’s size model so that subsequent changes made to the override site’s curve are reflected in the site it is assigned to. To achieve this link, do the following:

  • In the bottom grid of the Size Models screen, check the Sync Override field for any site with an override site assigned and click Save.

Checking Sync Override and saving for a site that does not have an assigned Override Site does nothing. The Sync Override box is unchecked when the screen refreshes.
