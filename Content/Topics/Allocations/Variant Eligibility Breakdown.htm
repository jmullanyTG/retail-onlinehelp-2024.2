Viewing inventory level detail

The variant eligibility breakdown section on the Product Groups screen displays inventory level details (item-site, or style-color-size-site) by size and by DC. For each variant there is a breakdown of the details at the DC level.

  • DC Code: The unique site code for the DC.
  • DC Name: The site name for the DC.
  • Item Code: The item master code for the item. Will be specific down to the size level.
  • Product Description: Description of the item. Will be specific down to the size level.
  • Size Name: Associated size name for the item. Will not be reflective of range.
  • Available SOH: Current available stock on hand at the DC (will use the available to allocate calculation).
  • Inbound: Quantity of any inbound orders to the DC.
