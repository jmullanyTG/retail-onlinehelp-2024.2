Manually overriding size models

It is also possible to override a site’s curve via manual entry. In the bottom grid of the Size Models screen.

  1. Click in the cell of the site and size whose contribution percentage is to be changed.
  2. Enter the desired value.
  3. Click Save.

A manual change to a site with an assigned Override Site also clears the Sync Override checkbox if it was checked.

When rebalancing a curve based on manual changes, the system attempts to retain all entered Contribution values for a given site’s curve. However, it is possible the system changes entered values to balance to 100%. For example, user entered changes to the values for all sizes mean the total curve does not equal 100%.

Example

Any override site is cleared. By making a manual change, the curve deviates from the override site’s curve values and substitutes a manual curve in its place.
