Allocation Strategy Save Point tab

The state of a strategy line is saved using the Allocate Strategy Save Point tab. The save point can be applied to other strategy lines for mass changes.

The grid in the left hand side of the Allocate Strategy Save Point tab displays save point details including creation date and who created the save point.

Field descriptions

  • Apply Save Point – Applies the save points for the selected strategies.
  • Create Save Point Before Restore – Creates another save point that the restore would override when applied.
  • Save Point ID – A unique identifier for the Save Point.
  • Allocate Strategy ID – A unique identifier for the strategy line. Any active product groups will also be assigned a unique allocate strategy ID.

  • Allocate Strategy Code – The identifying code for the level ID associated with the strategy. This is the item master group code of the hierarchical level defined in the allocation hierarchy.

  • Allocate Strategy Description – The item master group description of the hierarchical level defined in the allocation hierarchy.
  • Product Group ID – A unique identifier for the product group.
  • Level ID – The identifying code for the level ID associated with the strategy. This code is the item master group code of the hierarchical level defined in the allocation.
