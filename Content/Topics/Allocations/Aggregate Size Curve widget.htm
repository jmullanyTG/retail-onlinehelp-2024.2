Aggregate Size Curves across all sites

Aggregate Size Curves are reviewed and manipulated in the Size Models screen. Aggregate Size Curves display in the middle grid.

To edit an Aggregate Size Curve, check the Override Allowed? checkbox in the top grid and click Save.

The grid has two fields Contribution and Original Contribution Quantity.

Aggregate Size Curve fields

Changes to site-level curves (eligibility, percent values) do not impact the aggregate curve.

Field Definition
Contribution Editable. The percent contribution by size across all sites. The total must add up to 100%. If not, corrections are automatically done to ensure this when the user clicks Save.
Original Contribution Quantity Read-only. The original quantity by size across all sites. This value does not respond to changes of Is Eligible? or Contribution.

Selecting aggregate size curves in Assortment Planning

System policy Use Size Curve By Site controls whether Assortment Planning uses the by-site curves or aggregate curve. This is utilized in three places:

  • In Size Break calculations in AP orders in Order Manager
  • In Pack Type calculations
  • In supersessions, if only some of the inventories for Old Variant are mapped to inventories for New Variant, and the supersession quantities need to be % split.
