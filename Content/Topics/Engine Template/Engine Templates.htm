Engine task templates

Some engine chains require a default task template to be available before they can be run. The Forecast before Replenish (FR) engine chain requires that a Replenish Task template be created before it can be used. This task is required to prevent multiple engine task forms from being prompted to the user before the task can be processed. Currently, engine templates are only available for the Forecast and Replenish engines as they are the only ones that have a user-configurable task form.

The only template that can and will be utilized is the one for the Default Replenish FRChain Template. This is provided to allow for a mostly out-of-the-box running implementation of the FRChain if it is enabled. A user with the correct permissions may need to modify some settings in the template such as which Supply Chain to use, the proper Horizon Days, etc.

The Engine Template module is located under the Management tab.

Creating task templates

There are two default task templates that can be used by the Engine to run items requested by the user without requiring any input from the user about the configuration. These task templates are configured by editing a Replenish Task form or a Forecast Task form and saving the updated configuration into the template.

  1. Select the Engine Template module to display the Engine Template Maintenance screen.
  2. Click Add to open the Add / Update Engine Template dialog.
  3. Complete the Name and Description fields as required.

  4. Select the engine type by clicking the dropdown arrow in the Engine Type field.

    The engine type selected determines which Engine Task Form displays.

  5. Select the engine sub type by clicking the dropdown arrow in the Engine Sub Type field.

  6. Click the Open Engine Task Form button.

  7. Complete the fields within the Replenish Task form or the Forecast Task form as required.
  8. Click Ok.

Editing task templates

There are two default task templates that can be used by the Engine to run items requested by the user without requiring any input from the user about the configuration. These task templates are configured by editing a Replenish Task form or a Forecast Task form and saving the updated configuration into the template.

  1. Select the Engine Template module to display the Engine Template Maintenance screen.
  2. Click Edit to open the Add / Update Engine Template dialog.

  3. Edit the Name and Description fields as required.

    On editing an existing form, the engine type and subtype cannot be changed and the fields are set to read-only.

    The engine type determines which Engine Task Form displays.

  4. Click the Open Engine Task Form button.

  5. Update the parameters as required and click Ok.

  6. Click Ok to close the dialog.
