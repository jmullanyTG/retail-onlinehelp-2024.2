Engine Template Overview

Engine Template is located under the Management tab.

There are two default task detail templates that can be used by the Engine to run items requested by the user without requiring any input from the user about the configuration. These templates are configured by editing a Replenish Task form or a Forecast Task form and saving the updated configuration into the template.

The purpose of these templates is to allow processes that should not require user input to be able to create engine tasks with appropriate predefined settings. The immediate need for the engine templates are for engine chain tasks to be created with default predefined settings that require no task form popups for the user to configure.
