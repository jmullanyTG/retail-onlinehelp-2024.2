Copying and pasting data into and within financial plans

Data can be copied and pasted either directly into a financial plan from spreadsheets and from one section to another section within a financial plan. While the copy and paste process is essentially the same as standard copying and pasting:

  • data cannot be pasted onto a non-editable cell (non-editable cells are skipped)
  • the focused cell or top-left cell of the selection on the pivot grid (in the case of a multi-select) must be editable. If the focused cell is not editable, then the Paste option is not shown.
  • if pasting onto a normal cell and then a total cell (e.g. paste into Week 2021_wk34 and 2021_m08 total at the same time) then the lower level value is overwritten by the total cell.

Copying and pasting data

To copy and paste data from a spreadsheet into a plan:

  1. Select and copy the desired cells in the spreadsheet.

  2. Optionally, if a header row is copied, right-click and select Paste Options > Remove Headers.

    Remove Headers removes the first row of values that have been copied as they are considered to be headers.

  3. Within a target cell in a plan, right-click and select Paste From Clipboard from the dropdown menu.

To copy rows or sections of data from one part of a pivot to another:

  1. Highlight the desired values.

  2. Press CTRL+ C.

  3. Highlight the desired location to paste.

  4. Ensure that Remove Headers is not checked otherwise a row is lost.

  5. Either press CTRL + V or Ctrl-V or right-click and select Paste from Clipboard.

    The values are pasted using each selected cell as a starting point.
