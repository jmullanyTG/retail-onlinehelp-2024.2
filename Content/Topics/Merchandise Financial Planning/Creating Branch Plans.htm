Creating branch plans

In the Financial Plan Maintenance view, Branch plans can be created once the Master Plan has been initialized. Branch plans are financial plans that are created from a Master plan and are directly editable.  The Branch plan is used to make changes to a financial plan.  There is no limit to the number of Branches that can be created. 

Branch plan association with Master plans

Branch plans are always associated with a Master plan.  Once a Branch plan has been created it displays underneath the Master plan it is associated with in the Financial Plan Maintenance view.

To create a branch plan:

The steps to create a Branch plan are the same as for creating a Master plan. However, the criteria in each of the tabs is pre-populated with the selection criteria of the Master plan. 

  1. Highlight the Master plan that the Branch will be created from.

  2. Click on the Branch button.

    The Branch Financial Plan window displays.

  3. Complete the fields in each tab as required.

    The two most important tabs are the Product and Channel tabs.  These tabs contain the same selections as the Master plan. However, selections can be narrowed down.

    For field definitions, see Creating a master plan.

  4. Click Ok.

  5. Click the Initialize button.

    Once the Branch has been initialized, open it and begin making changes to the plan. To open a plan, highlight the desired plan and click Ok.
