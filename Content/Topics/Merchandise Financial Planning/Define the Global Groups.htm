Defining the Global Groups

Global Groups define the available Product and Location hierarchy levels that can be selected by a super user during the creation of an MFP plan.

  1. Navigate to Plan and select Merchandise Financial Planning

    The Financial Plan Maintenance window displays.

  2. If not, select Manage.

  3. Click the Configuration button on the toolbar.

  4. Select Global Groups from the dropdown menu to open the Global Group Maintenance window.

    Add the groups to the Included section and re-order.

The number of product, channel, and calendar levels selected directly affects the size of the Hybrid Planning data sets. Strict requirements should be set as to the merchandise and location planning levels.
