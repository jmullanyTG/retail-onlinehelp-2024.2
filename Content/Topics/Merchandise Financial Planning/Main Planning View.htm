Main planning view

A plan's data is viewed and edited in the grid. As it would be impractical to display an entire financial plan on screen at once, parts of the plan can be selected to display in the grid.

Hybrid financial planning grid

The Hybrid Financial Planning Grid is used to plan sales, sales units, and average unit retails regardless of which method of inventory valuation, Cost or Retail, is in use for the active plan. Growth and decline up and down the product and location hierarchies can be budgeted for over time to determine the sales activity in these segments of the business. 

Fields can be aggregated, disaggregated, and rearranged allowing for a more intuitive and fluid process. This view works in exactly the same way as the main planning view in terms of core functionality.

The parameters selected in the main planning view are connected to the Hybrid Financial Planning view. Changes to the parameters in the Hybrid Financial Planning view header area are reflected in the main planning view as well. Additionally, the calculation toggles and the locks that are invoked are consistent with the selections made in the main planning view.

Cell colors

Data cells in the grid are color coded to indicate what type of metrics they represent and whether their values are locked.

Cell Color Description
Light blue Plannable metrics. These cells are editable. Their values represent the financial plan data.
Light purple Comparison Plan 1 metric. These cells are read-only. Their values are read directly from the comparison plan selected in the Parameters pane.
Medium purple Comparison Plan 2 metric. A core subset of metrics compared to Comparison Plan 1. Read-only and read directly from the comparison plan selected in the Parameters pane.
Dark gray Actualized metrics. These cells are read-only. Their values come from the actual sales data that has been imported into Financial Planning.
Light green Ex-VAT metrics. For plans that include VAT, these are values that exclude any amount associated with value added tax. These cells are read-only.

If the VAT % is zero for the given product/site level, ex VAT metrics will be equal to the non-ex VAT metrics (i.e. Sales = Sales ex VAT, etc.)

Calculation: The generic calculation for any ex VAT metric is = non-ex VAT metric value / (1 + VAT %)
Light red This cell value is locked. Light-red coloring is applied only to sales-related metrics. Other types of metrics do not change color when locked.

Aggregate columns

Aggregate Total columns are added to the grid whenever lower-level data is displayed. These columns follow the same color-coding scheme as other data cells, but use a gradient instead of a solid background. Examples:

Color Description
An aggregated plannable value
An aggregated comparison value
A locked aggregated sales-related value

Hybrid financial planning grid right-click menu options

Right-click inside the data grid to see the following options.

Option Description
Best Fit Columns Adjusts the column widths to best fit the column names.
Best Fit Data Adjusts column widths to best fit data values.
Open Opens grid data in the selected format.
Export Exports grid data to a file in the selected format.
Export Options Sets text export options (Value or Display Text).
Restore Layout Displays plan data in the selected layout.
  Changes the data fields displayed to those included in the selected data layout.
Save Layout Saves the current configuration of filters, pivots, and data fields.
Status Bar Toggles showing and hiding the status bar.
