Merchandise financial planning metrics list

Current period BOP and last actualized period EOP

For a master plan, the:

  • BOP for the current period comes from actuals (ILA data).

  • EOP for the last actualized period comes from the current period BOP’s actuals (it is not calculated).

  • EOP for the current period and forward is calculated.

A default branch plan's current period BOP comes from the master plan, but it uses the same actuals as the master plan, instead of calculated values.

An original plan branch's very first period’s BOP seeds from the master’s BOP for the same period.

Metric formatting and nomenclature

  • Blue rows indicate user-editable values.
  • Green rows indicate ex VAT metrics.
  • Grayed out cells indicate metrics that are not available in a particular plan type.
  • If a metric has Has LY? answered with Yes, then there is an LY version of the metric, which is named with LY prepended to the metric name.
  • If a metric has Has Comparison? answered with Yes, then there is a comparison version of the metric, which is named with the comparison plan name appended to the metric name.

Custom metric actualization

There are some metrics in MFP that do not have a standard means of actualizing. These metrics can be actualized through the Custom Metric process, which allows the ingestion of any metric into the JustEnough system at the SKU ID (e.g. Item/Site) and Date level.
