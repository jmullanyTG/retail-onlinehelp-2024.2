Working with layouts

With the wide array of metrics available in Merchandise Financial Planning, having the full set visible while working on a plan can be overwhelming. Users can select a desired subset of metrics and save them as a particular data layout. Restoring a data layout changes the fields used as metrics in the data area of the planning grid without changing the current filters and pivot fields.

A layout is the combination of pivots, filters, and data fields (metrics) that appear in the Hybrid Financial Planning Grid view. Layouts can be customized by adding or removing fields, applying filters, and aggregating or disaggregating metrics. Custom layouts can be saved for later use.

For more information, see Adding, removing, and rearranging fields.

Saving layouts

  1. Use the Fields toolbar option to add or remove metrics as desired.

  2. Right-click in the data area of the grid and choose Save Layout.

  3. In the Save Layout dialog, enter or confirm the following layout properties:

    • Name: The name to give the layout.
    • Description: An optional description of the layout.
    • Group: The folder in which to save the layout.
    • Default: Whether this is to be the default layout for all financial planning documents.
    • Private/Public/Shared: The privacy level for the layout.
      • Private: The layout is only available to the user who created it.
      • Public: The layout is available to everyone in the organization.
      • Shared: The layout is only available to certain roles and users. Click the ellipsis to select who to share with.
  4. Click Ok to save the layout.

Restoring previously saved layouts

A previously saved layout can be opened using the Restore Layout context menu. Right-click in the data area of the grid, choose Restore Layout and select a layout.

Data layouts

The Restore Data Layout context menu option changes the fields used as metrics in the data area of the planning grid without changing the current filters and pivot fields.

  1. Right-click in the data area of the grid and choose Restore Data Layout.

    Layouts may be stored in a group. In the following screenshot, layouts are stored in the group My Layouts.

  2. Select the layout to be restored.

    The data fields in the grid are replaced with the data fields from the selected layout. The pivots remain unchanged.
