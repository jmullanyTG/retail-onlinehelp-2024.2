Changing multiple cells at once

Multiple cells can be changed in a single action by selecting the desired cells and right clicking to bring up a context menu.

Click Multiple Changes to launch the Multi Data Change Form For Column dialog.

Choose a number of different actions by which to manipulate the selected cells:

  • New Value: A new value with which to overwrite the values of the selected cells.
  • Increase/Decrease: Increases or decreases the selected cells by the specified amount or percent.
  • Split Value: Splits the entered value across the selected cells either as a weighted split based on the current cell values or evenly.

  • Copy From: a number of cells equal to the number currently selected can be chosen to copy values from.

    1. Click the Select Fields button.
    2. Select the desired cells in the view.
    3. Click Ok.
    4. Review the value mappings in the Results Preview section.
  • Round new value: A rounding method for the new values: up, down, or nearest. This only applies when new values have decimals.
