How FX neutralization impacts calculations for historical and future periods

In the following, the example metric used is Sales and the example plan currency is USD. However, the examples apply to all metrics that can be FX neutralized and all plan currencies. For a list of metrics, see Metrics Impacted by FX Neutralization.

Historical periods

If historical periods are brought back to display in MFP, the metrics function as follows:

  • Sales = sum of sales using the historical period conversion rate
  • LY Sales (standard mode) = sum of sales using the historical period’s LY conversion rate
  • LLY Sales (standard mode) = sum of sales using the historical period’s LLY conversion rate
  • LY Sales (neutralized mode) = sum of sales using the historical period conversion rate
  • LLY Sales (neutralized mode) = sum of sales using the historical period conversion rate

Future periods

If future periods are brought back to display in MFP, the metrics function as follows:

  • Sales = planned value
  • LY Sales (standard mode)
    • If the LY period is a past period (e.g. within 1 year of the current period) = sum of sales using the period’s LY conversion rate
    • If the LY period is a future period (e.g. after 1 year of the current period) = planned value
  • LLY Sales (standard mode)
    • If the LLY period is a past period (e.g. within the first 2 years of the current period) = sum of sales using the period’s LLY conversion rate
    • If the LY period is a future period (e.g. after the first 2 years of the current period) = planned value
  • LY Sales (neutralized mode)
    • If the LY period is a past period (e.g. within 1 year of the current period) = sum of sales using today’s conversion rate
    • If the LY period is a future period (e.g. after 1 year of the current period) = planned value
  • LLY Sales (neutralized mode)
    • If the LLY period is a past period (e.g. within the first 2 years of the current period) = sum of sales using today’s conversion rate
    • If the LY period is a future period (e.g. after the first 2 years of the current period) = planned value
