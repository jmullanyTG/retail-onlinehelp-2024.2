Metrics impacted by FX neutralization

Designated LY metrics either display non-FX neutralized numbers if the selection in the Currency Neutralization dropdown is set to Standard or they will display FX neutralized numbers if Currency Neutralization is set to LY Neutralized.

FX neutralized and non-FX neutralized are not simultaneously visible on the same view. To view both, the user must render the view with the first option in the Currency Neutralization toggle chosen, save off the data, and then re-view the data with the second option chosen.

The metrics that can be FX neutralized

The following metrics are FX neutralized when the LY Neutralized option is chosen:

  • AUR % +/- LY
  • AUT % +/- LY
  • Comp Sales % +/- LY
  • LY AUR
  • LY AUT
  • LY Comp AUR
  • LY Comp Sales
  • LY Markdown AUR
  • LY Markdown Sales
  • LY Markdowns
  • LY Markdowns % of Sales
  • LY Markdowns (ex VAT)
  • LY Non-Comp AUR
  • LY Non-Comp Sales
  • LY Promo AUR
  • LY Promo Sales
  • LY Reg AUR
  • LY Reg Sales
  • LY Sales
  • LY Sales (ex VAT)
  • LY Sales Build Ratio
  • LY Total Original Retail
  • LY Total Original Retail (ex VAT)
  • Markdown Rate +/- LY
  • Markdown Sales % +/- LY
  • Markdowns Var to LY
  • Non-Comp Sales % +/- LY
  • Promo Sales % +/- LY
  • Reg Sales % +/- LY
  • Sales % +/- LY

Metrics calculated based on FX neutralized and non-FX neutralized values

The following metrics are calculated based on a mix of FX neutralized and non-FX neutralized values, meaning they are computing metrics in “mixed” mode.

For example, for “LY COGS % of Sales,” LY COGS is converted to plan currency using the historical period’s conversion rates, but LY Sales is converted to plan currency using the viewing period’s conversion rates, so the ratio between the two is comparing a historical exchange rate number to a viewing period rate number.

  • COGS % of Sales +/- LY
  • IMU Rate +/- LY
  • LY COGS % of Sales
  • LY GMROI
  • LY IMU %
  • LY IMU Retail
  • LY Gross Margin % (Product)
  • LY Gross Margin (Product)
  • LY Gross Margin (Total)
  • LY Shrink Retail % of Sales
  • LY Stock to Sales
  • Gross Margin Rate +/- LY (Product)

If there are custom metrics that use any of the above in their calculations, then those custom metrics would also be affected by FX neutralization.

No other metrics such as COGS, AUC, EOP Cost / Retail, and BOP Cost / Retail will be FX neutralized. These metrics always show values with their LY currency conversion rates.
