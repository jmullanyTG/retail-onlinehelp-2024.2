MFP setup and configuration

The Merchandise Financial Planning module is designed with the end user in mind - combining essential planning functionalities, such as Cost and Retail methods of accounting, calendar and period level configurability, and plan approval workflow with an innovative approach to data management and user interaction.

During functional solution design, many important decisions will be made that will need to be well-defined prior to calculating the first plan. Even then, subsequent decisions will require changes to the configuration. The following is a road map to fine-tuning the client’s experience. It is important to understand a few parts of the functional design before initializing the first plan.

Initial Considerations:

  • Will the client use the Retail or Cost accounting method?

  • Which plan type will they primarily use?

  • How will they seed the initial plan?

  • Do they plan in multiple currencies?

  • Do they plan for VAT?

  • From which levels of the merchandise, location, and calendar hierarchy will they plan?

  • Which sales types will be plannable?
