Customizing the financial planning screen

The Financial Planning module makes it easy to design work spaces and layouts that drive versatility and take the pain out of switching between different areas of a financial plan.

The three ways to customize the Financial Planning screen are:

Work spaces

Work spaces represent the arrangement of tabs and panes on and off the screen.

To save or retrieve a changed work space, right-click above the grid and select an option.

Layouts

A layout is the combination of the row and column groupings that create the pivot grid (typically product, channel, and time period) plus any filters on row and column headers plus the data fields (metrics) whose values appear inside the Hybrid Financial Planning Grid.

Data layouts

A data layout is the set of data fields contained in a given layout. Restoring a data layout changes the metrics displayed in the grid while leaving current pivots (row and column groupings) and filters intact.

Data layouts cannot be saved by themselves. Instead, the entire layout is saved. It then appears as an option in the Restore Data Layout menu.

Work spaces, layouts, and data layouts can be mixed and matched. For example, the following can be created:

  • a workspace that maximizes the Hybrid Financial Grid by hiding the Calculation Direction Toggles and Locks On Selected Data panes.
  • a layout that contains the pivots (channel, product, and time periods) and filters for the areas of the plan the user is responsible for, along with the basic set of metrics to work on.
  • additional layouts containing different data sets.

To see a different set of metrics while preserving the current workspace, pivots, and filters, restore the data layout that contains those metrics. Only the data fields displayed in the grid change as workspace, pivots, and filters persist until users change them.
