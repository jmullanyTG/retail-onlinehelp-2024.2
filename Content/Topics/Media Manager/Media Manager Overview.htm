Media Manager Overview

Media Manager supports the management of product media such as images for the marketing and advertising teams. Digital assets are managed from a centralized dashboard including product images, manufacturer logos, and icons. The digital assets can be used for display in different channels. Thumbnails, low and high resolution images, and print-ready images are supported.

To access this solution, click Media on the Management tab.

File Storage API

The File Storage API can handle Uniform Resource Identifiers (URI) allowing for the import and usage within ToolsGroup JustEnough of a user's images. Users need to supply the image name, image file extension type, and the image's URL.

Additionally, Repository Manager can accept URIs and pass the images to File Store API for storage and usage. For Repository Manager, users need to provide the name of the file the image should be stored in.

There is a check to verify if the image's URL has changed and whenever the image is accessed past a specified amount of time. When the URL is changed for the document ID, the cache of the image is cleared and re-imported.

Supported image file extension types:

  • BMP

  • GIF

  • JPG

  • JPEG

  • PNG

Non-image files are not supported. Attempts to import unsupported files are blocked and a warning to users is generated.
