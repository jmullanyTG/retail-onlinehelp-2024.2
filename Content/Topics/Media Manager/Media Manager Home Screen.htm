Media Manager home screen

The Media Manager home screen displays all Media in alphabetical file name order. Placeholders for new image requests display here. The screen displays a thumbnail of the Media along with other file attributes. Buttons are located above the Repository grid that provide the ability to add and remove Media.

The Repository grid displays the following information for each stored media:

Field Description
Preview A thumbnail image representing the image or document. Double-click an image to open the Image window.

Use the Image window to view different sizes of the image.

Buttons Description
Fit Image Adjusts the image’s dimensions to fit in the window.
Actual Size Image Displays a full size version of the image in the window.
Name The file name of the media.
File Type The file type of the media.
Storage Type The type of storage where the Media was originally located.
  • Local DB Storage – All Media added manually
  • File Storage – Media imported via Repository import
  • Read Only From File System – Read-only Media imported via Repository import

There are two buttons that provide additional information regarding each existing Media:

Action Description
View A window displaying a larger preview of the image. Click View to open the document.
Attributes Displays the file attributes.
