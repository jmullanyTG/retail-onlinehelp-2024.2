Generating distribution orders

Distribution orders move items from distribution centers (DC) to stores, DCs to DCs, or stores to stores.

Setup

The following objects must be set up before generating a distribution order:

  • a filter that includes the items and/or sites to distribute.
  • the supply chain to use.
  • demand forecasts for the items to distribute.
  • stocking strategies for the items to distribute.

To generate distribution orders for items

  1. From the Execute tab, click Order Manager.

    The Navigation menu displays.

  2. In the Order toolbar, click Distribution.

    The Items window displays.

  3. In the Filter field, select the filter with the items and sites to include in the order.

    The Add Distribution Order: Distribution window displays.

  4. Select:

    • the item to be distributed from the Supply Chain dropdown menu.
    • what type of orders to distribute (for example: Today's Orders, This Week's Orders, and so on).

  5. Click Ok.

    The recommended orders calculate and display on the screen. Each row represents a different recommended order and contains basic details of the order.

  6. Double-click an order row to view the demand forecast, stocking strategy, stock holding projection, and other details that drove the order recommendation.
