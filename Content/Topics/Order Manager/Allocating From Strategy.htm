Allocating from strategy

  1. From the Execute tab, click Order Manager.
  2. In the toolbar, click From Strategy.

  3. In the Items dialog box, select the Item Master filter and Site filter to use for this order.

  4. Click Ok.

    The Add From Strategy Allocation Order: From Strategy Allocation window displays.

  5. Enter values as required.

    Parameter Description
    Workflow The work flow to follow for this order.
    Name The order document name.
    Description The order description.
    Owner The owner of this order.
    Security

    Whether this order is public, private, or shared with select others.

    • Private: Only visible to who created the order.
    • Public: Visible to all users.
    • Shared: Visible only to the roles and users selected by clicking the Change button.
    Today's Orders
    This Week's Orders
    Future Recommended Orders
    		 The order horizon to use for this order.
    Include Zero Orders Select to enable this option for this order.
    Exclude Uploaded Orders Select to enable this option for this order.
    Macro Optional. The macro to run with this order. Click Disable to toggle enabling and disabling the selected macro.
  6. Click Ok to generate the order.
