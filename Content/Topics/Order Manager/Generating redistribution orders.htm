Generating redistribution orders

A redistribution order is used to pull excess or leftover stock from stores and redistribute the stock to stores with unfulfilled demand. Redistribution orders are primarily filter driven to determine eligible items. Users select the desired inventories while the quantities to redistribute are driven by the source stock selection options.

Redistributions can only be executed across a sibling level tier within the normal supply chain (that is a DC may not redistribute to a store, as that is a normal distribution. A store may not redistribute to a DC as that is a reconsolidation action. To resolve these tiers, the Replenish Engine references the default supply chain defined using system policy Default Supply Chain (which is of type site/supplier).

Setup

The following objects must be set up before generating a redistribution order:

  • a filter that includes the items and sites to redistribute.
  • the supply chain to use.
  • demand forecasts for the items to redistribute.
  • stocking strategies for the items to redistribute.

To generate a redistribution order for an item

To action a redistribution order, users should select the order type within Order Manager.

  1. From the Execute tab, click Order Manager.

    The Navigation menu displays.

  2. Click the Redistribution option on the toolbar.

    The Items window displays providing the option to select a filter that determines the eligible items. Inventory filter types are available. A location dimension element can be added to any product filters if desired.

  3. In the Filter field, select the filter with the items and sites to be included.

    The Add Redistribution Order: Redistribution window displays.

    For field descriptions, see Generating rebalance orders.

  4. Select:

    • the item to be redistributed from the Supply Chain dropdown menu.

    • what type of orders to redistribute such as Today's Orders, This Week's Orders, etc.

      Selecting more than one source stock causes the minimum value (including zero) to be considered when building the redistribution order. As such, it is considered best practice to select a single source stock per redistribution order.

  5. Click Ok.

    The recommended orders display. Each row represents a different recommended order and contains basic details of the order.

  6. Double-click an order row to view the demand forecast, stocking strategy, stock holding projection, and other details that drove the order recommendation.
