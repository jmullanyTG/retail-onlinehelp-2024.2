Content Manager configuration

The Configuration option is only available to users with the Allow Access to Content Config. For more information, refer to the Content Manager section in the User Guide.

Permissions and policies

  • To configure user and role-based access to content-related features and functions, see Content Manager user rights.
  • To configure global policies related to content management, see Content Manager global policies.
  • To configure user and role-based edit privileges for content fields in a content version, see .

The Content Manager Configuration window

Configure content fields and content versions using the Content Manager Configuration window.

Accessing the Content Manager Configuration window

To open the Content Manager Configuration window:

  1. Click Content.

  2. In the Product Content pane toolbar, click Configuration.
