Content fields

Content Fields define the Copy Text Content form for a Promotion and are set up directly in the database.

Contact Professional Services or Support to set up Content Fields. Depending on the Content Type, some Content Fields may be modified in the Content Fields configuration screen. After changes are made to Content Fields, Event Promotion Content will need to be refreshed manually.

Content Fields cannot be deleted using Content Manager configuration since they are created directly in the database.

Modifying content fields

  1. To modify a Content Field, select the field and click Edit.

  2. Update the Content Field name and Content Type as required.

  3. Select the Read Only if the field is to be read-only to users within the Content windows accessed from Promotion, Event and Content Manager.

  4. Select the Active checkbox to display the Content Field.

    If unchecked, the Content Field is hidden.

Content Type descriptions

Field Description
Content Type If the Content Field can be configured further, this column appears in the grid as a dropdown list. The following Content Types are available for a Content Field. For fields that can be modified, the default value is listed first:
  • Checkbox (read-only) - This option, determined by the database configuration (i.e. not modifiable), displays a checkbox in the user interface.
  • Date and Time (read-only) - This option, determined by the database configuration (i.e. not modifiable), displays a date and time field that displays a calendar and clock interface when edited in the user interface.
  • Droplist (editable) - This option displays either a single droplist value or multiple droplist values in the user interface.
    • Droplist (Single), Droplist (Multiple)
    • The blue pencil icon indicates that the Droplist values are editable
  • Hyperlink (read-only) - This option, determined by the database configuration (i.e. not modifiable), displays single text box with a clickable hyperlink.
  • Numeric (read-only) - This option, determined by the database configuration (i.e. not modifiable), displays a field that only accepts numeric entry in the user interface.
  • Text (editable) - This option displays either a multiple line text box or a single line text box in the user interface.
    • Text (Long), Text (Short)
