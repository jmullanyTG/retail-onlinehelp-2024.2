Content details

When a product is in Edit mode, the selected item's details display in the top right. By default, the first Content Version is selected and the Text tab displays. The Text tab displays the product’s copy text for the selected Content Version. Each Content Version can have its own unique set of Copy Text fields. Copy text configured here is the Master Copy Text for the product to be used when initializing Promotion and Event Copy Text.

Copy text is modified by selecting the Content Version and updating the text in the appropriate field.

The Images tab can be selected to view Master Images. For more information on working with images, see Images.

Updating content fields

  1. Complete the fields in the Content Details pane as required.
  2. Click Save.
  3. In the Select Content to Update dialog, update the fields as required.

  4. Click Ok.

    Only modified Content fields are updated to prevent unchanged fields from overwriting modified content. This is implemented at all levels (Master, Event, and Content) as well as updating the modifications to the other levels.

Button Description
