Add existing images

If the image already exists in the database:

  1. Click Add Existing to display the Repository Explorer window.

  2. Select the checkbox next to images to be associated with the content version.

  3. Click Ok.

Previewing selected images

To preview selected images, in the Images tab click View. This displays the image with two tabs.

Tab Description
Fit Image Adjusts the image dimensions to fit in the window.
Actual Size Image Displays a full size version of the image in the window.

Functionality

The following functionality is available.

Functionality Description
View Clicking View or double-clicking the image displays a larger preview of the image in a window.

Attributes Displays the Attributes window for the selected Image. If more than one Image is selected, attributes display for the last selected image in the grid.
