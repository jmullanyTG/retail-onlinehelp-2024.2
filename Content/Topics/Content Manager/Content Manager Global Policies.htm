Content Manager related global policies

Global policies are configured to set up default settings. They are configured by Professional Services and should not be modified without assistance from ToolsGroup JustEnough Professional Services or Support. Changes to some Policies may require closing and opening the application before the changes are available. It is recommended to restart the application after any Policy change.

To configure Content Management policies, click Policies and choose Module: Content Manager in the navigation pane.

Policies

System Policy Description
Suppress Content Image Card's Add Option If set to True, prevents the adding of a local or network image to Content Manager by hiding the Add button from the Images toolbar.
