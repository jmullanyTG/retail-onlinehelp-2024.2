Group pane

The Product Group Details pane provides the ability to modify the child products in a product group or to clone a product group. To edit a product group, select it in the Groups pane and double click or click Edit to load it in the Group pane on the right.

Select the following to review details or edit product groups:

Button Description
Edit Opens a dialog for editing the contents of the product group. For more information, see Editing child products.
View Products Displays the Item Masters or Variants contained within the selected child group. For more information, see Viewing child products .

This button is not available if the child group contains other product groups.
