Editing child products

Click Edit in the Group pane to modify the child products in product groups. The Edit Group window differs depending on the way the product group was created.

Template filter product groups

When a product group is created using a Template Filter, the template values are transformed into a standard filter. Modifying a product group created using this method is the same as modifying a product group created using a Product Filter.

The Edit Group window displays the filter created from the Template Filter.

Action Description
Group Name Modify the defaulted Group Name as needed.
Product Filter Select a different existing filter from the dropdown list if needed. A blue hammer icon displays. To modify the selected filter click on the blue hammer icon. The black hammer icon provides the ability to modify the existing or create a new filter.

Filter product groups

If the product group was created using a filter, template or product, the Edit Group window displays the filter.

Child product groups

If the product group consists of other product groups, the Edit Group window displays the child groups. In addition to the standard functionality:

Action Description
Edit Select a child group before clicking on this button. If the selected child group contains Item Masters or Variants, the child group filter displays in a second Edit Group window.

If the selected child group contains other product groups, a second Edit Group window displays listing the child’s product groups. Modifications can be made as needed.

Modifying an existing filter permanently updates the filter and affects other product groups using the filter.
Remove Removes the selected child group from the product group.
View Products Select a child group and click View Items or View Variants to view the Item Masters or Variants respectively that comprise the child group.

This button is not available if the child group contains other product groups.

Click Ok to save the updates.
