Product groups

A common advertising set of content, text, and images can be assigned using product groups consisting of Item Masters, Variants or other product groups.

The following functionality is available:

Functionality Description
New Provides the ability to a new product group. There are several ways to create a product group. For more information, see Create a product group.
Clone Provides the ability to create a copy of the product group. The cloned product group can be modified as needed. For more information, see Cloning product groups.
Edit Displays the Content Details and Product Details pane with the selected product group. For more information, see:
  • Content Details
  • Product Group Details
Delete Provides the ability to delete the selected product group. For more information, see Deleting product groups.
Filter Provides the ability to limit the display of product groups to those containing the Items or Variants returned by the selected Filter. The black hammer icon provides the ability to modify the existing or create a new filter.
