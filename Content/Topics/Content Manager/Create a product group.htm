Creating product groups

Product groups can consist of Item Masters, Variants, or other product groups.

Template filters

Customized client specific filters (Template Filters) are created by Professional Services. Template Filters are used to create product groups using a predefined filter form. All components of a Template Filter are custom.

Template values are transformed into a standard filter when a product group is created using a Template Filter.

Product filters

Select the Use Product Filter radio button option to filter products. The filter returns a desired sub-set of products to be included in a product group.

Child groups

Select the Use Child Groups radio button to create a product group containing other product groups.

In addition to the standard functionality:

Button Description
Edit Select a child group before clicking on this button.

If the selected child group contains Item Masters or Variants, the child group filter loads in a second Edit Group window.

If the selected child group contains other product groups, an Edit Group window will load listing selected child’s product groups and modifications can be made as needed.

Modifying an existing filter will permanently update the filter and will affect other product groups using this filter.
Remove Select a child group in the grid and click Remove to remove it from the product group.
View Products Select a child group and click View Items or View Variants to view the Item Masters or Variants respectively that comprise the child group.

This button is not available if the child group contains other product groups.

Creating product groups

  1. Click New in the Product Groups pane in the lower left of the screen.

  2. Choose one of the radio buttons.

  3. Add a name and select a filter.

  4. Click Ok.

    The newly created product group and any child groups appear in the Product Groups pane.
