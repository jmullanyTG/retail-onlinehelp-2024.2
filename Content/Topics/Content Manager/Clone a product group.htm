Cloning product groups

To clone a product group, click Clone. The Clone Group window differs depending on the way the product group was created.

Groups with no child groups

Template filter product groups

When a product group is created using a Template Filter, the template values are transformed into a standard filter. Cloning a product group created using this method is the same as cloning a product group created using a Product Filter.

Filter product groups

To clone a Filter Product Group, click Clone, make the necessary changes to the Clone Group window and click Ok to save.

Groups with child groups

Field Description
Group Name Modify the default Group Name as needed.
Add Group Select one or more child groups to add to the new product group.
Edit Select a child group before clicking on this button. If the selected child group contains Item Masters or Variants, the child group filter displays in a second Edit Group window.

If the selected child group contains other product groups, an Edit Group window displays listing selected child’s product groups. Modifications can be made as needed.

Modifying an existing filter permanently updates the filter and affects other product groups using this filter.
Remove

 

 Select a child group in the grid and click Remove to remove it from the product group.
View Products Select a child group and click View Items or View Variants to view the Item Masters or Variants respectively that comprise the child group.

This button is not available if the child group contains other product groups.
