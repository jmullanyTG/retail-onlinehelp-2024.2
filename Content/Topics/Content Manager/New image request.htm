Requesting new images

If the required image does not exist, it can be requested by clicking Image Request > New on the toolbar to display a dialog. Complete the fields as required and click Ok to initiate the request.

Identifying images types using placeholder images

New image requests are indicated by varying placeholder images depending on the file type (for example, JPEG or GIF) of the image requested.

Modifying new image requests

It may be necessary to update a request for a new image to change the status, provide additional information or to modify the file name. This can be done by selecting the request and clicking Edit.

The Edit button is only available when a New Image Request is selected.

The New Image Request window displays and information can be modified as necessary. Click Ok to save the updated request.
