Content versions

Content versions are used to create variations in product content due to language, geography, or other advertising considerations. Each content version can have its own set of fields and be assigned to one or more site clusters.

  • Site clusters are logical groupings of stores created in the Site Clustering module. For more information, see Site Clustering Overview.
  • When adding a promotion to an event, all content versions are associated to the event by default unless Version Prompt has been enabled for the event slot set.

Content versions are configured by Professional Services and should not be modified without assistance from Professional Services, Support, or a trained on-site resource.

Content Versions tab

Manage content versions from the Content Versions tab of the Content Manager Configuration window. For more information, see Content Manager configuration.

Content Version actions

Action Description
New Creates a new content version.
Edit Edits the selected content version.
Delete Deletes the selected content version. A content version cannot be deleted if it is associated with a promotion.

Content Version fields

Field Description
Content Version Name The name of the content version.
Content Version Description The description of the content version.
Geography list The site clusters assigned to the content version.
