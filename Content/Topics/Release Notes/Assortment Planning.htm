Assortment Planning

Extended multiple currency planning

TG JE Assortment Planning v2024.1 separated out AUT into Home AUT and Blended AUT as part of an effort to better support planning in multiple currencies.

Please refer to the TG JE v2024.1 release notes for an extensive description of TG JE's ability to plan in multiple currencies via a Price Matrix.

TG JE v2024.2 extends this functionality, the Blended AUT metric has been added to the:

  • Historical section on the Hindsight Management screen.

  • Image view (View Options > Quick Text drop-down menu).

  • Assortment Planning Smart Start functionality. (Smart Start functionality is not enabled within the standalone Target Choice Count functionality.)

    Blended AUT and AUR are derived from a Home AUT. This means that the Smart Started AUR value is not guaranteed to match the planned scenario AUR.
