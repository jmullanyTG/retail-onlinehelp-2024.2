Profiles

A new Profile Series labeled Applied Profile Value with Causal has been added to the grid in the Profile Set Item Details section of the Profiles module. This new Profile Series allows users to see the impact of an applied manual causal.

Series Description
Applied Profile Value with Causal The Profile Value with manual causal affect applied. The manual causal affect is a percentage multiplier.

This value matches the Profile Forecast series found in the Level View within the Inventory Forecast module.
