Installation Notes

Refer to the ToolsGroup JustEnough Installation Guide for detailed instructions and information related to upgrading from prior versions of ToolsGroup JustEnough as well as installing new instances.

Installing multiple Client versions on the same system

Users who want to compare an existing installed version of the software alongside an upgrade version can now install the upgrade client UI version without uninstalling the existing client UI. The Microsoft Installer (MSI) verifies down to the Cumulative Update version, so if v2022.4.7.0 client is already installed, then the v2024.2 client can also be installed for a side by side comparison.

The requirement to uninstall any previously installed versions of the Application Server remains unchanged. Only one instance of Application Server may be installed at any one time.

The MSI still generates a message if users attempt to install a version that is identical to the version already installed.
