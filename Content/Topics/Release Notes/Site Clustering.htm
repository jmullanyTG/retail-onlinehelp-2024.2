Site Clustering

Select new Attribute Type Financial Plan Attribute, via either the Add Attribute and Edit Attribute dialogs, to specify a site-level MFP plan as a sales attribute and a time range to be used in conjunction to cluster sites with.

 

The hybrid MFP plans listed are those plans that are in status Initialized, In Progress, or Authorized.

The hybrid MFP plans must have the exactly the same product hierarchy levels down to the selected clustering level. These plans can have lower levels of the product hierarchy as well but can’t be at a higher level than the required clustering. Time range selections need to be within the time range of the financial plan.
