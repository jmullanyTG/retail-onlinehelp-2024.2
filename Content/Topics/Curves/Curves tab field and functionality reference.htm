Curves tab field and functionality reference

Toolbar functionality

Details: Toggles the build and period to mean y-axes.

Filter configuration: Displays all available groups (item master, inventory, and site) for inclusion. The order setting allows users to effectively construct their hierarchy (product, site, etc.) and include the appropriate values for the curve filters.

Tag configuration: Curve tags are used to give the individual curves searchable attributes. Curve tags can be added or removed within the Maintain Curve Tags wizard.

Curves section

Tags: A dropdown list of filters that allows users to select particular curves to view.

Field descriptions

  • Name – The name for the individual curve. Can be concatenated out of several associated tags.
  • Generation Type – The method by which the curve was generated.
    • Manual Curve – Values must be entered manually when creating a manual curve. A default value can be specified to set the scale.
    • Specified Conditions – A generated curve using a selected data sample.

  • Bucket Type – The type of time period used for the x-axis.

    Bucket types can be daily, weekly, or monthly, but should be selected to support the data.

  • Start Date – The effective start date for the sample.
  • End Data – The effective end date for the sample.
  • Number of Periods – The number of periods contained in the curve.

Curve Details section

The Curve Details section provides a graphical representation of the curve series values and a grid view.

Series values in the curve details display in a graphical format and in a grid.

  • Bucket – Associated time period (buckets) for the x-axis of the line graph display in the grid.
  • Calculated – The system generated value for each period within the curve.
  • Effective – Manual adjustments are represented as a value in the Effective series which can be compared to the static values in the Calculated series.
  • Build – Represents a multiplier for the period value’s relationship to the previous period value. This lets users see how drastically the curve values change. This is not applicable to the first period point in the curve (anchor point).
  • Contribution % - A percentage for each period of the representative value of the effective value when compared to the whole (e.g. a flat line curve with 10 periods would have a contribution of 10% for each period).
  • Period to Mean – The period to mean y-axis shows a graphical representation of the distance to the mean by period.
