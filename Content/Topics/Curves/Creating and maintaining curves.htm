Creating and maintaining curves

Curves are generated using data that has been imported into the system. The data can also be imported as-is using the standard import mechanism.

Curves are a representation of the available data and not the interface though which the data is applied.

Tags are used within the curves module as an attribute to help sort and categorize curves. They are distinctly different from the other forms of tags within the system.

Interacting with curve values

Interaction with curve values in the curve detail section is limited to two actions: manipulation of the effective series and the build series values.

Curve are graphically represented using an interactive line graph. Values can be manipulated by dragging bullet points up or down. The curve values are re-calculated simultaneously. Grid values change dynamically when manual adjustments are made to curves.

Curves can be manually edited in the grid as well as in the graph.

Creating and maintaining curves

The curve can be named and security rights added at the time of creation. If desired, tags may be applied by clicking the Tag Links button and adding tags using the Maintain Curve Tag Links dialog.

Remaining details are added in the second wizard screen. Filter values must be entered.

Once built, click the Preview Curve button to preview the curve.
