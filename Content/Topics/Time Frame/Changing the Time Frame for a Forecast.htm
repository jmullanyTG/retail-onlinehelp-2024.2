Changing the time frame for a forecast

To change the time frame used to generate demand forecasts for an item (for example, April 1 - 30 instead of the next 90 days), apply the desired forecast time frame to the item. The next time that a forecast is generated for that item, it will be for that time frame.

An item's forecast time frame is currently visible only in the Forecast Phase Time Frame Maintenance view.

Setup

The following objects are used when changing an item's forecast time frame.

  • An item filter containing the desired items.
  • The user chosen time frames.

To change the time frame for a forecast:

  1. From the Plan tab, click System Utilities > Views to open the Views screen.

  2. In the Navigation Menu pane, find the Forecast Phase Time Maintenance view, then double-click on it to open it.

  3. In the Forecast Phase Time Frame Maintenance document:

    1. Select a filter containing the relevant items.

    2. In the items' Phase Time Frame field, select the time frame to be used when generating forecasts for the item.

  4. Click Save in the grid toolbar to apply the changes.

Simultaneously changing multiple records

  1. In the grid, select the records to change.

  2. Right-click on the field value to change and choose Multiple Changes.

  3. In the Multi Data Change Form dialog box, enter the new value for the field.

  4. Click Save to apply the change.

    The new values appear in the selected records.
