Selecting a time frame

In modules which use time frames, choose a time frame from the Time Frames dialog box.

  1. Select the time frame field and click the ellipsis to open the Time Frames selector.

  2. In the Time Frames selector, choose a time frame.

    If none of the available time frames is suitable, click (Maintain Time Frames) to open the Time Frame Maintenance window. From there, create a new time frame or edit an existing time frame.

    For more information, see Creating and editing time frames.
