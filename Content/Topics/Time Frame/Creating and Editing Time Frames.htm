Creating and editing time frames

Time frames are managed using the Time Frame Maintenance view.

Accessing the Time Frame Maintenance view

Either:

  • navigate to the Management tab and click Timeframes, or
  • in a Time Frames field, click (Maintain Time Frames).

Creating time frames

To create a new time frame:

  1. In the Time Frame Maintenance view, click the Add button to open the Time Frame Editor window.

  2. Enter a name and description for the new time frame.

  3. In the Security field, click the dropdown arrow and select a security level.

  4. Select a Time Frame Definition Type in the Time Frame Definition Type field.

  5. In the Time Frame Definition section, click Add in the toolbar to add a blank row in the grid.

    The fields that display in the Time Frame Definition section vary depending on the type of time frame definition selected. For more information on the different fields, see Time Frame Maintenance view.

  6. Optionally, define multiple time frames by selecting the Allow Multi-Definition checkbox.

    If Allow Multi-Definition is selected, click Add to create additional time frame definitions. This option cannot be changed once a time frame is created.

  7. Click in each time frame definition field to define the time frame.

    The fields displayed depends on the time frame definition type selected.

  8. If required, click Add to define a time frame to be excluded in the Time Frame Exclusion section.

    Multiple Exclusions can be created that are identical and can overlap. However, the end date cannot be before the start date.

  9. Click Ok.

    the Time Frame Maintenance window view refreshes and displays the new time frame as a row in the Time Frame grid. Selecting the new time frame displays its details in the other sections.

Editing time frames

Select the time frame in the Time Frame Maintenance grid and click Edit to open the Time Frame Editor window. For field descriptions, see Time Frame Maintenance view.
