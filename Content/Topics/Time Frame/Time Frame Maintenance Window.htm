Time Frame Maintenance view

Time frames are managed using the Time Frame Maintenance view.

For more information on working with Time Frames, see Creating and editing time frames.

The Time Frame Maintenance view contains a toolbar and the following:

  • a grid listing all available time frames.
  • the Time Frame Definition pane listing the effective time ranges for the time frame selected in the grid.
  • the Time Frame Exclusion pane
  • a calendar with the effective date ranges for the time frame selected in the grid.

Accessing the Time Frame Maintenance view

There are two ways to access the Time Frame Maintenance view:

  • Navigate to Management > Timeframes.

  • From a Time Frame field, click (Maintain Time Frames). For more information, see Selecting a Time Frame.

Time Frame grid details

The read-only grid lists all available time frames. (Time frames for which the logged-in user does not have access do not appear in the grid.)

Field Description
ID The unique ID assigned to the time frame.
Name The name given to the time frame.
Description An optional description of the time frame.
Dynamic Rule If checked, the time frame covers rolling time ranges, such as last 90 days or next 30 days.

If unchecked, the time frame covers static dates only, such as "March 1, 2023 - March 31, 2023".
Allow Multiple Definitions If checked, the time frame can include more than one defined time range. Each time range appears in its own row.

A time frame covering the week after Christmas for the years 2022-2024 can include the following static time ranges: "December 26-31, 2022", "December 26-31, 2023", "December 26-31, 2024".

User Access Who has access to this time frame.
  • Private: Only the logged-in user access.
  • Public: All users have access.
  • Shared: Only selected users have access.

Time Frame Definition pane

This pane displays the definitions of the selected time frame. The fields shown depend on the time frame definition type (static, dynamic, or schedule) that was selected for the time frame when it was created.

Time Frame Exclusion

The Time Frame Exclusion section provides a way to quickly and easily add static exclusion dates to time frames. To exclude a sub-set of a time frame (for example, consider the last 3 years but exclude every December) over multiple time frames, click Add to set start and end dates.

Multiple Exclusions can be created that can be identical and can overlap. However, the end date cannot be before the start date.

Calendar

The calendar section contains two calendars that show the effective time periods covered by the time frame selected in the grid. The calendar on the left shows time periods at different scales. Right-click on the calendar’s header and select one of the menu options.

The calendar on the right shows time periods by month. Both calendars track each other. If the selected time frame allows multiple definitions, the calendars use the same color coding for each included time range (see screenshot below).
