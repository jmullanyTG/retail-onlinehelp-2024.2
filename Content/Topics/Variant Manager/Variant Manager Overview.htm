Variant Manager Overview

The Variant Manager screen is used to view and manipulate data specific to the variant entity which is used extensively within the planning functions of ToolsGroup JustEnough.

For more information on blades other than Rate Of Sale, see the help sections of the relevant planning module or open a ticket via the eSupport portal.

Applying method sets that are missing parameters

If one of the method sets contain ROS shape forecasting and is missing parameters, a pop-up message indicating not all parameters have been captured displays. In addition, the specific variant is listed in the error message.
