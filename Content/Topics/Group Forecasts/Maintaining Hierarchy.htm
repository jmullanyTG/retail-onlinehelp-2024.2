Maintaining hierarchies

Hierarchies are maintained through the Edit Hierarchy function located on the Add / Update Group Forecast Model window. This function is used to select the available levels for the model. These levels have targeted accuracy levels and a tolerance threshold value. Each level included is a forecasted series, based on the supplied filter.

Editing a hierarchy

  1. Navigate to the Execute tab, click Group Forecast and then Manage.

    The Forecast Group Model Management window displays.

  2. Select the model to change, and click Edit.

    The Add / Update Group Forecast Model window displays.

  3. Click the Edit Hierarchy button to display the Maintain Hierarchy window.

  4. From the Available Levels dropdown menu, select the available levels as follows:

    • All Aggregated Together: establishes a single rooted node. If included, this level must always be the highest level of the group.
    • Inventory Level: the item/site combination.
    • Site Level: the lowest level of site hierarchy.
      • Site Region: optional named site hierarchy level.
      • Site Division: optional named site hierarchy level.
    • Item Level: the lowest level of product hierarchy (independent of site).
    • Site Groups: configured site groups. Can be hierarchical, if desired.
    • Item Groups: configured inventory groups. These are distinctly different from item master groups.

  5. Click Ok.

    The available levels display.

  6. Enter the individual target percentages and target threshold tolerances as follows:

    • Description: the name of the level.
    • Period Over Period Target Percentage: used to grade the forecast based on MAPE. In the report, any value over the period-over-period target displays in green, indicating an acceptable forecast.
    • Period Over Period Target Threshold Percentage: the tolerance for the grading system as relates to the period-over-period target (e.g. a 5% tolerance on a target value of 70% must be at least 65% in MAPE accuracy). Any value between the period-over-period target and the low end of the threshold displays in yellow, indicating a mildly acceptable forecast. Any value below the threshold will appear red, indicating a poor forecast.
    • Year Over Year Target Percentage: the year-over-year target aggregates the values and is expected to have a much higher tolerance percentage (as the forecast is expected to be more accurate, but less precise).
    • Year Over Year Target Threshold Percentage: the tolerance for the grading system as relates to the year-over-year target (e.g. a 5% tolerance on a target value of 70% must be at least 65% in MAPE accuracy). Any value between the year-over-year target and the low end of the threshold displays in yellow, indicating a mildly acceptable forecast. Any value below the threshold displays in red, indicating a poor forecast.
    • Level Order: the order in which the levels are applied.
  7. Click Ok.
