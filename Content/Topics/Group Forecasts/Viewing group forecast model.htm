Viewing group forecast accuracy

Use the Group Forecast module to view forecast accuracy for each level in the hierarchy.

To view a group forecast model

  1. Click Manage.

    The Forecast Group Model Management window displays.

  2. Select the model to display.

  3. Click Ok.

    The Group Forecast model screen displays.

Group Forecast screen functionality

The Group Forecast model screen has the following options:

  • Forecast Model: Launches the Forecast Task window to select method sets and forecast methods. Depending on the size and complexity of the model, this action can take a considerable amount of time. For large models, the slice forecast functionality is the preferred method for re-forecasting.

  • Link: Isolates inventories in the model through a filter. The linked items are the overlap section of a Venn diagram between the model and the selected filter. This view is the inverse of the Inventory Apply Rules view.
  • Manual Push: Writes the group level forecast data to the live forecast tables. The values are written along with an accompanying forecast type.

Viewing settings

The View settings displays data for other series related to the currently selected hierarchy level.

  • Current Level – The highlighted level in the hierarchy navigation pane.
  • Children of Current Level – The hierarchically dependent levels.
  • Siblings of Current Level – The independent peered levels.
