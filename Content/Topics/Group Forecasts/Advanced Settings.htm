Maintaining advanced settings

Advanced setting are maintained through the Add / Update Group Forecast Model dialog. Select the columns to be displayed in the Group Forecast report.

Working with advanced settings

  1. Select a model and click Edit.

    The Add / Update Group Forecast Model dialog displays.

  2. Click the Advanced Settings button.

    The Maintain Advanced View Settings window displays.

  3. From the View Type dropdown list, select one of the following viewing period for the forecast series:

    • Rolling Start Period: displays the start period of the series to align with the current date (similar view to the standard Forecasts Module). As time passes, the forecasted view rolls forward with it.
    •  Offset Start Days: specifies the period ahead at which to view the forecast series. This is a useful setting for planners as they can set the offset days to several months ahead to align with their planning schedule.
    • Start Month Defined: locks in the start month for the series. If selected, the start month must be chosen.

  4. If Offset Start Days or Start Month Defined were selected:

    • Offset Start Days, then enter a numeric value for the offset number of days in the Offset Days.
    • Start Month Defined, then enter the start month in the Month field.

  5. In the Default Volume Metric field, select a measurement to calculate the appropriate volume or mass (described as weight) based on units and the supplied volumetric data (data feed).

    This is a performance heavy option as it requires calculations at every level of the hierarchy.

  6. Click Ok.

Maintain Advanced View Settings window layout functionality

Columns are added or removed using the Add or Delete buttons near the bottom of the screen. New columns appear as a new row at the bottom of the grid and can be reordered.

If the column is an expression column, the expression can be manually entered by clicking on the expression dropdown icon.
