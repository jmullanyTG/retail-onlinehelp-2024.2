Viewing current hierarchy level - Details

The Details section of the Current Hierarchy Level view graphically displays the forecast series, the accuracy dashboard, and the level statistical forecast information.

Chart

The Chart tab displays the series based on the columns included in the data section. Each column with non-null data displays as a series in a line graph.

Accuracy Dashboard

The Accuracy Dashboard view grades the statistical forecasts at level based on the parameters given during the Edit Hierarchy section when creating a model.

The default colors (red, yellow, green) highlight the forecast MAPE if they fall within the target % or within the threshold %.

Level Statistical Forecast Information

The Level Statistical Forecast Information view displays detailed information related to the forecast for each of the levels displayed.

  • Name – The name of the level.
  • Description – Description of the level.
  • Forecast Event Date – The date of the forecast run.
  • Statistical Forecast Accuracy – The mean absolute percentage error (MAPE) of the forecast.
  • Statistical Method Chosen – The winning base method of the forecast tournament (for more details on the forecast tournament, see the Forecasts Module).
  • Statistical Method Chosen Parameters – For forecasting base methods with several outputs such as slope, parsimony, and smoothing constants display, these display in this column.
