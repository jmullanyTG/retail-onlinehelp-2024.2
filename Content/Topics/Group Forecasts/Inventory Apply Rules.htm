Inventory Apply Rules

Inventory apply rules displays insight into the models that are currently or potentially affecting inventories.

Select a filter to see which models affect the inventories in that filter.

These rules are maintained through the Maintain Inventory Group Forecast Model Apply Rules window.

To display models that affect inventories

  1. In the Group Forecast window, click Manage.

    The Forecast Group Model Management window displays.

  2. Click the Inventory Apply Rules button.

  3. Select a filter in the Items dialog.

    The Maintain Inventory Group Forecast Model Apply Rules window displays.
