Viewing current hierarchy level – Data

The data section of the Current Hierarchy group forecast displays the columns selected in the Advanced Settings section when building the model. All visible columns display in the pivot grid. Data is broken down by period (based on bucket type).

The Current Hierarchy Level - Data grid displays the series values in a grid format for the currently selected hierarchy level (highlighted in the hierarchy navigation pane). The visible columns that have been selected during the model building process (advanced settings) display across the time periods.
