Attribute Management Overview

Attribute management has traditionally been restricted to defining core attributes in host systems and pushing these to the planning solution for use in reporting and other grouping functions.

The Attribute Management solution can be used to

  • define attributes throughout the system all the way down to a product/location level.

  • maintain new attribute families and storage of imported attribute families.

  • define groups of attributes using attribute sets which can be assigned to certain hierarchies of product.

Assignment of attribute values is a simple task that can be accomplished throughout the system.

The view contains two tabs, Attribute Management and Attribute Sets, that are interconnected through the assignment of attribute families to attribute sets.
