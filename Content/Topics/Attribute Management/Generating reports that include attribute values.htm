Generating reports that include attribute values

Certain reports are designed to allow attribute values to be included to provide for the rollup of data sliced by the attribute values. One such report is the Assortment Choice Count Report (Plan > Assortment Planning > Dashboard > Assortment Choice Count Report).

In the filter section of the report, select an attribute family to slice the data by. Also, select the specific attribute values that are relevant to the report.

The returned report shows the data sliced by the attribute values in question (for example, color). The counts of variants is sliced across regular, promotional and markdown counts although only regular counts are shown above.

Using attributes with filters

After linking attribute families and values to entities such as variants or item masters, filter the entities using the attribute families in question.

Creating filters or editing filters

Create new filters or edit existing filters in the Filters module (Management > Filters) .

An entity type must be selected to ensure that the appropriate attribute families can be returned and used in the filter.

All available attribute families associated with the selected entity will be available and multiple attribute families can be selected to further refine the attribute in line with filter creation criteria.

Users can now select single or multiple values related to the attribute family selected.
