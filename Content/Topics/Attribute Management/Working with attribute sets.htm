Working with attribute sets

Attribute sets define a list of attribute families that are applicable to the set. This function allows users to default a set of attribute families to a product hierarchy level so that each product in that hierarchy level is eligible to receive the attribute values in question or default certain attribute values if necessary.

These sets can be used to define attribute values that are only applicable to certain products in the appropriate hierarchy such as sleeve length which may only be applicable to shirts and not to shoe categories.

Creating attribute sets

To create an attribute set:

  1. Navigate to the Attribute Sets tab within the Attribute Management module.
  2. Select an entity type and a hierarchy level.
  3. In the Attribute Sets section, click New.
  4. Complete the Set Name and Entity Types fields as required.
  5. Click Save.

Specifying an attribute set for a hierarchy path

If a compatible hierarchy is selected which supports the entity types selected, users are presented with the hierarchy and can specify for each of the hierarchy paths an attribute set that provides the list of attribute families that are allowed for that hierarchy.

An attribute set at the higher level of the hierarchy defaults to all the levels below in the hierarchy.

In the above example, specifying the attribute set at the Mens level of the hierarchy means that all of the subsequent Accessories levels of the hierarchy inherit the default set for Mens. However, users could set a different attribute set for each level below in the hierarchy if necessary.

Parameter definitions

Entity Type – Which attribute sets are returned based on the attribute sets associated with the entity. Only attribute sets that are associated with the chosen entity are returned.

For information on a specific entity type, see Creating and maintaining attribute families.

Hierarchy – Users can select a single product hierarchy from the list available. This populates the bottom screen in the view with the appropriate product hierarchy to which an attribute set can be associated. Select the product hierarchy associated with the assortment plan to set defaults for assortment planning.
