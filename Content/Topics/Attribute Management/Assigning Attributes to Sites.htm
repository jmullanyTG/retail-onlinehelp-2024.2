Assigning attributes to sites

Product attributes are assigned to sites using the Site Attribute Management tab in Site Manager.

Site Manager is structured and functions very similarly to Variant Manager. Specify a site filter or select specific sites from a site filter to create a document in which to work with those sites.

Select sites once a site manager document is open. Details for selected sites appear in the bottom section of the screen.

Various aspects of sites can be worked with within a Variant Manager document by clicking the appropriate tab in the Site section. Click the Site Attribute Management tab to assign or edit site attributes for selected sites.

Both the upper and lower sections of the view can be toggled on and off by clicking the corresponding buttons in the upper right of the view.

Functionality

Buttons

Parameters - Shows or hides the Parameters area.

Execute - Updates the view according to the selected Site Hierarchy and the selected sites.

Parameters

Site Hierarchy: The site hierarchy for which to assign attributes.

As it is possible to create alternate site hierarchies which contain the same sites, it is also possible to have different attributes assigned to a site across those hierarchies. Ensure the correct site hierarchy is selected before proceeding.

Field descriptions

  • Family Name: The name of the attribute family.
  • Family Description: The description for the attribute family.
  • Single Values: The attribute value of the adjacent single-select attribute family for the selected sites. This field shows (Multi) to indicate when sites are selected that have different attribute values assigned.
  • Multiple Values: The attribute values of the adjacent multi-select attribute family for the selected sites. When multiple sites are selected that have different attribute values assigned, this field shows the combination of all values present.

Data population

The view populates with all attribute families that any of the selected sites is eligible for. Attribute values available for selection include all values that any of the selected sites is eligible for.

Applying changes across multiple items

 When multiple sites are selected, any conflicts that exist between the applied values and the attribute value eligibility of the sites the change is being applied to are automatically resolved.

  • Single-select attribute value changes are applied to all sites that are eligible for them. Ineligible sites remain unchanged.
  • Multi-select attribute value changes are applied additively to sites that are eligible for them. If only some of the applied values are eligible, those are applied. Ineligible sites remain unchanged.
