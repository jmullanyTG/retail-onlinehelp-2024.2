Configuring non-stocked settings

ToolsGroup JustEnough does not make replenishment recommendations for non-stocked items, except to address the following cases, and only if the need cannot be met with current stock on hand:

  • Immediate demand
  • Customer orders
  • Safety stock unit requirements

To configure non-stocked settings

  1. From the Execute tab, click Policy. The Policy Task screen displays.

  2. Click Edit. The Update Policy task screen displays.

  3. Select Non-Stocked from the Order Policy Type dropdown menu on the Policy section.
  4. Update the fields as required.
  5. Click Ok.
Field Description
Back Order Ratio The percentage of lost sales that are expected from returned items from customers.
Minimum Safety Stock Units The minimum number of units maintained in stock in addition to the bin level (e.g. presentation minimums).
Return Rate The percentage of items sold which are expected to be returned.
Days Before Return The expected number of days between the sale of an item and the return of that item to the store.
Reuse Rate The percentage of items that have been returned and may be considered as saleable stock again.
Days Before Reuse The expected number of days between the return date of an item and it being considered as regular saleable stock again.
Order Multiple The multiple of units in which the stock must be ordered.
Order Multiple Rounding
Excess for Redistribution Days The number of days of excess that should be considered to be protected from a redistribution action. As a day value, the calculated unit value will be a resolved function of projected demand.
Expedite Freeze Days The forward looking days in which orders should not be considered eligible for an expedite action.
Surplus Freeze Days The forward looking days in which orders should not be considered eligible for a surplus action.
