Ordering Constraints Overview

Ordering constraints are useful practical tools which are incorporated into the Replenishment workflow. Ordering constraints created within this module can be applied to orders within Order Manager.

Constraint sets are organized by their name and description:

  • Constraint Set Name – Lists the user supplied name of the constraint set.
  • Description – A description of the constraint set.

Constraint sets

Constraint sets allow for certain tolerances and rounding rules to be applied to constraints. Constraint sets are managed in a single screen with creation and maintenance performed using a common wizard. The constraint sets use mathematical rules to help drive the Replenishment process.

Cost types

Ordering constraints are an important consideration for the replenishment engine and are used to build supplier orders.
