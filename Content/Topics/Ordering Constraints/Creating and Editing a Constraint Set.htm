Creating and editing constraint sets

Constraint sets are created using the Order Constraints module and can be either new or cloned from an existing constraint set.

To create a constraint set

  1. Click New.

    The Add/Modify Constraint Set window displays.

    Selecting Clone or Edit also opens the Add/Modify Constraint Set window.

  2. Enter the name and description for the constraint set.
  3. Optionally, select the Public Access checkbox if required.
  4. If Public Access is not selected, configure permissions on the Access Permissions tab.
  5. Select one of the following radio buttons:
    • All items in the selection
    • Groups of items within the selection

  6. Optionally, click the Select Groups button to include or remove specific groups of items.

    The Constraint Groups window displays.

  7. Click Ok.

To add or edit an individual constraint

  1. Select a constraint in the Add/Modify Constraint Set window.

  2. Click Add Constraint or Modify Constraint as required.

    The Add/Modify Constraint window displays.

  3. Complete or edit fields as required.

    New constraints are then ordered within the constraint set.

    Special attention should be given to the rounding settings as they determine the conflict resolution when generating orders.

  4. Click Ok.

  5. Select the Edit Permissions tab to configure permissions for users and roles.

  6. Click Ok.
