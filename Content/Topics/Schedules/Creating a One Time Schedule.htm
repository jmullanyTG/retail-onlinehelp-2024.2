Creating a one-time schedule

A one-time schedule can be created to alter behavior for a single occurrence. This can be useful for exception management of a single date in an otherwise consistent schedule set.

To create a one-time schedule

  1. From the Maintenance tab, click Schedules.

    The Schedule Set screen displays.

  2. Select the schedule set to add to the new schedule, and click Add.

    A new schedule row is added under the selected schedule set.

  3. On the right side of the screen, enter the schedule name and description.
  4. From the Schedule Type dropdown menu, select One-time.
  5. From the Date dropdown menu in the One-time occurrence section, enter the date on which the schedule occurs.
  6. Click Save.
