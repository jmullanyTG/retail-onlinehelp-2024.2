Linking existing schedules

There is no need to create multiple versions of the same schedule for each schedule set. The Link Schedules feature allows for selecting existing schedules and including them in the schedule set. Unlinked schedules cannot be modified.

The appropriate user right permission is required to modify a linked schedule.

To link an existing schedule

  1. From the Maintenance tab, click Schedules.

    The Schedule Set screen displays.

  2. Select the schedule to link and click Link Schedules.

    The Link Schedules window displays.

  3. Use the arrows to move the schedules between the Excluded and Included panels.
  4. Click Ok.
