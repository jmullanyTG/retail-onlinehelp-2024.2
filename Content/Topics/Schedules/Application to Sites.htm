Application to sites

Schedules are applied to sites (stores and distribution centers) to help control ordering behavior. In Site Manager, there are two schedule types by site: a shipping schedule and a receiving schedule.

Schedule data is stored in the tbl_Site.Receiving table. Schedule sets are stored in StockReceiptAlamancID. Shipping schedule sets are stored in StockShipAlmanacID.

Applying default schedules to sites

Default schedules for sites can be applied in the Site Manager module. See Assigning Schedules to Sites. Refer to the Site Manager User Guide.

Overriding item at site schedules

Override item at site schedules are configured in the Inventory Manager module by clicking Choose and selecting a field under Inventory Schedules. Click within the field to select an option. See Selecting Items, Views, and Fields in Inventory Manager. Refer to the Inventory Manager User Guide.

These schedules override any default schedules set in the Site Manager.
