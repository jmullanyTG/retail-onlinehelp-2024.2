Deleting a schedule set

When a schedule set is deleted that is linked anywhere in the system to a site or supplier, the schedule set removes the link from that site or supplier.

To delete a schedule set

  1. From the Maintenance tab, click Schedules.

    The Schedule Set screen displays.

  2. Select the schedule to delete, and click Delete.

    A message displays that the current set is linked to a site or supplier.

  3. Click Yes to confirm the deletion.
