System policies related to Supply Chain configuration

System policy Description
Supply Chain Default Active Supply Chain ID The ID of the default supply chain
Supply Chain Garbage Collection Max Minutes Allowed for Processing The maximum number of minutes that garbage collection can be performed. Default value = 60.
Supply Chain Garbage Collection Max Number of Deleted Supply Chain to Process at a Time The max number of deleted supply chains to process at one time.

Default value = 100.
Supply Chain Garbage Collection Minutes to Wait Before Removal of Deleted Entries The max number of minutes ToolsGroup JustEnough waits after a supply chain is deleted to remove the entry plus related entries from the system.

Default value = 129600 (90 days).
Supply Chain Override With From Supplier Flag When set to True, the supply chain hierarchy is ignored for inventory that has a From Supplier Flag set to True. However, supply chain overrides are always used, if defined.
