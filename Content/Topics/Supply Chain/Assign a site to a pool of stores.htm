Assigning sites to pools of stores (redistribution supply chain)

Redistribution supply chains create site pools that indicate from where inventory can be redistributed when there are overages or shortages in stores or DCs.

To assign a site to a pool of stores

  1. From the Maintenance tab, click Supply Chain.

    The Open Supply Chain window displays.

  2. In the Open Supply Chain dialog box, select the Redistribution supply chain.

    The supply chain document displays.

    In the following, there is an Unassigned Sites list and a set of site pools. In this supply chain, a separate pool has been created for each site region. Each pool contains the stores from its region.

  3. Optionally, if a new pool is required click Pool > Add to create a new Pool.

  4. Drag unassigned sites from the Unassigned Sites list and drop them in the desired pool.

  5. Click on any site in a pool to view the pool details including lead time matrix for sites in the pool.

    Sites inherit the pool's default lead time value of 2 days.

  6. Optionally, change the lead time in the grid by clicking the LeadTime value for any site combination.

  7. If a store must get inventory before any other site in the pool, set the priority in the store’s row in the matrix by entering 1 in the Priority column.

  8. Click Save.
