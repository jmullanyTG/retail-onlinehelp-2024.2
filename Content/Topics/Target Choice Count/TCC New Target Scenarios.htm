Creating Target Scenarios

Use the Fields button to launch the Customization dialog and customize which columns display.

Permissions

The following permission is required to access and work with the Target Choice Count module:

  • Access Target Choice Count Module

Creating new targets

  1. Click New to launch the Create Target Scenario screen.

  2. Complete the fields in the left pane as required.

    For information on the fields, refer to .

  3. Click Save.

    The Edit Target Scenario screen displays.

  4. If desired, click New in the right pane to create a new Target Segment and complete the fields as required.

    For more information on the fields, refer to Target Segment field descriptions.

  5. Click Save.

  6. Click Execute.

Creating target segments

  1. Display the Target Creation screen by creating a new Target or selecting the Copy or Edit buttons on the toolbar.
  2. Select a currency in the Display Currency parameter field.

  3. Complete the fields as required.

  4. Click Save.
