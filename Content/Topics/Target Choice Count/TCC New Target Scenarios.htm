Creating Target Scenarios

Use the Fields button to launch the Customization dialog and customize which columns display.

Permissions

The following permission is required to access and work with the Target Choice Count module:

  • Access Target Choice Count Module

Creating new targets

  1. Click New to launch the Create Target Scenario screen.

  2. Complete the fields in the left pane as required.

    For information on the fields, refer toTarget scenario field descriptions.

  3. Click Save.

    The Edit Target Scenario screen displays.

  4. If desired, click New in the Segment Definitions section to create a new Target Segment and complete the fields as required.

    For information on the fields, refer to Configuring target segments.

  5. Click Save.

  6. Configure floor set updates as required.

    For information on the fields, refer to Creating floor set updates for varying time frames.

  7. Click Execute.
