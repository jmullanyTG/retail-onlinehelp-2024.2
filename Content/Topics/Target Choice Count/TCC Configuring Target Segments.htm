Configuring target segments

For the following metrics, refer to Toolbar functionality: Copying and editing Target Scenarios

  • Average Inclusion Threshold

  • Active Variant Threshold Factor

  • Active Variant Time On Offer Minimum

  • Median Period Total Threshold

  • Active Variant Total Minimum

Toolbar buttons

  • New: Creates a new target segment line in the Target Segment Definitions section.
  • Delete: Deletes selected target segments.
  • Fields: Launches the customization dialog.
  • Save: saves changes made to Target Segment Definitions parameters.
  • Cancel: reverts changes made to the Target Segment Definitions parameters.
