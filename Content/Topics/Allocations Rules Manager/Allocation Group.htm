Allocation groups

Allocation groups function as a container for allocation rules. The Allocation Groups tab displays the details for each allocation rule group using a dropdown menu. There are 3 allocation rule groups that are labeled 1, 2, and 3. Allocation rules are assigned to these groups in the Allocation Rules tab.

Creating or editing groups

Groups can be created or edited by clicking Add or Edit in the Allocation Group section of the toolbar and using the Add/Modify dialog.

Allocation rule group values are stored in tbl_ARM_AllocationRuleGroupValue.
