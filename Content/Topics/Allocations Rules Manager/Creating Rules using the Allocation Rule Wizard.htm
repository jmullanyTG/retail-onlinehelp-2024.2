Creating or editing allocation rules

Allocation rules are created or edited using the Allocation Rule Wizard.

Creating allocation rules

  1. Click Add in the Allocation Rule section of the toolbar.
  2. Name the rule, choose the type and complete other fields as required.

  3. Click Next.

  4. Include appropriate inventory, itemmaster (product hierarchy), or site groups.

  5. Click Next.

  6. Select the ellipsis (three dots) in the Included Level Items field.

  7. Choose items to be included or excluded.

  8. Click Finish.
  9. Click Finish.

Editing allocation rules

To edit an allocation rule, click Edit in the Allocation Rule section of the toolbar.
