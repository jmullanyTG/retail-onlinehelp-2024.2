Displaying metadata for a column

For any screens that have the Choose button, there is an option on the pop-up window called Column MetaData. Selecting Column MetaData displays a tooltip with any metadata for the column.

The tooltip is activated when hovering the mouse over the column name. If:

  • no information is on the filter metadata tables for the column then nothing is shown.
  • the column is a calculated column the pop-up window will show the expression using the friendly name of the columns involved.

The existing metadata tool was updated to allow users to populate their own metadata description and Override fields.
