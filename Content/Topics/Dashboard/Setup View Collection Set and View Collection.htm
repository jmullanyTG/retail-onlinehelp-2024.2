Setup view collection set and view collection

The dashboard must be set up in its own view collection set and view collection.

Adding the View Collection Set ID to the appropriate system policy

To tie the dashboard view to the actual dashboard, the View Collection Set ID must be entered into the Dash Board View Collection Set ID system policy.

The ViewCollectionID may be checked by searching tbl_VW_ViewCollection.
