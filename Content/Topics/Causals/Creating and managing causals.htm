Creating and managing causals

Use the Causal Manager module to:

  • create new causals, edit and clone existing causals, and delete causals.

    All links related to a causal are also deleted when the causal is deleted.

  • dock causal manager documents next to one another for a side-by-side comparison.
  • configure a variety of components within Causal Manager, such as column selections, time series, save layouts, and even bring in additional views.
  • work between Inventory Manager and Causal Manager, with the ability to add or remove multiple causals at a time and have the forecast viewer update in real time.

Selecting Causal as a filterable entity type

A Causal filter entity type is available. The Causal filter entity type allows for filtering based on a variety of causal-specific metadata in addition to other metadata such as item and site. These causal-specific filters can be used to create Causal Manager documents and views.

For more information, refer to Creating a new filter.

For more information on creating filters and the Causal filter entity type, refer to the Causal Manager User Guide.

Adding a new causal

  1. From the Execute tab, click Causal Manager.
  2. In the Causal Manager Document toolbar, click New.

  3. In the New dialog, complete the fields as required and click Ok.

    The bucket type selected can be daily, weekly, or monthly. However, it is important that the bucket type selected match the forecasts bucket type.

    The new causal displays as a separate tab.

  4. Optionally, click Choose to display additional columns.

  5. On the Causal Details tab, enter values that correspond to the promotion.

  6. Click Save.

Configuring the time series

On the Causal Details tab, click Config to display the Visible Bucket Period Limits dialog.

The time series automatically updates.

Cloning causals

To clone causals:

  1. On the Causal Details tab, click Clone.

    The New dialog displays.

  2. Optionally, enter a name for the new causal.

    The name is prefixed with Clone -.

  3. Click Ok.

    The clone opens in a new Causal Manager Document. The cloned causal name is editable.

The Clone button is disabled whenever changes are made. Click Save or Undo to enable the Clone button.

Comparing causals

To compare causals:

  1. Select a causal’s tab and right-click.
  2. Select New Vertical Tab Group to display the causals as a side-by-side comparison.

Deleting causals

To delete a causal, click Delete and then Save.

Click Save after clicking Delete to actually delete the causal.
