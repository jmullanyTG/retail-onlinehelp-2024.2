Managing cluster sets

Two recognized clustering methods are used: K-means and K-medoids. These methods provide a general understanding of clustering useful data sets on which to build and maintain entity clusters.

A cluster set is the base unit for a specific cluster analysis run through the clustering engine. A new cluster set must be created and maintained for each clustering method, entity, and filter type, or other parameter.

The first step to create a cluster set is setting up the logical clusters that are used throughout the application. These clusters are built out of a variety of entities (Sites/stores, suppliers, items, etc.)

To manage cluster sets

Create, update, and delete cluster sets using the Cluster Set Management window.

  1. Navigate to Maintenance > Cluster Manager.

  2. Click Manage.

    The Cluster Set Management window displays the available cluster sets.

  3. Add, edit, or delete a cluster set as required.
  4. Click Ok.
