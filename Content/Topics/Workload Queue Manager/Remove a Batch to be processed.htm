Removing batches to be processed

Batches can be removed from being processed. A super user with the appropriate rights and permissions to access the Workload Queue screen can remove a batch that has not started its execution.

To remove a batch

  1. From the Maintenance tab, click Workload Queue.

    The Active screen displays the active batches.

  2. Select a batch from the Active list.

  3. Click the Remove button.

    The batch is removed from the list.
