Modifying a batch priority

There may be a need to escalate or de-escalate a batch’s priority. A super user with permissions to change a batch’s priority can modify a batch that has not started its execution.

To modify a batch’s priority

  1. Navigate to the Workload Queue module.

  2. Select the batch to be re-prioritized.

  3. Click in the Priority Type field and select the new priority from the dropdown menu.

  4. Click Save.

    The batch's priority is changed.

The de-escalation of a batch's priority affects the batch's expiration date.
