Customizing workload queue priority thresholds

The ability to override the default priority thresholds for any given process type requires user right Allow Access to Workload Queue Configuration.

To customize a workload queue priority threshold

  1. From the Maintenance tab, click Workload Queue.

    The Active screen displays the active batches.

  2. In the Workload Queue toolbar, click Configuration.

  3. In the Workload Queue Configuration window, expand the process type.

  4. In the Override Min and Override Max columns, enter the desired minimum and maximum threshold values for each priority level.

  5. Click Save.
