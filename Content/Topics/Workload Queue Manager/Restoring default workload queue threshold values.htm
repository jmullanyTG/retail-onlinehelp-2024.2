Restoring default workload queue threshold values

The ability to reset customized thresholds to their default values for a given process type requires user right Allow Access to Workload Queue Configuration.

To restore the default workload queue threshold value

  1. From the Maintenance tab, click Workload Queue.

    The Active screen displays the active batches.

  2. In the Workload Queue toolbar, click Configuration.
  3. In the Workload Queue Configuration window, expand the process type.
  4. In the Reset column, click the red button to restore all priority thresholds for the selected process type to their default values.

  5. Click Save to save the changes.
