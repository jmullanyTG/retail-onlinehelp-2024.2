Slot set status state change

The Status State changing to Complete signifies that the current Event Workflow Status is completed for the selected Slot Set. An Event Workflow Status change is triggered when all Slot Sets in an Event have a status of Complete.

A Slot Set may automatically advance to the next Event Status, even if the Status State has not been marked as Complete, if the Event Workflow is configured to auto advance based on an expected completion date.

To mark a Slot Set’s Status as complete:

  1. Double click an Event in the List screen.
  2. Select the Slot Set to be updated from the Event navigation pane.
  3. In the Page Status field, choose Complete from the dropdown list.
  4. Click Save.

The Event Schedule updates to reflect the new Status State change for the Slot Set.
