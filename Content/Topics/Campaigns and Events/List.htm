List

The List view is the default view when accessing the Events module. This view is used to create and manage Campaigns and their related Events. There are multiple ways to filter the list of Campaigns and Events in this screen. Any of the methods listed below can be used to filter Campaigns and Events.

Method Description
Campaign Selection The simplest way to filter Events is to select a campaign from the Campaigns grid and only the associated Events display in the Events grid.
Show All Click this toolbar button to toggle between displaying all campaigns or only those that are associated with the Events in the Events filter.
Events Filter Click Filter to display the Events filter window. Select a filter or click on the black hammer icon to create a new or edit the current filter.
Filter rows for Campaigns The list of Campaigns is filtered by using the filter row below the Campaign Name field in the Campaigns grid. The filter refines the list beginning with the string entered in the filter.
Filter Rows for Events The list of Events can be further filtered by using the filter row below the Event Name field in the Events grid. The filter refines the list beginning with the string entered in the filter.
Filter Events by Campaign Name Select the down arrow at the bottom of the screen to select a Campaign. All associated Events display in the Events Grid.

By deselecting the check mark, all Events display regardless of which Campaign they belong to. Clicking on the X hides the results.
Edit Filter Opens a filter with the selected Campaign Name defaulted in the Filter Editor.
