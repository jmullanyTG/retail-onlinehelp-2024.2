List

The List view is the default view when accessing the Events module. This view is used to create and manage campaigns and their related events. There are multiple ways to filter the lists in this screen.

The simplest way to filter events is to select a campaign from the Campaigns grid and only the associated events display in the Events grid.

Filter Description
Show All Click this toolbar button to toggle between displaying all campaigns or only those that are associated with the events in the Events filter.
By Event Click Filter to display the Events filter window. Select a filter or click on the black hammer icon to create a new or edit the current filter.
By field value Within any label field, hover at the top right and click the filter icon that displays. Select values or filters as required.

Within the Events grid Within the field below the label field, begin typing to narrow down the displayed events.

By deselecting the check mark, all events display regardless of which campaign they belong to. Clicking on the X hides the results.
Edit Filter Select the Edit Filter button at the bottom right of the Events grid to open the Filter Editor dialog.
