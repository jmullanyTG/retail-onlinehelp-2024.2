Create a versioned content output file row

To add a row to the Configuration, click on New, fill in the required columns, and click on Save.

Field Description
Output Column An optional column containing the data for the desktop publishing software plugin in the appropriate format. This can be the same as the Source Column or different if the Source Column needs to be reformatted before it can be output to desktop publishing software. If Output Column is left blank, it is given the same name as Source Column. A new column is created in the database if one does not already exist.
Data Type (read-only) Displays the Output Column data type.
Export Select if the field should be included in the output file. Deselect this column if a field should no longer be included in the file.
Source Column Select the associated column where the necessary data is stored.
Source Type Select from the following source types. The selection determines what columns populate the Source Column.
  • System ( this option should not be selected after initial configuration)
  • Content
  • Price Output
  • Price Output Column
  • Repository
  • Promotion Attribute
  • Event Info
  • Promotion Info
Order An optional order number for this field in the file. Unless specified otherwise, the new field is appended to the file.
Friendly Name A unique user-friendly name that appears in the file header. This is required for the desktop publishing software plugin to work successfully.
Comments An optional description of the purpose of this field in the output file.
