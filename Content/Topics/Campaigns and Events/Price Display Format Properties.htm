Editing price display format properties

Price display format properties are edited in the Edit Price Display Format window.

  1. From the Influence tab, click Events.
  2. In the toolbar, click Configuration.
  3. In the Event Configuration window, click the Price Display Formats tab.
  4. In the Price Display Formats tab, select a strategy and click Edit.

Modifying properties

Click a container in the Edit Price Display Format window to view and modify the properties of a price display format. As Price display formats can include multiple containers, verify the correct one is selected before modifying any fields.

  • Click Undo and Redo on the toolbar as needed to undo or redo any unsaved changes.
  • Click Save on the toolbar to save any changes made.
Property Description
Name The name of the price display format is the same as its associated promotion strategy.
Units The unit of measure used (centimeters or inches).
Width The width of the selected container. The size of the box within item display formats should be taken into consideration when determining the appropriate dimensions of the price display format. The minimum width allowed is 6 DIUs (about 1/32 inch).
Height The height of the selected container. The size of the box within item display formats should be taken into consideration when determining the appropriate dimensions of the price display format. The minimum height allowed is 6 DIUs (about 1/32 inch).
Font The font used for text (including price fields) in the selected container.
Size The font size used for text (including price fields) in the selected container.
Color The font color used for text (including price fields) in the selected container.
Order Number (Optional) An order number can be set up for each container in a price display format. The order number determines the order in which price format containers appear in reports. Reports are generally configured by Professional Services and should not be modified without assistance from Professional Services or Support.
Price Field (Optional) To add a price field to a container, select the field from the Price Field list and click the down arrow. The selected price field is appended to the container. Edit the location of the price field in the Text Preview box.
Text Preview The text displayed in the selected container. Changes to text in this box appear automatically in the price display format preview.
