Images

There are three ways to add images to the Content Version:

Button Description
Add Existing Select an existing image. Select the checkbox next to images to be associated with the Content Version and click Ok.
Add Select to upload images stored locally or on a network.
New New images can be requested by clicking on Image Request New. The New Image Request window is a form used to initiate requests.

The three methods of adding images are not mutually exclusive and all three methods can be used within a Content Version.

Each new image added should be associated with a View Type so it can display in an Item Display Format. This association can be made directly in the Images tab grid by selecting the View Type dropdown list and selecting a value for each image. Each View Type can only be associated to one Image.
