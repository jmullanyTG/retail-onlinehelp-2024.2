Modifying a slot set template

To modify a slot set template, select it from the Slot Set Template list and click Edit. The Edit Slot Set Template window opens.

For descriptions of the individual template properties, see Slot Set Template Properties.

Action Description
Edit Properties The name, event type, size, default space and item format and other properties appear on the left side of the window.
Activate or deactive the template The Active checkbox in the property list indicates whether the slot set template is enabled for use in events.
Renumber slots Each slot in the slot set template is automatically assigned a number (displayed in the Slot Num property). Clicking the #button next to the Zoom slider toggles entering or exiting slot renumbering mode. Slots can be renumbered as desired.
Save changes Saves changes made to the slot set template.
Split a slot Splits the selected slot into multiple slots. Slots cannot be split into widths or heights smaller than 48 DIUs (about 1/2 inch).
Merge slots Merges selected slots.
Equal size Resizes selected slots into equal heights or widths across their combined area. Slots must be adjacent to each other and aligned vertically or horizontally.
Clear all Removes all slot configurations and resets the template to a single slot with default settings.
Undo and redo changes Undoes and redoes unsaved changes to the slot set template.
Refresh Refreshes the Edit Slot Set Template window with the last saved changes.
