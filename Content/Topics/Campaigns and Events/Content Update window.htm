Content update window

The Select Content to Update window displays when the Ok or Save buttons are clicked to save the updated Promotion Content.

Event copy and images can be pushed to the following:

  • Master Content – If the Promotion consists solely of a single Product or Product Group that exists in Content Manager, it is available to update with the Event Content.
  • Promotion Content – If selected, the Event Content changes update the original Promotion.
  • Event Content – Any other Events associated with the Promotion are listed. Updates can be selected to push to the other Events

Click Ok to save and update selected instances.
