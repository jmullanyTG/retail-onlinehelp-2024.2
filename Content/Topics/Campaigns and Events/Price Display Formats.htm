Price display formats

Price display formats determine how price fields appear in an item display format. A price display format exists for every promotion strategy defined in the Promotion module.

With appropriate permissions, price display formats are viewed and edited using the Price Display Formats tab in the Event Configuration window.

Previewing a price display format

The Strategy Name list displays defined promotion strategies. Select a strategy to preview its associated price display format. Use the Zoom slider as needed to magnify the preview.

Containers

A price display format contains one or more containers. Each container can include its own text and price field values, and its properties (size, font options, etc.) can be individually customized.

Modifying price display formats

To modify a price display format, select it from the Strategy Name list and click Edit. The Edit Price Display Format window opens.

For more information on updating the properties that display in the left pane, see Editing price display format properties.

Click in the Text Preview property box to add, delete and edit the text in a price display format. The preview updates automatically.

Toolbar functionality

Functionality Description
Save Saves all changes.
Split Splits the selected container into multiple containers. Containers cannot be split into widths or heights smaller than 6 DIUs (about 1/32 inch). See, Splitting objects.
Merge Merges selected containers. See, Merging objects.
Equal Size Resizes selected containers into equal heights or widths across their combined area. Containers must be adjacent to each other and aligned vertically or horizontally. See, Applying equal size to selected objects.
Clear all Removes all sizing, font attributes, text and price fields from the price display format.
Undo and redo changes These buttons undo and redo unsaved changes to the price display format.
Refresh Refreshes the price display format with the last saved changes.
