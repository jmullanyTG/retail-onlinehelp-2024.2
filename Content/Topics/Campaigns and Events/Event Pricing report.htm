The Event Pricing Report

The Event Pricing Report displays the price status for every Promotion in the Event for each Geography. Using this report, price conflicts and other issues can be identified easily and rectified at the appropriate Workflow Status.

This report shows:

  • if a Promotion within a Slot has multiple prices for a Geography
  • whether issues are caused by a gap because of a missing Site in the Promotion
  • multiple prices within a Geography as the Site is associated with Price Clusters that are not aligned with the Geography.

Once these issues are resolved the Geography should only show once for the Promotion with a report status of Ok.

Running a pricing report

  1. Select the Event Pricing Report tab.
  2. Click the Execute button.
