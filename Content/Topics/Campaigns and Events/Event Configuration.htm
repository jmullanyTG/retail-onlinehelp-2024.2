Event configuration

Configuration is required in terms of permissions, event types, slot set (e.g. event page) templates and geographies. The following areas assist with configuring items to specific needs.

To open the Event configuration window, click Events > Configuration.

Permissions, policies, and content configuration

Configuration requirement Module
User and role-based access to event-related features and functions Users module
Global policies related to events Policies module
Content fields and content versions Content module
Geography sets Site Clustering module

The Event Configuration window

Use the Event Configuration window to configure the following:

  • Spaces
  • Price display formats
  • Attribute display formats
  • Item display formats
  • Slot set templates
  • Cost types
  • Event types
  • Geographies and sites
