Configuring slot sets

Slot sets display as rows in a grid. For each Slot, the attributes display as columns.

Every Slot Set has a Position 0 Slot. This is where all unadvertised Promotions are located for the Slot Set. Position 0 does not have an associated Item Display Format, Space, and the Status is Unassigned.

Access to this screen depends on assigned rights.

Private Promotions

If a Promotion has not been shared with a user, then:

  • the Promotion name in Assigned Promotion displays as Private.
  • all fields except Item Display Format and Space are masked.
  • the ability to modify Item Display Format and Space is disabled.

Viewing or editing a Promotion in a Slot

Double-click on a row in the grid to display the View/Edit Promotion screen to view or edit the details of a Promotion in a particular Slot.

Previewing a Promotion

Selecting a slot in the grid refreshes the preview pane (located to the right of the Promotion pane). If the Promotion has been shared with a user (the Promotion’s privacy option is Public or Shared), the read-only Promotion preview dialog displays.

The Promotion preview pane is masked and displays the following graphic if a slot with an unshared Promotion is selected.

Selecting Geographies and available Promotions

Selecting a Slot in the grid refreshes the panes below the grid.

Geography pane

The Geography pane indicates which Geographies will be automatically assigned to the Promotion when it is added to the Slot. The Assign button is disabled for Promotions not shared with the logged-in user (the Promotion is private).

For a Promotion already assigned to a Slot, it is possible remove one or more Geographies by clicking Assign Geography.

Geographies can be moved from the Included pane to the Excluded pane and vice versa. Click Ok when finished.

Promotion pane

The Promotion pane indicates which Promotions are available to add to the Slot. A Promotion with a check mark indicates that it has been added to the Slot Set but not necessarily to the selected Slot. If a Promotion is assigned to the selected Slot, it is the only one displayed in the Promotions pane.

The available Promotions pane displays Promotions with Sales Dates that overlap the Event Sale Dates. There are several buttons available above the Promotions pane:

The available Promotions pane:

  • does not display Promotions that have been designated Private.

    Promotions that have not been shared with the logged-in user (the Promotion is private) do not display though they do display as Private Promotions in the Slot Set List view to denote the Slot has been populated.

  • displays Promotions designated as Public, shared with the logged-in user, or are Private to the logged-in user.

Depending on how system policy Promotion Double Click Action is configured, double clicking on an assigned Promotion in the grid loads the Promotion in either a window or in the Promotions module.

Promotions menu

The following functionality is available in the Promotions menu on the toolbar:

Functionality Description

The following functionality is disabled when a Promotion with a privacy option of Private is selected.

Remove All is only applied to Promotions the logged-in user has access to. Promotions that are private are not removed. For example, if all the Promotions are shared except for the Promotion in slot 6, then all the Promotions are removed except for the private Promotion in slot 6.

New Requires a Slot have one or more Geographies available and the user has the appropriate Workflow permissions and rights. Clicking New opens the Create New Promotion wizard. For more information, see Create a new promotion.
Remove Enabled if a Promotion is selected in the Slot Set. Removes only the selected Promotion from the Slot. Any other Promotions are still associated with the Slot. A different Promotion can be added to the newly open Geographies in the Slot.
Remove All Enabled if a Promotion is selected in the Slot Set. Removes all Promotions from the Slot. The Slot is open after this action.

Promotions that are private are not removed.

Unassign Enabled if a Promotion is selected in the Slot Set. Moves the Promotion from an assigned Slot to the Unassigned Position 0. The Promotion is still associated with the Slot Set.
View Enabled if a Promotion is selected in the Slot Set. Displays the Promotion in Promotion Manager where modifications can be made. Changes reflect in the Event after the Slot Set is refreshed.
Content For any Promotion in an Event, it is possible to modify the content and images associated with it using the Content Editor. For more information, see Content Editor.

Slot Set grid columns

Field Description
Slot Number (read-only) The position of the Slot in the Slot Set.
Assigned Promotion (read-only) A Promotion’s name displays if it has been added to the Slot.
Content Version (read-only) If a Promotion has been added to the Slot and more than one Content Version exists for it, each Content Version has its own row in the grid. The Content Version Name displays in this column.
Item Display Format (Optional) The Item Display Format to be used across all Promotions in the Slot. Clicking on this field displays a dropdown list populated with the available Item Display Formats. The available Item Display Formats are filtered to be within corporate compliance to ensure consistent messaging to customers via the Slot Set Template Configuration screen.

If an Item Display Format is assigned that is no longer applicable to the Slot Set Template the option remains until it has been substituted with a different Item Display Format. The invalid Item Display Format is no longer available once another Item Display Format has been selected and saved.
Space (Optional) The Space assignment for the Slot. This can be modified by selecting a new Space from the dropdown list. Changing the Space assignment can affect user permissions to the Slot.
Status (read-only) If a Promotion has been added to the Slot, the Status is Assigned. Otherwise, it is Open. It is possible for Slots to have these statuses if not all of the available Geographies are assigned a Promotion.
Notes (Optional) Instructions or additional information related to the treatment of the assigned Promotion. Read-only if the Slot is Open.
Start Date (read-only) The Promotion Start Date.
End Date (read-only) The Promotion End Date.

Promotion pane

The following table describes the functionality in the available Promotions pane:

Functionality Description
Add to Event Assigns the selected Promotion to the selected open Slot in the current Slot Set. This button may be restricted depending on permissions and the Slot’s Space assignment.
View Opens the selected Promotion in the Promotions solution.
Filter Limits which Promotions display in the pane. By default, all Promotions with dates that overlap with the Event Sale dates display. To limit Promotions, select from the dropdown list of existing Promotion filters. If an appropriate filter does not exist, click on the black hammer icon to create a new filter that better matches the list of Promotions to display in the pane. Once the appropriate filter is selected, click Ok.
Unassigned If the Slot Set contains any Promotions that are associated with the Slot Set but unassigned to a visible Slot, then the Promotions are unassigned. All unassigned Promotions belong to Slot 0. To view only unassigned Promotions, click Unassigned to filter the Promotions pane to display only these associated Promotions.
Bids Refreshes the pane and displays only Promotions that have a Bid for the current Event Slot Set.

The following Bid information displays for each Promotion to assist with planning the Event and each Slot Sets:

  • Bid Comments – Comments providing additional information to support decision making for the Bid.
  • Vendor Supported – If the Bid is supported by the vendor.
  • Bid Status – The Bid’s status.
  • Financial Forecast – The projected financial sales for the Bid.
  • Unit Forecast – The projected unit sales for the Bid.

Selecting Edit Bid opens the Edit Bid Details window.

It is possible to change the Bid Status by selecting one of the following options in the Bid Action dropdown list:

  • Accepted – The Bid has been accepted. The Promotion should be placed on the Slot Set if this action is taken.
  • Rejected – The Bid has been rejected. If desired, enter comments to explain why the Bid was rejected.
  • Revision Requested – The Bid requires further information before a decision can be made. Include an explanation of the revisions requested in the Comments field.

Enter comments to provide any further information on the Bid Action taken.

After making any necessary changes, click Ok to save the updated Bid.
Version Prompt The Version Prompt button can be toggled on and off. When activated, a prompt requires the selection of a Geography to associate with the Promotion when Add to Event is clicked. By selecting only a subset of Geographies for a Promotion, the excluded Geographies remain available to associate with a different Promotion and the Slot is not considered fully assigned. Version Prompt can be disabled at any time.
