Merging objects

Merging combines two adjacent objects into a single object. It is available to the following object types:

  • Price display format
  • Attribute display format
  • Item display format
  • Slot set templates
  • Event slot sets

Options are merged using the Merge menu on the toolbar or by right-clicking an object and choosing the option from the context menu.

Merge options

The Merge toolbar menu contains the following options:

  • Merge Up
  • Merge Down
  • Merge Left
  • Merge Right

Merging considerations

  • Merge options are only available when there is at least one another object adjacent (above, below, right, or left) to the selected object.

  • Only the merge options that apply to a selected object are enabled. For example, in the following screenshot, only the Merge Up option is available because the only other adjacent container is located above the selected object.

  • One object can be merged with multiple other objects if the other objects combine to share the same border width or height as the original object. (See example below.)

When merging, only the content in the originally-selected object is preserved (see examples below).

Merging examples

The following examples are shown using price display formats, but apply to the other supported objects as well.

Merging equally-sized objects

  1. Select an object with at least one other object adjacent to it. The selected object is outlined in blue.

    A purple line along the selected object's border indicates that merging in that direction is possible.

  2. Select Merge Right.

    The merge removes the content that was in the adjacent object. Only the selected object's contents and properties are preserved.

Merging multiple adjacent objects

In this example, an object is merged with two other objects. The merged objects share the same height as the original selected object.

  1. Select an object that has multiple objects along its entire top, bottom, or side.

    In this example, the selected object is adjacent to two horizontally-stacked objects that combine to be the same height as the selected object.

    The purple line along the selected object's left border indicates that merging in that direction is possible.

  2. Select Merge Up.

    The merge removes any content that was in the adjacent horizontally-stacked objects. Only the selected object's contents and properties are preserved.
