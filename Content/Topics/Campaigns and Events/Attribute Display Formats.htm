Attribute display formats

Attribute display formats are viewed and edited using the Attribute Display Formats tab in the Event Configuration window.

Previewing an attribute display format

Select a name from the Attribute Display Format Name list to preview its associated attribute display format. Use the Zoom slider as needed to magnify the preview.

Formats can contain one or more containers which display in the right pane. Each container can include its own text and attribute values, and its properties (size, font options, etc.) can be individually customized.

Attribute display formats are managed using the toolbar functionality.

Attribute display format properties

Attribute display format properties appear in the New Attribute Display Format and Edit Attribute Display Format windows.

The New Attribute Display Format window is the same as the Edit Attribute Display Format window, xcept it starts with a blank Name field and a single empty container.

Modifying properties

To view and modify the properties of an attribute display format, select a container in the New Attribute Display Format or Edit Attribute Display Format windows. As attribute display formats can include multiple containers, verify the correct one is selected before modifying any fields.

Property Description
Name The name of the attribute display format.
Units The unit of measure used (centimeters or inches).
Width The width of the selected container. The size of the box within item display formats should be taken into consideration when determining the appropriate dimensions of the attribute display format. The minimum width allowed is 6 DIUs (about 1/32 inch).
Height The height of the selected container. The size of the box within item display formats should be taken into consideration when determining the appropriate dimensions of the attribute display format. The minimum height allowed is 6 DIUs (about 1/32 inch).
Font The font used for text (including attribute fields) in the selected container.
Size The font size used for text (including attribute fields) in the selected container.
Color The font color used for text (including attribute fields) in the selected container.
Attribute (Optional) To add an attribute field to a container, select the field from the Attribute list and click the down arrow. The selected field will be appended to the container. Edit the location of the field in the Text Preview box.

Attribute fields are configured by Professional Services. Contact Professional Services or Support for assistance with these fields.

Text Preview Add, delete and edit the text in an attribute display format by clicking in the Text Preview property box and making changes there. The preview updates automatically.

Functionality

Functionality Description
Save Saves changes made to the attribute display format.
Split Splits the selected container into multiple containers. Containers cannot be split into widths or heights smaller than 6 DIUs (about 1/32 inch). See, Splitting objects.
Merge Merges selected containers. See, Merging objects.
Equal Size Resizes selected containers into equal heights or widths across their combined area. Containers must be adjacent to each other and aligned vertically or horizontally. See, Applying equal size to selected objects.
Clear all Removes all sizing, font attributes, text and attribute fields from the attribute display format.
Undo and redo changes These buttons undo and redo unsaved changes to the attribute display format.
Refresh Refreshes the attribute display format with the last saved changes.
