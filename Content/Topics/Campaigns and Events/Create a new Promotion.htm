Create a new promotion

If there is no Promotion available that is appropriate for a particular Slot, a new Promotion can be created from both the List and Layout views. This can be done by navigating to an open Slot and clicking New in the Promotion section on the toolbar. This opens the Create New Promotion wizard.

Click Save and Close once all of the required information has been entered.

After the Wizard closes, users either return to the Event Slot Set or are directed to the Promotions solution Promotion Detail screen depending on the Global Policy Go to Promotion Module after Promotion creation.

If the Promotion Detail screen displays, further modifications to the Promotion can be made. If the Events solution displays, navigate to the Promotion in the Event and click on View to make changes. Content can be entered specifically for the Event by clicking on Content. For more information, See Content Editor.

Field Description
Promotion Name The Promotion Name.
Start Date (read-only) Defaults to the Event Ad Start Date or the Event Sale Start Date depending on the Global Policy setting Default new Promotion with Event date range.
End Date (read-only) Defaults to the Event Ad End Date or the Event Sale End Date depending on the Global Policy setting Default new Promotion with Event date range.
Workflow The default Workflow if desired for the Promotion.
Price Cluster Set Select the appropriate Site Cluster Set for the Promotion. The Site Cluster Set loaded to the Promotion is determined based on the Site Cluster Set start date and Promotion Start Date. The Promotion for each Site Cluster is customized in the Promotion Detail screen.
Promotion Strategy Optional. Select a default Promotion Strategy for the Promotion from the dropdown list. Populated with available Promotion Strategies defined in the Promotion Configuration Promotion Strategy area. When modifying the Promotion, the Promotion Strategy can be changed for one or more Site Clusters.
Private, Public, Shared The default privacy option is Public. The default can be changed to designate certain promotions as sensitive and restrict the ability for other users to view the promotion. Selecting Shared restricts viewing the promotion to certain users and roles.
Display Grid Fields Optional. The display grid on the left pane shows pricing fields that have been configured for the selected Promotion Strategy. If an editable field is available, input a Promotion level value to apply across all Site Clusters. This can be customized at the Site Cluster level in the Promotion Detail screen.
Products Optional. If a Promotion Strategy has been specified, products can be added to the Promotion. These Products apply to all Site Clusters in the Promotion. They can be customized per Site Cluster in the Promotion Detail screen. Depending on the type of strategy selected the manner in which products are added differs.
