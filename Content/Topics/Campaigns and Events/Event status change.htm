Changing an Event Status

The ability to manually advance an Event Status depends on the Event Workflow Transition Rights configuration.

To manually change the Event Status:

  1. Click on Change Status.

  2. Select the new Status from the dropdown list.

  3. Optionally, enter a message for the next person working on the Event.

    The Message icon turns green to indicate a message has been entered.

  4. Click Ok.

The Slot Sets inherit the new Status even if the Slot Set Status State is not Complete or is currently overridden. The Status State resets to In Progress for all Slot Sets.

The Event Workflow advances to the next Status and the Event Schedule reflects the Status change.
