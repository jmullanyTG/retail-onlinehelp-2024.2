Item display formats

Item Display Formats define how promotions appear in an event slot. Different item display formats can be set up for variations in size, prominence in the slot set, and display elements. For example, item display formats may be used to differentiate between a featured promotion versus a sub-feature promotion.

Item display formats can contain content fields, images, attribute display formats and price display formats.

Previewing an item display format

On the left side of the Item Display Formats tab is an Item Display Format Name list. Select a name to preview its associated item display format. Use the Zoom slider as needed to magnify the preview.

An Item Display Format contains one or more containers. Each container can include its own content, and its properties (size, font options, etc.) can be individually customized.

Deleting an Item Display Format

Specific Item Display Formats can be removed as corporate standards evolve. Removing or deleting an Item Display Format will not affect any current Event Slot Set Slots that are assigned it.

To delete an Item Display Format

  1. Navigate to the Event Manager home screen.
  2. Click the Configuration button on the right side of the screen.
  3. Select the Item Display Formats tab.
  4. In the Item Display Format Name list, select the Item Display format to be deleted.
  5. Click Delete.

  6. Click Yes.

    The Item Display Format Name list refreshes and the deleted Item Display Format no longer displays. Both active and inactive Event Slot Set Templates are updated.

Deleted Item Display Formats assigned to Event Slot Set Templates

Once an Item Display Format that is not assigned to an Event Slot Set Template's Slot is deleted, the system removes the association to any Event Slot Set Template so that it cannot be used in Events going forward. Existing use of the deleted Item Display Format on Event Slot Sets will not be affected until a new Item Display Format is selected, at which time it will no longer be available.

A dialog displays if deleting an Item Display Format that is assigned to an Event Slot Set Template.

Click on the dropdown arrow in the dialog’s Item Display Format field to select a new default Item Display Format for slots in the template. Click Ok to close the dialog and update the affected templates. The affected templates will be updated with the newly selected Item Display Formats.

Existing use of the deleted Item Display Format on Event Slot Sets will not be affected until a new Item Display Format is selected, at which time it will no longer be available.

Toolbar buttons

Button Description
New Creates a new item display format.
Clone Creates a new item display format by copying the selected item display format. The clone contains all the content and properties of the selected item display format. A new name can be provided as needed.
Edit Modifies the selected item display format. See Modifying an Item Display Format.
Delete Deletes the selected item display format.

Functionality

Functionality Description
Save Saves changes made to the item display format.
Split Splits the selected container into multiple containers. Containers cannot be split into widths or heights smaller than 6 DIUs (about 1/32 inch). See, Splitting objects.
Merge Merges selected containers. See, Merging objects.
Equal Size Resizes selected containers into equal heights or widths across their combined area. Containers must be adjacent to each other and aligned vertically or horizontally. See, Applying equal size to selected objects.
Clear all Removes all sizing, font attributes, text and attribute fields from the attribute display format.
Undo and redo changes These buttons undo and redo unsaved changes to the attribute display format.
Refresh Refreshes the attribute display format with the last saved changes.
