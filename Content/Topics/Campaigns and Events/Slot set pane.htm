Slot set pane

If a Slot Set is selected in the Event Navigation Pane, the Slot Set Pane is displayed at the bottom.

Depending on permissions and the current Event Workflow Status, the following attributes can be modified in the Slot Set Pane:

Field Description
Name The name of the selected Slot Set can be modified to adhere to the corporate naming convention. This is part of the Slot Set header.
Template Select from the dropdown list of available Templates for the Event Type. This is part of the Slot Set header.

If the selected Slot Set contains at least one Promotion that has not been shared, then the Slot Set Template dropdown is disabled.
Status Each Slot Set in the Event automatically inherits the Status of the Event. The Slot Set Status State indicates whether the selected Slot Set is In Progress, i.e. currently working on this Status, or if it is Complete. To mark the selected Slot Set as complete, choose Complete from the dropdown list and click Save on the Event toolbar. For more information, see Slot Set Status Change.

User Rights are required to enable the dropdown.
Status Override In some situations (e.g. to provide additional permissions or to remove permissions), it may be necessary to change one or more Event Slot Set Status to a Workflow Status that is different from the rest of the Event.

Click Status Override to set the selected Slot Set to an Event Workflow Status that occurred previously. Click Ok and click on Event Save. For more information, see Slot Set Status Change.

User Rights are required for this button to be visible.
Description Enter the purpose of the Slot Set.
Compositions (read-only) Some Events may have more than one ordering of its Slot Sets, known as Compositions, to customize the Event for specific Geographies. The Compositions grid displays the position of the selected Slot Set within each Composition of the Event.
