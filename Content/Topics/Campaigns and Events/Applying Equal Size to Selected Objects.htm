Applying equal size to selected objects

Use the Equal Size toolbar button to resize multiple objects so that they are of equal size in height and width. Objects cannot be smaller than 6 DIUs (about 1/32 inch) in width or height.

  • Objects must be adjacent to each other along the same dimension (horizontal or vertical).
  • Multiple objects can be selected as long as each is adjacent to the original or succeeding objects.

The screenshots below are from the Edit Price Display Format window. The steps are the same for any container or slot.

  1. Use CTRL + left-click to select adjacent objects.

    Objects which do not qualify for equal sizing include:
    • objects along a different dimension (vertical or horizontal) than previously selected objects.
    • objects that do not span the entire border of a previously selected object.

  2. Click Equal Size to divide the selected containers into equal widths or heights.
