Remove a promotion

Removing Promotions from Slots does not delete the Promotion from the system. However, it does make the Slot, or some Geographies in the Slot, available to add a different Promotion. Promotions are removed from both the List and Layout views of a Slot Set by using the buttons near the top of the screen.

Toolbar funtionality

The following are enabled if a Promotion is selected in the Slot Set:

Button Description
Remove Removes the selected Promotion from the Slot. This only removes the selected Promotion. Any other Promotions are still associated with the Slot. A different Promotion can be added to the newly open Geographies in the Slot.
Remove All Removes all Promotions from the Slot. The Slot is open after this action.
Unassign Moves the Promotion from an assigned Slot to the Unassigned Position 0. The Promotion is still associated with the Slot Set.
