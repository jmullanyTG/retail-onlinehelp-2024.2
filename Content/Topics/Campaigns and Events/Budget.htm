Managing Cost with the Budget view

An Event’s Budget view provides information regarding the Event’s budget. Enter actual Event costs to track against the total budget allocated to the Event.

If default Event Costs  were defined in the Event Type configuration, those Costs display and are modifiable to allow for more accurate estimates.

Create a cost

Add a new cost by selecting New. This adds a new row in the grid which is modified as needed to reflect the new cost. Click Save on the Event toolbar to save the new cost information and to update the Total Event Costs and Available Event Budget fields.

Enter the following information for each cost:

Field Description
Cost Type Select the Cost Type.
Estimated Amount The projected monetary cost for the Cost Type.
Actual Amount The actual monetary cost for the Cost Type.
Cost Level (Optional) Select one of the following from the dropdown list:
  • Event
  • Promotion
  • Slot Set
Applied To (Optional) This dropdown list refreshes after the Cost Level is selected. The options depend on the selection in Cost Level:
  • Event – The system defaults the current Event
  • Promotion – A list of all Promotions in the Event
  • Slot Set – A list of all Slot Sets in the Event
Supplier (Optional) A dropdown list suppliers.

Suppliers are maintained in the Suppliers module.

Invoice Number (Optional) The invoice number associated with the cost.
Comments (Optional) Any additional comments for the cost.

Modify a cost

To modify a cost, make changes directly in the grid as needed. Click Save to save the new cost information and to update the Total Event Costs and Available Event Budget fields.

Delete a cost

To delete a cost, select cost from the grid and select Delete and click Yes. This removes the row from the grid. Click Save to update the Total Event Costs and Available Event Budget fields.
