Renumbering slots

Slots in a slot set layout are automatically assigned slot numbers. Select a slot to see its assigned slot number in the read-only Slot Num property.

  • In the Edit Slot Set Template window, Slot Num is included in the property list. For more information, see Modifying a slot set template.

  • In the Layout view's Layout Edit Mode, Slot Num is a property in the Slot Information pane. For more information, see Editing slot set layouts.

To renumber a slot

  1. Above the slot set preview, click the # button next to the Zoom slider to turn on the slot numbering feature.

    Numbers appear over each slot and the slot properties are grayed out.

  2. Click on the slot to be slot number 1.

    The rest of the slots renumber themselves accordingly.

  3. Clicking on additional slots renumbers them in sequence (slot number 2, slot number 3, etc.).

    Click Undo or Redo on the Slot toolbar as needed.

  4. Click Save on the Event toolbar to save changes.

  5. Click the # button again to turn off the numbering feature and return to normal functionality.
