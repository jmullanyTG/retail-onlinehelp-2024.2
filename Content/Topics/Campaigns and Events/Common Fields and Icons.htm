Event toolbar information fields and icons

There are fields and icons that are available on an Event's details screen regardless of what is selected in the Navigation pane on the left side of the screen.

Field Description
Event The name of the Event.
Campaign The name of the Campaign associated with the Event.
Ad Date Range The Ad Start and End Dates of the Event.
Sale Date Range The Sale Start and End Dates of the Event.
Event Status The current Workflow Status of the Event. This affects what actions can be performed on the Event. The Event Slot Sets inherit the current Event Workflow Status, unless a Slot Set has had its Status temporarily overridden to provide user permissions for a limited time period using the Slot Set Status Override button in the Slot Set Pane.
Event Schedule Click the calendar icon to display the entire Event Workflow schedule including Status projected dates and current Slot Set Status States.

Messages This icon is green if there’s a message available for the current Event Status.
