Event Calendar view

The Event Calendar view displays a Gantt chart view of all Campaigns and Events. Campaign colors are selected when a Campaign is created, whereas the Event Type dictates Event colors. By double clicking a Campaign, a window will open with the Campaign Header for review or edit. Double clicking an Event, within the Calendar, will open a window with the Event Header for review or edit.

The Date Display Type dropdown list displays Events by their Ad Dates or their Sale Dates. The Time Scale + and – buttons can be used to see Campaigns and Events displayed by time of day, day, week, month, quarter or year.

Filtering campaigns and events

Which Campaigns display can be adjusted.

Option Description
Show All Click to toggle between displaying all Campaigns or only Campaigns that are assigned to the Events in the Events Filter.
Filter Click Filter to display the Events filter window. Select an existing filter or click on the black hammer icon to create a custom filter.

Changing the grid view

Right-click in the header and select Change View To to choose one the following:

  • Month View – Displays one Campaign and up to three of its Events at a time in a monthly view. The vertical scroll bar changes the month and the horizontal scroll bar changes the Campaign.

  • Gantt View – Default for Event Calendar.

  • Agenda View – Display Campaigns and Events in a daily view
