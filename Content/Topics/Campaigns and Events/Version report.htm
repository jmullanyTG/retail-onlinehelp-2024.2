The Version Report

The Version Report displays distribution versions for the selected Event Slot Sets.

The versioning order is based on the order of the Geography Set used for the event. The first Geography in the set provides a constant for determining the Base Version for the Slot Set.  Further versions are grouped by their highest ranked Geography.

Bases

Versioning is a two-step process in which the Slot Set is first broken down into color / content versions (Bases), after which each Base Version is further broken down into additional black plate / price versions.  The result is a distinct set of content data by slot set and slot needed to create a complete Composition that is applicable to one or more Geographies.

Base Versions are named using alpha characters beginning with the letter A. Should the number of bases exceed 26, the base name becomes a 2-character string starting with AA and continuing as AB, AC, etc.  Plate / Price versions are named numerically (Preceded with the letter P) within their computed Base Version, and reset back to 1 for each base afterward.

To run a Version Report:

  1. Select the Version Report tab and ensure Parameters is selected.

  2. Using the Select Pages dropdown list, check the pages to include in the report.

  3. Select the Show Layout Details checkbox if needed.

  4. Click Execute to display the Report.

The report toolbar can be utilized to print, alter page setup, zoom in, out or to page, find details, and export the report in multiple formats.
