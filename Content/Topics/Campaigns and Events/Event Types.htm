Event types

Event Types provide the ability to group Events based on shared media type, geographies, and workflow. In some cases the Event Type includes a set of default Slot Set layouts and Slot display properties used to define the basic construction of like Events.

Creating event types

Click New to create a new Event Type, enter the required fields and click Ok.

Configuring event type default estimated costs

After the Event Type has been saved, click Cost Type to associate default Cost Types and default projected costs for each.

Using the arrow keys, select Cost Types to include for the Event Type. If known, enter the default estimated cost amount and a note for each Cost Type.

Modifying event types

Select an Event Type, click Edit, make all necessary changes, and then click Ok.

Deleting event types

Select an Event Type, click on Delete and click Yes. If the Event Type has been associated with an Event, an error message displays to prevent the deletion of the Event Type.

Field descriptions

Field Description
Event Type Name A descriptive label summarizing the purpose of the Event Type. This identifies the Event Type when creating an Event.
Media Type A Media Type for the Event Type.
Geography Set A default Geography Set for the Event Type.
Color The color to display Events of this Event Type in a calendar.
Workflow (Optional) A default Workflow for Events of this Event Type selected from the dropdown list of configured Event Workflows.
Description (Optional) A summary of the Event Type.
Composition Label A descriptive label for the Composition to be used when setting up Event Compositions by Geography.
Allow Multiple Compositions (Optional) Select the checkbox if the Event Type allows for more than one Composition per Event.
Slot Set Label A descriptive label for Slot Sets to be used when working in Events.
Geography Label A descriptive label for the geographies to be used when working in Events.
Event View Collection Set (Optional) A View Collection Set from the dropdown list to display in the third tab of the Event. Event Workbench is the standard View Collection Set provided for this tab.
Slot Set View Collection Set (Optional) A View Collection Set that displays in the bottom pane of the screen after clicking Details in an Event Slot Set.
Valid Slot Set Templates (Optional) Click Edit to select the slot set templates that can be used for Events of this Event Type. The selected templates appear as available options in the Slot Set Information pane when editing existing events and in the New Event wizard when reviewing the Event's default composition.

The Default checkbox indicates the default template used when adding new slot sets to Events of this Event Type.
Slot Set (Optional) The Slot Set grid provides configuration for the default Composition (ordered series of Slot Sets with Templates) for new Events of this Event Type. Using the Add and Remove buttons, set up the default Slot Sets including the Slot Set Template and the Slot Set order for Events.
