Campaigns

A Campaign is a coordinated series of Events (promotional vehicles) with a single idea or theme. Campaigns have a start and end date which must encompass the Events’ Ad and Sale Start and End Dates created within it.

Creating Campaigns

To create a new Campaign, click the New button to display the Create New Campaign window. Enter the required fields and click Ok.

The ability to create Campaigns depends on assigned rights.

Field Description
Campaign Name The Campaign name
Start Date The default start date and time. This can be set to the Campaign start date and time.
End Date The default end date and time. This can be set to the Campaign end date and time.
Description A optional summary of the Campaign.
Tags (Optional) Enter a new Tag. Existing Tags are selected by clicking on the ellipsis to open the Tags window and selecting the appropriate Tags.
Budget (Optional) The allocated budget for the Campaign.
Color The default color to help identify the Campaign in the Calendar view.
Owner Name The user creating the Campaign.

Modifying Campaigns

Campaign start and end dates must encompass all associated Event start and end sale dates.

  1. Select a Campaign.
  2. Click the Edit icon.
  3. Make the necessary changes.
  4. Click Ok.

Deleting campaigns

Campaigns can be deleted from the List or Calendar views.

  1. Select a Campaign.
  2. Click Delete.
  3. Select Yes to confirm.
  4. Click Ok.

If a Campaign has Events associated with it, then the Events are also deleted.
