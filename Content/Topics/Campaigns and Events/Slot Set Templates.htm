Slot set templates

Slot set templates define the default layouts for slot sets. Each slot set template is a grid of one or more slots, and each slot in the template can have its own:

  • size dimensions

  • space

  • item display format

    slot name

    slot number

For convenience, designate the event types for which a slot set template can be used and choose whether a template should be active (available for use) or inactive (not available to users).

Slot set templates are configured in the Slot Set Templates tab (Event Configuration). For more information, see Event Configuration.

Previewing a slot set template

On the left side of the Slot Set Templates tab is a Slot Set Template Name list. This list contains all of the user's slot set templates and indicates whether a template is active. Select a name in the list to preview that slot set template. Use the Zoom slider as needed to magnify the preview.

Toolbar functionality

Action Description
New Creates a new slot set template.
Clone Creates a new slot set template by copying the selected slot set template. The clone contains all the content and properties of the selected template. Give the new clone a name as needed.
Edit Modifies the selected slot set template.
Delete Deletes the selected slot set template.
