Add an existing promotion

Promotions can be added to an Event Slot Set from both the List and Layout views. To add an existing Promotion to a Slot Set, select a Slot in the Slot Set to refresh the Promotions pane. This pane is displayed below the Slot list grid in the List view and on the right in the Layout view. The Promotions pane displays the following information for all Promotions that overlap the Event Sale Dates.

Field Description
checkbox If checked, the Promotion has already been added to the current Slot Set
Promotion Name The Promotion name.
Promotion Description A description for the Promotion.
Start Date The start date.
End Date The start date.
Promotion Status Name The current Workflow Status.
Workflow Name The Workflow associated with the Promotion.
