Editing slot set layouts

The design of event slot layouts can only be edited using the Layout view. From here, customize slots, their positions, space assignment, and item display format assignment.

Certain actions may be restricted based on Events management user rights.

To edit the design of event slot slots, click the Layout Edit button (located about the slot set preview, next to the Zoom slider).

Once in Layout Edit mode, the toolbar functionality refreshes and new panes appear in the right side of the Layout view.

  • The Slot Information pane (top right) shows property details of the selected slot.

  • The Space Allocation pane (bottom right) shows space allocation by slot set and the overall event.

Promotions that are not shared with the logged-in user

Users cannot split or merge Promotions that are not shared with them. Promotions adjacent to an unshared Promotion cannot be merged to the private Promotion. For example, a public Promotion to the right of a private Promotion cannot be merged left and the purple line does not appear.

Promotions in the Slot Set that have not been shared with the user are indicated by the following:

Functionality

Functionality Description
Edit slot properties In the Slot Information pane, the following properties of a selected slot can be edited:
  • Slot name
  • Event space

  • Item display format

    This field displays a dropdown list with the available Item Display formats. The list is filtered to be within corporate compliance to ensure consistent messaging to customers via the Slot Set Template Configuration screen.

    If an Item Display Format is assigned that is no longer applicable to the Slot Set Template the option remains until it has been substituted with a different Item Display Format. The invalid Item Display Format is no longer available once another Item Display Format has been selected and saved.

  • Notes

A user can only modify the above properties if the slot contains a Promotion that has been shared with them. In not shared, Slot Name and Notes are masked and Event Space and Item Display Format are read-only.
Display headers

Toggles showing or hiding slot headers. Slot headers display the position and item display format for each slot.
Display dimensions

Toggles showing or hiding each slot's dimensions and relative size.
Drag promotions to different slots Dragging a promotion from one slot to another is possible within the same slot set as long as the promotion has at least one site cluster that matches the destination slot's geographies. Dragging removes the promotion from the original slot.

Drag and drop functionality for moving around slots is disabled if either the source or target slot has a private promotion assigned.
Renumber slots Clicking the # button next to the Zoom slider toggles entering and exiting slot renumbering mode.

This functionality is disabled if the slot contains a Promotion that has not been shared with the user.

Save changes Saves changes made to the slot set.
Split a slot Splits the selected slot into multiple slots. Slots cannot be split into widths or heights smaller than 48 DIUs (about 1/2 inch).

This functionality is disabled if the slot contains a Promotion that has not been shared with the user.

Merge slots Merges selected slots.
Equal size Resizes selected slots into equal heights or widths across their combined area. Slots must be adjacent to each other and aligned vertically or horizontally.

This functionality is disabled if at least one Promotion is selected on the Slot Set that has not been shared with the user.
Clear all Removes all slot configurations and resets the slot set to a single slot with default settings.

This functionality is disabled if the Slot Set contains at least one Promotion that has not been shared with the user.
Undo and redo changes The Undo and Redo toolbar buttons undo and redo unsaved changes to the slot set.
