Campaign and Events toolbar functionality

User Rights and Event Workflow permissions determine which buttons are available.

Functionality

Button Description
Save Saves any changes that have been made to the Event Information screen.
Clone Creates a copy of the current Event. All associated Promotions are duplicated for the new Event.
Edit Header Provides access to modify Event Header details including Campaign, Workflow, Ad Dates and Sale Dates.
Change Status Provides access to change the Event Status based on the associated Workflow. Access depends on the Event Workflow Transition Rights configuration. For more information, see Event Status Change.
Slot Set: New Appends a new Slot Set to the Event using the default Slot Set Template defined in the Event Global Policies. The Slot Set is also appended to the first Event Composition.

The label for this button changes depending on the Event Type’s configuration.
Slot Set: Delete Removes the selected Slot Set from the Event. All Promotions that were placed on the Slot Set are disassociated from the Event but not deleted.

The label for this button changes depending on the Event Type’s configuration.

The following functionality is disabled when a Promotion with a privacy option of Private is selected.

Remove All will only be applied to Promotions that the logged in user has access to.
New A Slot must have Geographies available and the logged-in user must have the appropriate Workflow permissions and rights. Clicking New opens the Create New Promotion wizard. For more information, see Create a New Promotion.
Remove (Enabled if a Promotion is selected in the Slot Set.) Removes only the selected Promotion from the Slot. Any other Promotions are still associated with the Slot. A different Promotion can be added to the newly open Geographies in the Slot.
Remove All (Enabled if a Promotion is selected in the Slot Set.) Removes all Promotions from the Slot. The Slot is open after this action.
Unassign (Enabled if a Promotion is selected in the Slot Set.) The Promotion from an assigned Slot to the Unassigned Position 0. The Promotion moves to Slot 0 and is still associated with the Slot Set.
View (Enabled if a Promotion is selected in the Slot Set.) Click on this button to view the Promotion in Promotion Manager. The Promotion can be modified with the appropriate rights. Changes reflect in the Event after the Slot Set is refreshed.
Content For any Promotion in an Event, it is possible to modify the content and associated images using the Content Editor. For more information, see Content Editor.

Event toolbar buttons

Action Description
List Displays Slot Sets in a grid format. For more information, see List.
Layout Displays the layout with images and formatted text. For more information, see Layout.
Versions View all versions of the Slot Set that may occur due to variations in Promotions, Content or Pricing for one or more Geographies. For more information, see Versions.
Views Displays a customized area at the bottom of the screen that is configured for the Event Type.
