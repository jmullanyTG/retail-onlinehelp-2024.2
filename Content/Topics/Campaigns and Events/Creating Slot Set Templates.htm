Create a slot set template

In the Slot Set Templates tab, click New to create a new slot template.

Complete the following fields as required:

Field Description
Name A descriptive label to summarize the purpose of the slot set template. This is used to identify the slot set template when setting up an event type or event slot set.
Units Modify the default unit of measure from the dropdown list.
Width Modify the width of the slot set template.
Height Modify the height of the slot set template.
Rows Select the number of rows to add to the slot set template. This value combined with the Columns value generates a grid of slots in the slot set template.
Columns Select the number of columns to add to the slot set template. This value combined with the Rows value generates a grid of slots in the slot set template.
Default Space Select the default space to assign all slots to in the new slot set template.
Item Display Format Select the default item display format for the slots in the new slot set template.

New slot set templates can be modified once created. See Modifying a slot set template.
