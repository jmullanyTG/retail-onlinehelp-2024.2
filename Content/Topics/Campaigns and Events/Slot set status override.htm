Slot set status override

In some situations such as the need to provide additional permissions or to remove permissions, it may be necessary to change a specific Slot Set to a previous Status, different from the rest of the Event Slot Sets. Select a previous Slot Set Status by clicking Status Override, choosing an option from the dropdown list and clicking Ok.

Once the modifications to the Slot Set have been completed, the Slot Set can be returned to its normal Status by selecting Complete & Advance from the Slot Set Status dropdown list and clicking Save. This pushes the Slot Set to the current Event Workflow Status.

If the Event Status is changed before the Status Override is complete, the Slot Set inherits the new Status and the Status State is reset to In Progress.

The Event Schedule updates to reflect that the Slot Set has a Status State of Pending (Override) for the current Status and a Status State of In Progress (Override) for the Status that the Slot Set was reverted back to.
