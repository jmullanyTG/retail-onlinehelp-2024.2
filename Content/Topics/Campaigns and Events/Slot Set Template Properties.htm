Slot set template properties

Slot Set Template properties appear in the Edit Slot Set Template window.

Accessing the Edit Slot Set Template window

  1. From the Influence tab, click Events.
  2. In the toolbar, click Configuration.
  3. In the Event Configuration window, click the Slot Set Templates tab.
  4. In the Slot Set Templates tab, select a template name and click Edit.

Modifying slot and slot set properties

The properties listed in the Edit Slot Set Template window apply to either the entire slot set template or to the specific slot selected in the preview.

  • Click Undo and Redo on the toolbar as needed to undo or redo any unsaved changes.
  • Click Save on the toolbar to save any changes made.

Template properties

Property Description
Name The name of the slot set template.
Active Whether the slot set template is active. Active templates are available for use in events. Deactivated templates are not. (This prevents users from applying templates without having to delete them.)
  • Clicking the checkbox toggles activating or deactivating the template.
  • By default, all new slot set templates are active.
  • An alternate event type must be selected when deactivating a slot set template that is in use.
Item Display Formats Multiple non-deleted Item Display Formats can be selected. By default, all Item Display Formats are available when creating a new Slot Set Template.

To remove Item Display Formats from a Slot Set Template, click on the dropdown arrow and uncheck the Item Display Formats to be removed. To re-include Item Display Formats, select the appropriate checkboxes.

After Ok is clicked, the list of Item Display Formats available for selection is updated.

At least one Item Display Format must be selected to be available.

If Item Display Formats are removed, that are currently being used on the Slot Set Template, users are prompted to select a new Item Display Format for each affected Slot.

Click on the dropdown arrow in the dialogâ€™s Item Display Format field to select a new default Item Display Format for slots in the template. Click Ok to close the dialog and update the Slot Set Template. Click Save on the toolbar to save any changes made.

Existing Slot Set Templates are not affected by the inclusion or removal of Item Display Formats in the currently displayed Slot Set Template.
Event Types The event types associated with the slot set template. Slot set templates are only available to events of the type selected here.
Units The unit of measure used (centimeters or inches).
Width The width of the entire template. The template width cannot be so narrow as to cause the size of slots in the template to shrink below 48 x 48 DIUs (about 1/2 inch x 1/2 inch).
Height The height of the entire template. The template height cannot be so small as to cause the size of slots in the template to shrink below 48 x 48 DIUs (about 1/2 inch x 1/2 inch).

Slot properties

Property Description
Slot Name The name of the selected slot.
Slot Num The slot number assigned to the selected slot. To renumber slots, see Renumbering slots.
Default Space The default space slots are assigned to in the template. For more information, see Spaces.
Item Display Format The default item display format for slots in the template. The Slot Item Display Format dropdown populates with selections made available in the template Item Display Formats field. For more information, see Item display formats.
Slot Width The width of the selected slot. The minimum width allowed is 48 DIUs (about 1/2 inch).
Slot Height The height of the selected slot. The minimum height allowed is 48 DIUs (about 1/2 inch).
% of Template ) The size of the selected slot as a percentage of the entire template.
