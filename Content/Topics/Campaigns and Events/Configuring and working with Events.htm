Configuring and working with Events

Events display as a separate tab immediately after creation or if opened from the List or Event Calendar views. Depending on assigned permissions configured for the Event’s current Workflow Status, users can review or modify the Event header information, Slot Sets, Geographies, Budget and Composition.

The screen consists of a navigation pane on the left and a workspace area on the right. The navigation pane displays Events and their Slot Sets in a tree style format. Selecting an Event refreshes the workspace area displaying the Geography, Budget and Workbench tabs along with several buttons.

Selecting Slot Sets

Event planning activities mostly occur at the Slot Set level. Selecting a Slot Set loads a pane below the navigation tree and refreshes the workspace to provide access to the Promotion menu (View, Content) and the following: List, Layout, Versions.
