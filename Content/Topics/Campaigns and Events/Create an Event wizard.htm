Creating Events

To create a new Event, click Event New to display the Create New Event Wizard window. Enter the required fields and click Ok.

Review and modify the default Slot Sets by clicking Next. Otherwise, click Finish to create the Event with the default Slot Sets for the Event Type.

To add a new Slot Set, click Add.

To remove a Slot Set, select the checkbox beside it, click Remove, then click Yes to confirm the delete.

To re-order Slot Sets, select a Slot Set and click on the up or down arrows to the right of the grid to change its position in the Event.

For each Slot Set, the following may be modified as needed:

Field Description
Slot Set Name Enter a name for the Slot Set that follows corporate naming standards.
Slot Set Template Select a Slot Set Template from the list of Templates associated with the Event Type.

Once the Event Slot Sets have been set up appropriately, click Finish. The Event Information screen refreshes.

Field descriptions

Field Description
Event Name The Event name.
Campaign If required, use the dropdown list to modify the associated Campaign.
Campaign Start Date The Campaign start date and time.
Campaign End Date The Campaign end date and time.
Event Type If required, modify the Event Type by selecting from the dropdown list for the Event.
Ad Start Date Modify the default date and time to match the starting date that the Event will be shown or distributed to customers.
Ad End Date Modify the default date and time to match the ending date that the Event will be shown or distributed to customers.
Sale Start Date Modify the default date and time to match the starting date that the Event’s Promotions will be available to customers.
Sale End Date Modify the default date and time to match the ending date that the Event’s Promotions will be available to customers.
Event Code Enter an Event Code if required by corporate standards.
Event Bid Status Select from the dropdown list to determine if the Event is open or closed for bids.
Bid Due Date Modify the default date and time to match the end date that the Promotion Bids are due for the Event.
Workflow Modify the default Event Workflow for the Event Type by selecting from the dropdown list, if needed.
Tags (Optional) With the appropriate rights, a new Tag can be entered this field. Existing Tags can be selected for the Event by clicking on the ellipsis to open the Tags window and selecting the appropriate Tags.
Budget The allocated budget for the Event. During Event planning, actual costs are aggregated and compared against this budget.
Owner Name The user creating the Event.
Description A summary of the Event.
