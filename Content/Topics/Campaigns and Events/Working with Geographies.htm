Working with Geographies

Creating Geographies

When a new Geography is created, it is possible to copy Event Promotions and Content from an existing Geography to the new one.

In the Geography tab:

  1. Click New.
  2. Select an existing Geography to copy Promotions and Content from.
  3. Click Ok.

  4. Click the Save button.

    The dates of the Geography default to the Event Ad dates but can be modified. Add or remove Sites to the Geography as needed.

Managing sites in Geographies

To add Sites:

  1. Select a Geography.
  2. Click Add in the Sites section.
  3. Optionally, check the Select only some items from the current filter checkbox and choose a sub-set of Sites.
  4. Click Ok.

To remove Sites:

  1. Select a Geography.
  2. Check the checkbox beside the Site to be removed.
  3. Click Remove.
  4. Select Yes to confirm the removal.

Modifing Geographies

Usually, modifying a Geography involves adding or removing Sites from it.

In addition to changing the Name, click within the date fields to display a calendar to alter the dates. Altering the dates adjusts the distribution of the Event to the Geography’s associated Sites to the new dates specified.

A Geography’s Start and End Dates must fall within the Event’s Ad Dates.

Deleting Geographies

Deleting a Geography implies that the Sites associated with the Geography are also removed from the Event unless they have been associated with another Geography in the Event.

To delete a Geography:

  1. Select the checkbox beside the geography.
  2. Click Delete.

    A confirmation message displays.

  3. Click Yes to confirm the deletion.

  4. Select Save.

    Associated Promotions remain in the Event if they are associated with other Event Geographies, otherwise they will be available to add to a Slot Set.
