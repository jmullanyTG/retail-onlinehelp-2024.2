Content editor

For any Promotion in an Event, it is possible to modify the Content and Images associated with it using the Content Editor. Clicking Content opens the Content Editor window for the selected Promotion.

This functionality is disabled if the Promotion was not created by or shared with the user.

In the left pane of the window, every Content Version associated with the current Slot Set is displayed. Selecting a version in the left pane loads the associated Text and Images tabs in the right pane. Each Content Version can be unique if desired.
