Cost types

Costs are associated with Event Types and projected costs can be specified during Event Type configuration. The Cost Types and projected costs are used in the Budget tab of an Event where the actual costs can be entered.

Creating cost types

Click New, enter the required fields, and click Ok.

Field Description
Cost Type Name A descriptive label summarizing the purpose of the Cost Type. This identifies the Cost Type when setting up Event Types or when entering actual costs in the Budget tab.
Description An optional summary of the Cost Type.

Modifying cost types

Updated Cost Type Names are reflected in Events referencing that Cost Type, when the Budget tab is refreshed or the next time the Event is loaded.

  1. Select a Cost Type.
  2. Click Edit.
  3. Make all necessary changes.
  4. Click Ok.

Deleting cost types

 If the Cost Type has been associated with an Event Type or Event, a message displays to prevent the deletion of the Cost Type.

  1. Select a Cost Type.
  2. Click Delete.
  3. Click Yes.
