Output

When an Event is to be published in a desktop publishing application, all relevant Event and Promotion fields must be prepared for the desktop publishing software plugin. Select the Output view to generate output or configure the output generated for an Event.

Users can continue working while the output is being generated. A notification is received once output generation begins and again when it is completed.

Generating output for page layout

To generate output:

  1. Select the Generate Output for Page Layout tab.

  2. Choose the Slot Sets to output.

  3. Select an Output Method from the dropdown list.

    The available output methods is dependent on Global Policy Allowed Versioned Content Output Methods.

  4. Click Save.

The output is produced in accordance with the associated Global Policies and Versioned Content Output Configuration.
