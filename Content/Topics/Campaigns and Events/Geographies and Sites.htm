Geographies and sites

The Geography view displays all the Geographies for the Event and their Site assignments when the Event Name is selected in the Event Navigation Pane. The Geographies and Sites assigned to the Event are defaulted by the Event Type configuration. If needed, the default can be modified for each specific Event.

The label of this tab is configurable by Event Type.

To view the Sites for a Geography, select it in the left pane and the Sites display in the right pane.
