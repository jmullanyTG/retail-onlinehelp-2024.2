Spaces

Spaces are set up to facilitate Event planning and can be used to restrict user access to Slots within a Slot Set of an Event.

Creating spaces

In the Spaces tab, click New to create a new Space, enter the required fields and click Ok.

Field Description
Space Name A descriptive label to summarize the purpose of the Space. This identifies the Space when setting up Slot Set Templates or modifying the Layout of an Event Slot Set.
Select Color Change the default color if desired. The color identifies the Space to users when working in an Event.
Assign Access Rights (Optional) By default, no users have access to add Promotions to the Space. To assign access rights, click Assign Access Rights.

Select roles or users that should have access to the Space and click Ok. Both roles and specific users can be assigned to the Space. Event Slot Set access is also restricted by Event Workflow Status and User Rights.

Modifying spaces

To modify a Space, select a space and click Edit. Click Ok once the necessary changes are made.

Events that are currently being planned may be affected if the modifications include restricting user access or which products can be promoted in the space.

Deleting spaces

To delete a Space:

  1. Select a Space.
  2. Click Delete.
  3. Click Yes.

An error message displays preventing the deletion of the Space if the Space is associated with an event slot or slot set template.
