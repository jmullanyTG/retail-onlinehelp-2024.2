Maintain filters dialog

The Maintain Filters dialog is the starting point for creating and maintaining filters.

Accessing the Maintain Filters dialog

From the Management tab, click Filters or click the Maintain () button to display the Maintain Filters dialog.

Working with filters

The following options are available:

Option Description
Creating a new filter Creates a new filter.

Select Clone From Existing Filter to create a copy of an existing filter.

Editing a filter Opens the filter selected in the Filter list for editing.
Delete filters Deletes the selected filters.
Export or import filters These options are used to move data from one instance of the application to another and are reserved for technical consultants only.
