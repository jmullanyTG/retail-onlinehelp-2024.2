Editing a filter

A filter’s details, criteria, and conditions are editable.

To edit a filter

  1. In the Maintain Filters dialog, select Update the Following Filter .

  2. Make the necessary changes, and click Ok.

Filter details

Field Description
Filter Name The name given to the filter.
Filter Note An optional notes section.
Filter Group Optional. Filter groups function in a similar fashion to a folder structure and are used to organize filters for ease of navigation.
Share This Filter Shares the filter with all or some users. (If sharing with other users, click the three dots ‘…’ to choose the set of users.)
Filter is Static Prevents the data displayed by this filter from updating after the filter has been created.

Select Rebuild Data on Save to update the data in this filter when saving changes to this filter.
