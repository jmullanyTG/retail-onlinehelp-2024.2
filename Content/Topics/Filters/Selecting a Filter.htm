Selecting a filter

The following is an example of the filter dialog box found across the various modules.

Opening the filter list to select an existing filter

  • Static data filters appear in orange text.
  • Static condition filters appear in black text.
  • Dynamic condition filters appear in blue text.

Filter functionality

Click the button to rebuild the filter using the most current data.

Only static filters can be refreshed.

Click the button to select dynamic conditions to apply to the filter.

The Conditions button only displays when a dynamic filter is selected.

Click the button to open the Maintain Filters window. To display a subset of the data included in the selected filter, select Select only some items from the current filter.

Selecting dynamic conditions

  1. From the Filters dropdown menu in the Items dialog box, select a filter with dynamic conditions.
  2. Click the button to select the condition to apply.
  3. In the Filter Condition dialog box, select a value and click Ok.
