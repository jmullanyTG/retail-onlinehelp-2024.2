Creating a variant site manager document

Variant Site Manager documents are created using the Variant Site Manager module.

Creating Variant Site Manager documents

  1. From the Management tab, click Variant Site Manager.

    The Items window displays.

  2. From the Filter dropdown menu, select the filter that contains the site variants to be managed, and click Ok.

    The Variant Site Manager Document displays.

  3. Click Choose to select additional fields to display.

  4. Select additional fields and click Ok.
