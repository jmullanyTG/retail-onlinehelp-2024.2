Metadata in Variant Site Manager

A great deal of metadata about sites is available in Site Manager. This data is added as columns in Variant Site Manager Documents and used in site filters to find sites which match specific criteria.

The following example shows how metadata in Site Manager can help users understand how sites are being used across ToolsGroup JustEnough.

On hand and on order filters

Allocators and demand planners may need to filter for available On Hand and On Order information to actively mitigate potential shortfall situations and better understand the total allocated stock between on-hands and orders.

Creating on hand and on order filters

  1. Type filter in the Search field.

  2. Click Filters.

    The Maintain Filters window displays.

  3. Select Create a New Filter and click Ok.

    The Select Filter Entity Type window displays.

  4. Using the Filter Entity Type dropdown menu, select Variant Site and click Ok.

    The Maintain Filter window displays.

  5. In the Criteria section, click New.

    The Setup Filter Criteria Form window displays.

  6. Select Criteria where all of the following conditions must be met and click Ok.

    The Filter Condition window displays.

  7. Using the Field dropdown menu, select Available On Hand > On Hand And On Orders Units.
  8. Define the condition in the Condition field.
  9. Enter a numeric value in the On Hand And On Orders Units field.

  10. Click Ok.

    The filters are created and can now be displayed as columns in the Variant Site Manager Document.

    For details, see Creating a Variant Site Manager Document.
