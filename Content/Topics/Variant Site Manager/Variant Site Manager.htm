Variant Site Manager Overview

The Variant Site Manager is used to view and manage data specific to each site variant, which is used extensively in the planning functions of ToolsGroup JustEnough.

Product attributes must be assigned to variants before working with site variants.

Accessing the Variant Site Manager screen

To access the Variant Site Manager screen:

  • Navigate to the Management tab and select Variant Site Manager.

For information on creating documents and choosing fields, see Creating a variant site manager document.
