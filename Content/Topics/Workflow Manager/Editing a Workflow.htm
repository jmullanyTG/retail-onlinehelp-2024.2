Editing workflows

  1. From the Maintenance tab, click Workflow.

    The Workflows list displays.

  2. From the Workflows list, select a workflow.

    For more information, see Workflow manager screen.

  3. Click Edit to display the Workflow window.

  4. Edit the details in the Workflow window as required.
  5. Click Save.

Field descriptions

Field Description
Name The workflow name that appears in the Workflows list.

For more information on the Workflows screen, see Workflow manager screen.
Solution (read-only) The ToolsGroup JustEnough solution the workflow will govern.
Active Whether the workflow is live in the system. Consider making workflows inactive while creating or editing them to avoid unintended behavior.
Controlled User Rights The set of module specific user rights to be controlled by this workflow. All user rights related to the selected module are controlled by default. The set can be customized by moving rights from the Workflow Controlled list to the Available list in the Workflow User Rights window.
