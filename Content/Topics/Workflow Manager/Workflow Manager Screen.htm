Workflow manager screen

The Workflow Manager screen contains three panes.

Workflows

The Workflows pane on the left side lets users select a workflow to view in the Design pane.

  • A blue workflow icon indicates that the workflow is currently active in the system.

  • A black workflow icon indicates that the workflow is currently deactivated or not used in the system.

Design

The Design pane in the center displays the diagram of the selected workflow. At the bottom of the pane are the following zoom options:

  • Fit to Page: zooms out to display the entire workflow page in the pane.
  • Fit to Width: zooms in or out to display the entire width of the workflow page in the pane.
  • Fit to Items: zooms in or out to fit all of the workflow items in the pane.
  • Zoom: manually zoom in or out.

Information

The Information pane on the right side displays information about the item selected in the Design pane.

Workflow manager screen functionality

Click Save on the Workflow toolbar after making changes to a workflow design, including adding, deleting, or editing statuses or transitions.

Tasks Procedure
Creating a workflow Click New on the Workflow toolbar to add a new workflow.
Editing workflows Click Edit on the Workflow toolbar to make the following changes to a selected workflow:
  • change the workflow name.
  • activate or deactivate the workflow.
  • change the user rights controlled by the workflow.
Activate or deactivate a workflow Click Edit on the Workflow toolbar and then click Active in the Workflow window to toggle activating or deactivating the workflow.
Save changes to a workflow Click Save on the Workflow toolbar to save any changes made to the workflow's design, status information, or transition information.
Delete a workflow To delete a workflow, select it in the Workflows list and click Delete on the Workflow toolbar.
Add a new status to the workflow Click New on the Status toolbar to add a new status box to the open workflow. See, Adding a status to a workflow.
Edit a status Select a status in the Design grid to edit its properties in the Status Information pane. See, Adding a status to a workflow.
Editing permissions for a status Select a status in the Design grid to edit its permission blacklist. See, Editing a status permission blacklist.
Delete a status Select the status in the Design grid and click Delete in the Status toolbar to remove the status from the workflow. Click Save on the Workflow toolbar to save the change to the workflow.
Add a transition between statuses Click Draw on the Transition toolbar to toggle draw mode in the Design pane. When in draw mode, click on the From status and drag the cursor to the To status.

For example, to create a transition from a Planning status to a Review status, click on the Planning status and then drag the cursor to the Review status. The transition arrow appears as the cursor is dragged from one connection point to another.

After the transition is created, edit its properties in the Transition Information pane. See, Adding a transition to a workflow.
Edit a transition Select the transition in the Design grid and edit its properties in the Transition Information pane.
Delete a transition Select a transition in the Design grid and click Delete in the Transition toolbar.
