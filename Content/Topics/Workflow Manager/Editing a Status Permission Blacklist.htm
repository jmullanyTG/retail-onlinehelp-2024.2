Editing a status permission blacklist

To edit a status permission blacklist:

  1. Navigate to the Workflow Manager screen.
  2. Select the workflow that contains the permissions to be edited.
  3. In the Design pane, select the status whose permission blacklist is to be edited.
  4. In the Status Permission Blacklist, select the permissions to be edited and click Edit.

    In this example, the Edit Price Action and Delete Price Action user rights are selected.

    For more information, see Status State Permission Blacklist.

  5. In the Remove User Rights dialog box, do the following:

    1. Select the user right and status state combinations for which permissions will be customized.

      In this example, only customize the edit and delete permissions for validated price actions.

    2. Select the roles and users from which rights will be removed.

      In this example, only users with Administrator or ToolsGroup JustEnough Advanced user roles will be able to edit or delete price actions that are in the Ready for Approval status and have been validated.

    3. Click Save. The Status Permission Blacklist is updated.

      In this example, the blue user icon indicates that the Allow Deleting of Price Cluster right is removed when a price action is in the Validated state during this status.

Alternatively, click on a user icon to view or edit permissions for that specific user right and status state combination.
