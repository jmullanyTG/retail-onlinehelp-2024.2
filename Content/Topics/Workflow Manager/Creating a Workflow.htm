Creating a workflow

  1. From the Maintenance tab, click Workflow.

    The Workflow manager displays.

  2. From the Workflow toolbar, New .

    The Workflow window displays.

  3. Enter the following details in the Workflow window:

    Field Description
    Name

    The workflow name that appears in the Workflows list.

    For more information on the Workflows screen, see Workflow manager screen.
    Solution

    The ToolsGroup JustEnough solution the workflow will govern.
    Active Whether the workflow is active in the system. Consider making workflows inactive while creating or editing them to avoid unintended behavior.
    Controlled User Rights

    The set of solution specific user rights to be controlled by this workflow. The selected rights appear in the status permission blacklist of each status in the workflow.

    All user rights related to the selected solution are selected by default. Customize this set by moving rights from the Workflow Controlled list to the Available list in the Workflow User Rights window.

  4. Click Save.

    The new workflow appears in the Workflows pane. The Design pane shows the default workflow diagram for the solution selected.
