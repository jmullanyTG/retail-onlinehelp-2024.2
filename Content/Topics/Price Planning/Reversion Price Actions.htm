Reversion price actions

Price reversions are applied from within Pricing Plans by navigating to the Price Reversion tab. Select a price reversion type and the effective date of the reversion for a series of Variant-Sites.

The grid displays the latest past markdown up to today's date along with all future planned markdowns.

Multiple reversions for the same Variant-Site

The same Variant-Site can’t have 2 or more reversions of the same Reversion Type and the same Start Date. To set multiple reversions for the same Variant-Site, use different reversion types, different start dates, or submit them separately.

Price Action logic separates price actions by Effective Date (PA Start Date), Channel, and Reversion Type ID (Price Method, Workflow, Source Type).

The Price Reversion tab is only visible if:
  • system policy Price Reversion Tab Visibility is set to True
  • the user has been assigned User Rights:
    • Add Price Action
    • Allow Access to Plan Price Reversion

Creating reversion price actions

Within a new or existing Pricing Plan:

  1. Click Select Items to display the Select Items screen displaying all Variants in the Pricing Plan.

  2. Select one or more Variants and click Ok.

    The Price Reversion tab refreshes displaying data for the selected Variants and Sites in the Pricing Plan.

  3. In the Reversion Type field, select a Reversion Type.
  4. Enter a start date in the Start Date field.
  5. Click Submit.

Price Actions are created using the entered start dates with strategy, workflow, source price type from the selected reversion type.

Configuring multiple grid rows simultaneously

  1. Select the multiple rows to be updated.
  2. In the Reversion Type or Start Date fields, right-click and select the Multiple Changes menu option.

  3. Enter a value in the New value field.

  4. Click Save.

Default and additional grid columns

In addition to columns that display data relating to Variants and Sites, the following Pricing columns display by default:

  • Previous Price Type - Based on Source Price Type in Reversion Type
  • Previous Price Date
  • Previous Price
  • Current Price Type
  • Current Price Date
  • Current Price

Click Choose on the toolbar to view Variant-Site metadata in addition to current and previous price data. Any date based data is evaluated using the current date.

Additional price types may be displayed as columns within the grid. These additional price types are defined using system policy Price Reversion UI Displayed Price Types.

Toolbar functionality

Functionality Description
Select Items Displays the Select Items screen displaying all Variants in the Pricing Plan.
Choose Launches the Choose Fields dialog in which users can select and include additional columns to provide more information.
Clear Clears the screen without saving any work in progress.
Submit Validates all modified rows have a valid Reversion Type assigned and a valid future Start Date.

Clicking Submit creates Reversion Price Actions using entered start dates with Strategy, Workflow, Source Price Type from the selected Reversion Type.
