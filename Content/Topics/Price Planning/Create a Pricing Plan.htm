Creating and editing pricing plans

Pricing Plans are created and managed in the Home tab of the Price Manager module. Additional products and sites can be configured at a later time.

Creating pricing plans

In the Home screen toolbar, click New to create a new pricing plan, enter the required fields and click Ok.

Editing pricing plans

In the Home screen toolbar, select a plan and click Edit in the toolbar.

Price Details section: fields and descriptions

Field Description
Plan Name The display name of the pricing plan
Start Date Select a future date that associated pricing actions default to. The start date is defaulted and selection of an earlier date is restricted, by a globally configured number of days forward from today. Special permissions such as Emergency Pricing Action allow for overriding this restriction and selecting tomorrow at the earliest.

When modifying the start date, the date forward cannot be any further than by any price action start date.

Original Plan Start Date = 01/01/2025
Earliest Price Action Start Date (already exported) = 02/01/2025
Result: A new Plan Start Date greater than 02/01/2025 is blocked.
Plan Description An optional description of the pricing plan.
Variant Filter Select a filter from the dropdown or click Maintain Filter to configure a new filter to designate the products to be used within the pricing plan's actions. This is the initial set of products to be added to the price action. Additional products can be added or removed at a later time.
Site Cluster Set Site cluster sets are available based on site clustering configuration.
Product Pricing Level A dropdown menu with the following options:
  • Variant Site Plan
  • Item Site Plan

This field only displays when system policy Product Pricing Level is set to Variant\Site and Item\Site Pricing.
Allow Price Matrix Sets the plan to allow price point conversion through a price matrix. If unchecked, then price actions use the system policy.
Allow Auto Price Sets the plan to allow auto pricing. When checked, users with the appropriate rights can view and edit the plan’s auto pricing parameters.

Auto Price Settings section

If Allow Auto Price is unchecked, then the Auto Price Settings section does not display.

Pricing plans can be configured to allow for the automatic creation of price actions using prices imported through standard import. For plans where the Allow Auto Price checkbox has been selected, price items in those plans can be scanned, identified, and priced automatically at regular intervals.

To configure a pricing plan to allow automatic pricing, select the Allow Auto Price checkbox when creating the plan. However, plans can be set at any time to allow or not allow auto pricing.

Editing auto pricing parameters

When the Allow Auto Price checkbox is selected, the Information tab displays an Auto Price Settings section for reviewing and editing the automatic pricing parameters.

Field Description
Strategy A dropdown list filtered to only include strategies configured with the system input variable.
Auto Price Filter A dropdown filtered by Variant-Site.

For information on configuration requirements for auto pricing, see Configuring auto pricing to automatically create price actions.
