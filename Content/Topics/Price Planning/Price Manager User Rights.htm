Price manager user rights

There are two sets of permissions within Price Planning. One set is not affected by a Price Action Status (i.e. global), the other is affected by status. Below is a list of global permissions and their purpose. Permissions are configured by clicking Users from the Maintenance tab and using the > and < buttons to add or remove rights for a user or role.

User Right Functionality
Access to Price Manager Config View the Price Manager configuration screen.
Access to Price Manager Access Price Manager by clicking Price Manager from the Plan tab and view Pricing Plans and Actions configured.
Allow Emergency Pricing Action Modify a Plan's start date and Pricing Action start date to occur tomorrow. By default, the plan's start date and associated Price Action start dates are restricted by the number of days, from today, configured in system policy Plan Start Date Number Of Days Out.
Add Price Plan Create (i.e. start date, add and remove products and sites) associated to a Pricing Plan.
Edit a Price Plan Edit (i.e. start date, add and remove products and sites) associated to a Pricing Plan.
Delete Price Plan Delete Pricing Plans. Pricing Plans cannot be deleted if there are associated Price Actions.
Add Price Action Create Price Actions which set pricing for site products.
Edit Price Action Edit Price Actions which set pricing for site products.
Delete Price Action Delete Price Actions which set pricing for site products.
Allow Override of Validation Failures Access to mark the override checkbox for a failed Price Validation in the Price Validation Log. Overriding a Price Validation error replaces the Validation failure severity icon with the Failure Reviewed icon to visually identify to users that a Validation failure exists but has already been reviewed and accepted.
Modify Price Action Status Access to the Change Status button for a Price Action.
Allow Access to Plan Auto Price Configuration Access to Plan Auto Price Configuration.
Allow Access to Plan Price Reversion Allows users to view and edit.
