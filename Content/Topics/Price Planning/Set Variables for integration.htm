Set variables for integration

Price Action variables can be configured to write back to other modules in ToolsGroup JustEnough. Based on the purpose of the variable, select the appropriate Price Type from the Export to Price Type column. The dropdown is populated with defined Price Types.

Upon export of a Price Action, this value is available for other modules to consume.

Price Actions with a Price Matrix assigned

For Price Actions with a Price Matrix assigned, selecting the Use Price Matrix checkbox sends the value to the Price Matrix for resolution.
