Default workflows

A default workflow can be specified for price actions that are created through the UI or other automated methods.

Default workflows can be based on:

  • Feature - The price action creation source. (Price Manager UI, Price Planner, New Store etc.)

    Selections available in the Feature field include:

    Feature Description
    Auto Price Auto Generated Price Action for Automated Pricing
    Price Reversion Auto Generated Price Action for Price Reversion
  • Channel - Based on the majority sites being passed into the price action.

Toolbar functionality

  • New - New row for configured data.
  • Delete - Deletes a configuration row.
  • Save - Saves changes. All values must be set.
  • Cancel - Clears unsaved data from the screen.

Grid fields

  • Feature - All existing Price Manager features.
  • Channel - All valid site channels.
  • Workflow - All valid Price Manager workflows.
