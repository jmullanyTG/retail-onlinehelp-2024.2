Create a price strategy

Within the Price Strategies tab, click New to create a new Price Strategy, enter the required fields, and click Save.

Restricting access to a new Price Strategy

By default, all Users and Roles have access to a new Price Strategy. Removing access to a new Price Strategy does not affect any current Pricing Actions.

The list is filtered to display only active Users and Roles that have the User Right Add Price Action.

To restrict access to a new price strategy:

  1. Click on the User Access field to display the Remove User Rights dialog.
  2. Select the Users and Roles that will be restricted from having access to the new Price Strategy.
  3. Click Save.

Price Strategies screen: fields and descriptions

Field Description
Price Strategy Name (Required) A descriptive label to summarize the purpose of the Price Strategy.
Price Strategy Description (Optional) A summary of the Price Strategy.
Active (Optional) Unmarking the checkbox removes the ability to configure the Calculation in any new Price Strategies.
Is Default (Optional) Designate a Price Strategy as the default when creating a new Pricing Action.
User Access Click Edit to open the Remove User Rights dialog to grant or restrict User and Role access to Pricing Strategies.
