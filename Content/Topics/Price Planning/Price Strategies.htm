Price strategies

Price strategies provide the ability to define calculations and validations that adjust pricing across a retailer’s omni-channel distribution points.

The rule grid (top right) defines the priority for the validations and calculations per corporate standards as well as defining severity levels of Validations.

The variable grid at the bottom lists all variables used in the Price Strategy and allows defaults to be entered (i.e. user input only). The Use Price Matrix column allows variables to be flagged for Price Matrix Translation and Rounding. The default is unchecked.
