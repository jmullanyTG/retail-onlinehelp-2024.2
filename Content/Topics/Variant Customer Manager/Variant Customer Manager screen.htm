Variant Customer Manager Overview

The Variant Customer Manager screen displays variants and the wholesale accounts which purchase them.

Accessing the Variant Customer Manager screen

To access the Variant Customer Manager screen:

  1. Navigate to the Management tab and select Variant Customer Manager.

  2. From the Filter dropdown menu in the Items dialog, select the filter that contains the variants to be managed.

  3. Click Ok.

    The Variant Customer Manager screen displays.

Creating a Variant Customer Manager document

  1. Click the Variant Customer Manager Document button to display the Items dialog.

  2. From the Filter dropdown menu, select the filter that contains the variants to be managed.

  3. Click Ok.

    The Variant Customer Manager Document displays.

  4. In the Columns toolbar, click Choose to select additional fields to display.

  5. Select fields and click Ok.
