Column settings

  • Column Friendly Name – An optional user readable name.
  • Column Order – The stacked numerical order for the columns in the widget. Any column without the Visible flag checked has an order of 0.
  • Format String – The format string settings affect how data displays. Format string entries vary depending on the data type. Examples include:
    • P2 – A percentage with two decimal places.
    • F3 – A fixed point decimal with three decimal places.
    • C – A currency.
    • D6 – A decimal with six decimal places.
  • Column Name Format String – A format string applied to the column name.
  • Horizontal Alignment – The horizontal alignment adjusts how the data is aligned within the cells.
  • Vertical Alignment – The vertical alignment adjusts how the data is aligned within the cells.
  • Allow DB Null – When adding a row, validates that the proper data has been supplied.
  • Default Value – If there is a default value which should be displayed in the column, this value should be listed here. When relying on the application or the user to provide the value, this field should be left blank (null).
  • Fixed Width – If selected, the column width is not editable by the user.
  • Allow Show Hide – If selected, determines whether the column can be hidden or restored by an end-user.
  • Allow Move – If selected, users can move and re-order the columns.
  • Allow Group – Can be used to prevent aggregation of a column. 
  • Write Enabled – If selected, users can write values to the cells within the columns.
  • Visible – If selected, the column is visible within the grid.
  • Allow Size – If selected, users can re-size the column.
  • Allow Filter – If selected, users can add a filter to the column.
  • Allow Sort – If selected, users can sort the column.
  • Parameter – Defines the parameter which is broadcast to the other widgets upon selection (highlight).
  • Show Caption – If selection, the column header displays.
  • Fore Color – The text color for the column.
  • Back Color – The background color for the column.
  • Allow Auto Filter – If selected, auto filter row is read-only for this column.
  • Allow Grid Group Summary – When using the pivot functionality to create groups, the subtotal count for the group displays. 
  • Allow Grid Totals – If selected, displays the count totals for the grid.
