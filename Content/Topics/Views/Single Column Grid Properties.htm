Single column pivot grid properties

  • Allow Column Moving – If selected, users can rearrange columns within the grid.
  • Allow Column Resizing – If selected, users can change the width of the columns.
  • Allow Filter – If selected, users can add a filter to the column.
  • Allow Group – If selected, determines whether users can drag the column's header to the group panel to group data against the column's values.
  • Allow Quick Hide Column – If selected, users can temporarily hide a column within the grid by dragging it to the data area.
  • Allow Row Sizing – If selected, users can change the height of the rows.
  • Allow Sort – If selected, users can sort the column data by ascending or descending order.
  • Multi-Select – If selected, users can highlight and select multiple rows within the grid (by using the shift key).
  • Show Auto Filter Row – Displays the auto filter row.
  • Show Group Panel – Displays the group panel if groups are allowed.
  • Show Horizontal Lines – Displays the horizontal grid lines.
  • Show Vertical Lines – Displays the vertical grid lines.
  • Enable Appearance Even Row – If enabled, adds a subtle color highlight to every other row (even numbered) within the grid making it easier to read. 
  • Enable Appearance Odd Row – If enabled, adds a subtle color highlight to every other row (odd numbered) within the grid making it easier to read. 
  • Allow Add Rows – If selected, lets users add rows to the widget.
  • Allow Delete Rows – If selected, lets users delete rows from the widget.
  • Allow Grid Group Summary – When using the pivot functionality to create groups, the subtotal count for the group displays.
  • Allow Grid Totals – If selected, displays the count totals for the grid.
