Aggregation settings

The aggregation settings establish and apply conditional aggregation rules for specific columns within the widget.

  • Order – If there are multiple ways by which the column can be aggregated, sets the order in which these aggregation methods are applied.
  • Enabled – Aggregation rules can be enabled or disabled on a case-by-case basis.
  • Conditional Check Field Name – The column which contains the condition that determines if the special aggregation type applies. This is a field in the data area.
  • Column Scope Field Name – Defines the column qualifier for the conditional aggregation.
  • Row Scope Field Name – Defines the row qualifier for the conditional aggregation.
  • Summary Type – The aggregation type which is applied.
  • Apply To Row Totals – If selected, the aggregation type is applied to the row totals.
  • Apply To Row Grand Totals – If selected, the aggregation type is applied to the row grand totals.
  • Apply To Column Totals – If selected, the aggregation type is applied to the column totals.
  • Apply To Column Grand Totals – If selected, the aggregation type is applied to the column grand totals.
