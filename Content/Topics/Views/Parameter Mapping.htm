Parameter mapping

The Parameter Mapping blade lets users build-in parameters to the URL, such as user name and passwords.

Browser parameters can be configured using code similar to this example:

IF EXISTS(SELECT TOP 1 * FROM sysobjects WHERE name='cusp_Browser_SelectURLParameters') DROP PROCEDURE cusp_Browser_SelectURLParameters GO   CREATE PROCEDURE cusp_Browser_SelectURLParameters ( @iUserID INTEGER = NULL, @iInputID INTEGER = NULL )   AS BEGIN   CREATE TABLE #URLParameters (Color NVARCHAR(10), Size NVARCHAR(5), Code INTEGER) IF @iInputID = 1 BEGIN INSERT INTO #URLParameters (Color, Size, Code) VALUES ('Red', 'L', 101) END ELSE IF @iInputID = 2 BEGIN INSERT INTO #URLParameters (Color, Size, Code) VALUES ('Green', 'XL', 102) END ELSE BEGIN INSERT INTO #URLParameters (Color, Size, Code) VALUES ('Blue', 'M', 103) END   SELECT * FROM #URLParameters END GO   EXEC usp_Common_AddUpdate_StoredProcedureLookup 'Browser_SelectURLParameters','cusp_Browser_SelectURLParameters','Select URL Parameters for Browser Widget' GO
