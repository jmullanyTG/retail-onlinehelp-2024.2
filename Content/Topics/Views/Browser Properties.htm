Browser properties

The Browser Properties blade determines how the browser behaves.

  • URL – For instances where the URL is hard coded and not intended to be returned as a parameter.
  • Allow URL Edit – Makes the URL field available to viewers so that they can edit it.
  • Allow Context Menu – If enabled, allows the right-click functions within the browser.
  • Web Browser Shortcuts Enabled – If selected, enables familiar shortcuts within the browser.
  • Script Errors Suppressed – Suppresses any script errors encountered by the browser.
  • Show URL Edit – If selected, the URL bar displays.
  • Show Back Button – Shows the back button.
  • Show Forward Button – Shows the forward button.
  • Show Refresh Button – Shows the refresh button.
  • Allow Navigation – Restricting Allow Navigation prevents users from navigating to another page after its initial page has been loaded within Internet Explorer.
  • Allow Web Browser Drop – If selected, lets users use the drag-and-drop features of some web sites which permit users to upload files by dropping them within the browser. window.
