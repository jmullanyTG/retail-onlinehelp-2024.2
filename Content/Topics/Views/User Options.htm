User options

  • Pivot Chart Display User Options – When using the pivot chart, populates the chart control options for the end user.
  • Pivot Chart Display User Chart Type – Allows users to edit the chart type. Certain chart types are more conducive to pivot data.
  • Pivot Chart Display User Show Legend – Allows users to toggle the legend on and off.
  • Pivot Chart Display User Show Point Labels – Allows users to toggle the point labels on and off.
  • Pivot Chart Display User Generate Series From Columns – Allows users to toggle the series generation options.
  • Pivot Chart Display User Graph Column Grand Totals – Allows users to toggle the column grand totals on and off.
  • Pivot Chart Display User Graph Column Totals – Allows users to toggle the column totals on and off.
  • Pivot Chart Display User Graph Row Grand Totals – Allows users to toggle the row grand totals on and off.
  • Pivot Chart Display User Graph Row Totals – Allows users to toggle the row totals on and off.
