Adding additional data to the Promotions module Planner tab

The Promotion Planner Custom View Collection Set view can be configured to display additional product information pertinent to promotion planning in the Planner tab of the Promotions module. The additional data can assist in determining what items to promote.

The view label is at the direction of the user.

The View Collection Set is linked to the Promotion Planner UI through system policy Promotion Planner View Collection Set. If the system policy is not configured with the View Collection Set ID, then the View Collection Set is not used and the custom tab will not display.

Garbage collection for obsolete Promotion Planner data

Three system policies (Module: PromoPlanner) can be used on a daily or weekly basis to remove obsolete Promotion Planner data. All three are optional. The system policies are:

  • Days elapsed following a Promotion assignment to the Plan, after when the Plan becomes Delete-eligible
  • Days elapsed following the Plan's end-date, after when the Plan becomes Delete-eligible
  • Days of no Plan modification activity, after when the Plan becomes Delete-eligible
