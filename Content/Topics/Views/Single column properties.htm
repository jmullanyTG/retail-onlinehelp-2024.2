Column properties

The column properties section lets users add, modify, and delete columns as needed within the grid.

Column

  • Column – The column name.

Band Settings

Band Type – Determines if the column is of type Pivot, the default band type will place the column in the center band.

If the column is of type Nonpivot, and does

  • not contain the JESUM expression, then the column is placed in the left band.

  • contain the JESUM expression, then the column is placed in the right band.

    Band types can also be manually set to determine the band placement.
