Selecting and editing views

The Navigation Menu pane on the left of the Views module displays all available views. Double click a view in the Navigation Menu pane to display the view in a separate tab in the right pane.

Multiple views can be displayed which open in individual tabs. The views have full functionality within this module.

Editing existing views

Views can be customized by clicking the Views button in the Configure toolbar. Views can be edited, imported, and exported using the View Management window.

To edit a view

  1. Select a view and click Edit.

    The Update View dialog displays.

  2. Revise the view as required.

    For more information on completing the Update View dialog, see Field Definitions.

  3. Click Ok.
