Creating and managing View Collection Sets

View collection sets act as containers for view collections and can be thought of as the workflow presented to the user upon clicking on a module name.

To properly configure a view collection set, the view collections intended for inclusion must exist. Once the necessary view collections have been created, they can be included and ordered within the view collection set.

Use the Maintain View Collection Set window to create and manage groups, view collection sets, and view collections.

In the context of view creation and maintenance, groups are an organizational tool intended to resemble a folder like structure. Groups should not be confused with other grouping systems such as inventory groups, item master groups, etc.

The Maintain View Collection Set window contains a toolbar and a grid showing the full tree structure of existing groups, view collection sets, and view collections.

Navigating to the Maintain View Collection Set window

To open the Maintain View Collection Set window:

  1. From the System Utilities tab, click Views.
  2. On the Views toolbar, click View Collection Sets.

Creating view collection sets

Creating view collection sets is similar to creating groups. View collection sets must be created within a group or any sub-group.

Click the New button in the Maintain View Collection Set toolbar or right-click on the desired group for the new view collection set. The New View Collection Set window displays.

The New View Collection Set window

  1. Enter a unique name for the view collection set and add a description.

  2. Confirm the parent group listed in the Group field (it can be changed by clicking Change and selecting a new group (folder)).
  3. Select the desired views in the Available Views field.
  4. Use the arrow buttons on the right to order the views.
  5. Click Ok.

View collection set maintenance

Click Edit to update the preferences of a specific view collection set. The most common tasks are the addition or subtraction of specific view collections and the reordering of the selected view collections.

Click Ok to save the new preferences.

It is recommended to click the Preview button on the Maintain View Collection Set window to confirm the layout is as desired.

Creating groups

Groups (folders) are created by clicking the New button in the Group section on the toolbar or by right-clicking on the desired location for the new group.

The New Group window displays.

Enter a unique name on that level for the group and add an optional description. Once a group has been created, it can be moved into another group or moved to the root level by dragging the folder icon and dropping it.

Toolbar functionality

The toolbar contains the following:

Action Description
Group
New Adds a new group.
Edit Edits the selected group.
Delete Deletes the selected group.
View Collection Set
New Creates a new view collection set. See,
Edit Edits the selected view collection set.
Delete Deletes the selected view collection set.
Import New Imports a new view collection set (.jevcs file).
Import Existing Imports an existing view collection set (.jevcs file).
Export Exports the selected view collection set to a .jevcs file.
View Collection
New Adds a new view collection. View collections can only be created within a view collection set. See, View collections.
Edit Edits the selected view collection.
Delete Deletes the selected view collection.
Remove Removes the selected view collection.
Preview Shows the current layout of the selected view collection set or view collection.
Refresh Refreshes the data in the grid.
