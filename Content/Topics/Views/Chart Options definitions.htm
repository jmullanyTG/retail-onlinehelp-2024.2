Chart options

  • Pivot Chart Enabled – If checked, the chart options are available for the pivot grid.
  • Pivot Chart Data Vertical – If selected, displays the pivot data vertically. 
  • Pivot Chart Selection Only – If enabled, the pivot chart only maps the highlighted data cells for the chart features (as opposed to charting every row and column).
  • Pivot Chart Show Column Grand Totals – If checked, displays the grand totals for columns which may be summed in the chart view.
  • Pivot Chart Show Column Totals – If checked, displays the totals for columns which may be summed in the chart view.
  • Pivot Chart Show Row Grand Totals – If checked, displays the grand totals for rows which may be summed in the chart view.
  • Pivot Chart Show Row Totals – If checked, displays the totals for rows which may be summed in the chart view.
  • Pivot Chart Vertical X Axis Labels – If checked, displays the X axis labels within the chart.
  • Pivot Chart Show Legend – If checked, shows the chart legend.
  • Pivot Chart Show Point Labels – If checked, displays the data related to the point within the chart.
  • Pivot Chart View Type – The type of chart (see chart widget for further details on chart types).

    Not all chart types may be available or practical for use with pivot data.
