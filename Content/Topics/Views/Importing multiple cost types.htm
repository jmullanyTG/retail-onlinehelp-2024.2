Defining and managing cost types

To import new Cost Types into ToolsGroup JustEnough, they must first be defined and managed using the Common Cost and Price Type Maintenance view.

Cost Type Maintenance

The following Cost Type Name and Active Flag is read-only:

  • Standard

Price Type Maintenance

The following Price Type Names and Active Flag are read-only:

  • Regular

  • Promo

  • Markdown

  • MSRP

Field notes

  • Cost Type and Price Type names must be unique within each grid.

  • Cost Type External ID is available in the Cost Type grid only and is for informational purposes.
