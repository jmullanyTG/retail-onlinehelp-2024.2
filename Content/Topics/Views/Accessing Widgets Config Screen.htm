Working with widgets

In addition to being created in the application and edited, widgets can be imported and exported from a Quality Assurance or development environment to a production or alternative testing environment, or from an older version of the application to a newer version (when compatible).

Widgets are stored as a .widgetf file type.

Application of widgets varies greatly depending on the widget type.

To access the Maintain Widgets window

  • From the System Utilities solution, click the Views tab, and then Widgets.

    The Maintain Widgets window displays.
