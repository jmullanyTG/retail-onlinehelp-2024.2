Analyzing irregular sales history

Items with anomalous spikes in sales history are reviewed using the Irregular Sales History Analysis screen. Products that are either over- performing based on recent history or under-performing based on recent history can be identified using filters in the Irregular Sales Analysis group. The filters are available at the Inventory level.

The filters are:

  • Has Irregular Sales
  • Has Underperforming Irregular Sales
  • Has Overperforming Irregular Sales.

Filter irregular items to review the forecasts and potentially apply causals.

Accessing the Irregular Sales History Analysis screen

This screen can be accessed by:

  • navigating to System Utilities > Views > Inventory > Irregular Sales History Analysis
  • adding the view to the Site Manager module. (For more information, see Site Manager Configuration.)

Setting the calculation parameters for irregular sales history

The calculation parameters for anomalous sales history as set using the Irregular Sales History Analysis screen. This screen can be accessed by navigating to System Utilities > Views > Inventory > Irregular Sales History Analysis.

Viewing the results of the calculation

The results of the calculation can be filtered in Inventory Manager using the columns in the Irregular Sales Analysis column group.

Field definitions

Field Definition
Filter All the inventories the system will run calculations over.
Time Frame To select a time frame, double-click within this field to display the Time Frames selector. Click the dropdown arrow to choose a time frame.
Sales Type The sales types to consider.
Lower Factor Factor * Standard Deviation below the mean that will be labeled irregular.
Upper Factor Factor * Standard Deviation above the mean that will be labeled irregular.
Create Date When the filter was created.
Create User Who created the filter.
Last Modified Date When the filter was last modified.
Last Modified User Who last modified the filter.
