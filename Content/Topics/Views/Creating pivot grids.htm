Creating pivot grids

To create pivot grids:

  1. Select the Views tab, and then Widgets.

    The Maintain Widgets window displays.

  2. Click New.

    The Choose Widget Type window displays the available widgets.

  3. Select Pivot Grid from the dropdown list in the Widget Type field.
  4. Click Ok.

  5. Complete each tab as required.

  6. Click Save.

Widget tab fields and definitions

General Tab: Displays header level information about the widget
Field Description
Name The unique name for the widget
Description The optional description for the widget
Read Only The widget can be set up as read-only to prevent manipulation of the presented data
Data Selection Tab: Determines how the information displays in the widget
SQL Connection Type Indicates from which database the widgets retrieve information:
  • Operations: the regular database that contains most of the current and operational data for ToolsGroup JustEnough. This is the default selection for most applications.
  • Warehouse: if set up, the warehouse database can function as a container for archived data.
Method Displays every SQL stored procedure in the database.
Method Parameters Once the stored procedure method is selected, the Method Parameters window displays the required parameters for the procedure. These parameters vary depending on the requirements of the stored procedure:
  • Method Parameter Name: the name of the required parameter as specified in the stored procedure.
  • Parameter: the selected parameter entry to be passed.
  • Maintain Parameters: displays the Select Parameter window with all displaying all available parameters
  • Default Value: the default value, if any, passed to the parameter.
Grid Properties Tab: Determines how the information displays in the grid
Grid Options Data Field Area: specifies the location of the data field. The default value is “none”.

Horizontal Scrolling: determines how the grid can be scrolled horizontally:

  • Cells Area: enables the scroll bar to allow for horizontal scrolling through the data cells only
  • Control: enables the scroll bar to allow for horizontally scrolling through the entire widget, including the control area.

Show Horizontal Line: similar to MS Excel, toggles the horizontal row lines on and off.

Show Vertical Lines: similar to MS Excel, toggles the vertical column lines on and off.

Totals Location: determines the location for totals columns:

  • Near: displays the totals column to left of the column field values.
  • Far: displays the totals column to left of the column field values.

Allow Compact View: when selected, compact layouts can be created for hierarchical pivot table row values, thus creating more space horizontally without sacrificing the distinction between the hierarchical row values.

Show Filter Drop Area: displays the drag-and-drop area to add fields to the filter area.

Show Data Drop Area: displays the drag-and-drop area to add fields to the data area.

Show Column Drop Area: displays the drag-and-drop area to add fields to the column area.

Show Row Drop Area: displays the drag-and-drop area to add fields to the row area.

Show Defer Rolling Actions: displays a button to prevent rolling.

Pivot Level Calculated Fields Use Summary Values For Calculations: if selected, the pivot grid uses the aggregated level values for the calculation. If not selected, calculations perform at the lowest level and aggregate using the aggregation type (e.g. SUM, AVG).

Editor Show Mode: specifies how an in-place editor is activated for a cell in the grid.

Enable Word Wrap: if enabled, the text wraps within the cell (mostly used for column names).

Enable Runtime Expression Edit: if enabled, displays the expression when there is a right-click on a row.
Column Total Options Show Column Grand Totals: displays the grand totals for columns that can be summed.

Show Column Totals: displays the total for the column if the column can be summed.

Column Totals Editable: if enabled, the column totals become editable.

Column Grand Totals Editable: if enabled, the column grand totals become editable.

Column Custom Totals Editable: if enabled, makes any custom column totals editable.
Column Total Options Show Row Grand Totals: displays the grand totals for rows that can be summed.

Show Row Totals: displays the total for the row if the column can be summed.

Row Totals Editable: if enabled, the row totals become editable.

Row Grand Totals Editable: if enabled, the row grand totals become editable.

Row Custom Totals Editable: if enabled, makes any custom row totals editable.
Update Options Confirmation Message on Update: if enabled, displays a confirmation message.

Confirmation Message: the text of the confirmation message. Defaults to Save Changes?.

Refresh Data on Update: enabled by default. Promptly updates a field when the field's information is changed.

Refresh Widget Siblings: must be enabled for dependent widgets in the view requiring data updates.
Chart Options Pivot Chart Enabled: if checked, the chart options are available for the pivot grid.

Pivot Chart Data Vertical: if selected, displays the pivot data vertically.

Pivot Chart Selection Only: if enabled, the pivot chart only has the highlighted data cells for the chart features (as opposed to charting every row and column).

Pivot Chart Show Column Grand Totals: if checked, displays the grand totals for columns that can be summed in the chart view.

Pivot Chart Show Column Totals: if checked, displays the totals for columns that can be summed in the chart view.

Pivot Chart Show Row Grand Totals: if checked, displays the grand totals for rows that can be summed in the chart view.

Pivot Chart Show Row Totals: if checked, displays the totals for rows that can be summed in the chart view.

Pivot Chart Vertical x Axis Labels: if checked, displays the X axis labels in the chart.

Pivot Chart Show Legend: if checked, displays the chart legend.

Pivot Chart Show Point Labels: if checked, displays the data related to the point in the chart.

Pivot Chart View Type: the type of chart (see chart widget for further details on chart types). Not all chart types may be available or practical for use with pivot data.
Row Total Options Show Row Grand Totals: displays the grand totals for rows that can be summed.

Show Row Totals: displays the total for the row if the column can be summed.

Row Totals Editable: if checked, the row totals can be changed.

Row Grand Totals Editable: if checked, the row grand totals can be changed.

Row Custom Totals Editable: if checked, turns any custom row totals editable.
