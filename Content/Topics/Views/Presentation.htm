Presentation

The Presentation section determines the details and additional features of the chart.

  • Chart – Allows users to affect the appearance of the chart, including background colors, fill styles, borders, and other settings.
  • Diagram – The diagram section allows users to manipulate the layout, rotation, scrolling, and zoom functions of the chart.
  • Panes – Panes will frame the chart. This feature may not be available for all chart types. 
  • Axes – The axes settings determine if and how the chart axes display. This feature may not be available for all chart types. 
  • Series Views – Series views allow users to manipulate how the individual series in the chart display and are treated. 
  • Point Labels – The Point Labels section affects how the data points are graphically represented within the chart. If it is intended for the values to be presented as part of the chart, users must determine how they would like to view that information. 
  • Chart Titles – Lets users add titles to charts. This may be unnecessary as the widget name typically displays at the header level of the container. 
  • Legend – If the chart requires or would benefit from the use of a legend, this can be added using the legend settings. 
  • Annotations – Any required annotations can be added using the annotations settings.
