OLAP pivot grid options

  • Data Field Area – Specifies the location, or lack thereof, of the data field. It is common to leave this as None.
  • Horizontal Scrolling – Determines how the grid may be scrolled horizontally.
    • Cells Area – Enables the scroll bar to allow for horizontal scrolling through the data cells only.
    • Control – Allows users to scroll horizontally through the entire widget, including the control area.
  • Show Horizontal Lines – Similar to MS Excel, toggles the horizontal row lines on and off.
  • Show Vertical Lines – Similar to MS Excel, toggles the vertical column lines on and off.
  • Totals Location – Determines the location for totals columns.
    • Near – Displays the totals column to left of the column field values.
    • Far – Displays the totals column to left of the column field values.
  • Allow Compact View – When checked, this value allows users to create compact layouts for hierarchical pivot table row values thus creating more space horizontally without sacrificing the distinction between the hierarchical row values.
  • Show Filter Drop Area – Displays the drag-and-drop area to add fields to the filter area.
  • Show Data Drop Area – Displays the drag-and-drop area to add fields to the data area.
  • Show Column Drop Area – Displays the drag-and-drop area to add fields to the column area.
  • Show Row Drop Area – Displays the drag-and-drop area to add fields to the row area.
