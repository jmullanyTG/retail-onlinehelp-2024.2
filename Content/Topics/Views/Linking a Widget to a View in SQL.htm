Linking a widget to a view using SQL

The following example was verified as valid and correct JULY 2022. DMDEV-82353.

While it is typical to link a widget to a view in the user interface, they can also be linked with a SQL script as follows (the example is for merchandise financial planning):

LINK VIEW AND WIDGET

IF @iViewID IS NOT NULL AND @iWidgetID IS NOT NULL AND NOT EXISTS ( SELECT 1 FROM dbo.tbl_VW_ViewWidget WHERE ViewID = @iViewID AND WidgetID = @iWidgetID ) BEGIN INSERT INTO dbo.tbl_VW_ViewWidget ( ViewID, WidgetID, WidgetOrder ) VALUES ( @iViewID, @iWidgetID, 0 ) END
