View collections

View collections, like view collection sets, exist as containers for a collection of views. These views display within the view collection as a series of horizontally ordered tabs. A view collection set is the highest level of view organization at which user rights and permissions may be assigned.

A view collection can be assigned a representative image from those available in the icon library. This icon represents the view collection throughout the user interface.

Within the view collection maintenance window, user access rights may be assigned to each individual view in the collection. Users may be granted read access, or write (full) access to the view. This access may be assigned based on roles or to individual users.

Creating view collections

Creating view collections is similar to creating groups and view collection sets. View collections must be created within a view collection set.

Click the New button in the View Collection section of the Maintain View Collection Set toolbar or right-click on the desired location for the new view collection.

The New View Collection window

  1. Enter a unique name for the view collection and add a description.

  2. Assign an icon in the Image field.

    Available icons display in the dropdown menu.

  3. Select the desired views in the Available Views field.

  4. Use the arrow buttons on the right to order the views.
  5. Assign user rights in the User Access Rights section.

Maintaining view collections

Click Edit on a view collection to update its preferences. The most common tasks are the addition or subtraction and reordering of specific views and the structuring of user access rights.

User access rights

User rights are listed by both role and by individual user. Any assignment of roles to specific user accounts must be done within the ToolsGroup JustEnough user maintenance screen by a system administrator.

Once roles have been defined, they may be assigned access to specific views within the view collection. The access rights displayed will represent the highlighted view in the above window.
