Update options

  • Refresh Data on Add – Refreshes the grid if data is added which may be returned.
  • Refresh Data on Update – Refreshes the grid if data is updated which may be returned.
  • Refresh Data on Delete – Refreshes the grid if data is deleted which may no longer be returned.
  • Confirmation Message on Successful Update – Displays a confirmation message upon successfully updating data.
  • Confirmation Message – The message which displays (default: Changes Saved).
  • Allow Confirmation Message Before Add Operation – If selected, displays a confirmation before data is added.
  • Confirmation Message Before Add Operation – The message which displays (Default: Allow Add Operation?).
  • Allow Confirmation Message Before Delete Operation – If selected, displays a confirmation before data is deleted.
  • Confirmation Message Before Delete Operation – The message which displays (Default: Allow Delete Operation?).
  • Refresh Widget Siblings – Must be checked if dependent widgets within the view require data updates.
