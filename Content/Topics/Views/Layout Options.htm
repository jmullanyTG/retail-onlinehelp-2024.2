Layout options

  • View Caption – When using the form view, the caption appears as a header to the form.

    This field is technically a format string and may use the features inherent in their creation to return desired data.

  • Show View Caption – Displays if selected and in cases where there is a view caption.
  • Card Caption Format String – The format string for the card caption.  Format string entries vary depending on the data type. Examples include:
    • P2 – A percentage with two decimal places.
    • F3 – A fixed point decimal with three decimal places.
    • C – A currency.
    • D6 – A decimal with six decimal places.
  • Show Card Caption – Displays if selected and in cases where there is a card caption.
  • Show Header Panel – Displays the header panel.
  • Show Expand Button – Displays the “expand button”.
  • Allow Hot Track Fields – When using the form view, adds additional highlight functions to mouse-over.
  • Allow Sort – Allows sort options for the entire grid. Overrides the column level settings.
  • Allow Filter – Allows filter options for the entire grid. Overrides the column level settings.
  • Single Card Stretch to Width – In layout mode with a single card, automatically stretches the width of the card to fit.
  • Single Card Stretch to Height – In layout mode with a single card, automatically stretches the height of the card to fit.
  • Multi Card Stretch to Width – In layout mode with multiple cards, automatically stretches the width of the cards to fit.
  • Multi Card Stretch to Height – In layout mode with multiple cards, automatically stretches the height of the cards to fit.
  • Layout View Mode – Determines how the layout displays if the view mode is set to Layout.
  • Use Advanced Runtime Customization – When using the form view, enables the customizable functions so uses can edit and fine tune the layout.
  • Show Save Load Layout Buttons – Displays the save and load layout buttons.
  • Allow Customization – Lets users customize the form at run time.
