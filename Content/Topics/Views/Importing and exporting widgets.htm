Importing and exporting widgets

Widgets can be imported and exported using the Maintain Widgets window.

Importing widgets

  1. Select the Import New button to open the Windows file manage.

  2. Navigate to the folder containing the .widgetf file to be imported.

    Once imported, the file is located in a folder labeled import followed by an 18 digit number string (this is the datetime in ticks) which can be used to link the widget to any simultaneously imported parameters.

The imported widget functions as designed if left in its import folder. However, users can move the widget to the proper folder for ease of navigation.

Exporting widgets

To export a widget:

  1. Highlight the widget in the Maintain Widgets window.
  2. Click the Export button.
  3. Navigate to where the widget is to be saved.

  4. Click Save.

    The exported widget saves as a .widgetf file type.
