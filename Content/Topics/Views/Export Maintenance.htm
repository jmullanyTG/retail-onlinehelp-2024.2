Configuring the export of promotions

The Export Maintenance view provides the ability to configure the Promotions export. The screen is grouped into two areas: Exportable Statuses and Export Configuration. Export Configuration consists of the definition of the export process and which columns should be exported for each process.

Exportable statuses

When an export will occur is determined using the Exportable Statuses grid. Only those Promotions and Events with a status that is considered exportable triggers an export. All statuses are dynamically displayed based on the statuses defined in the Workflow solution.

To identify a status as exportable, select the checkbox in the Exportable column of the status and click Save.

Export configuration

Exports are configured by specifying the process used to export data in the Export Definition and, if necessary, the associated columns to be exported in Export Columns. All exports defined in Export Definition are run when an export is triggered.

Create a new export configuration

For information on creating an export configuration, see:

  • Creating an Export Configuration for Promotion export

Modify an export configuration

Modifications can be made directly to the Export Definition and Export Columns grids. Click Save above the applicable grid to save any modifications. Click Cancel to discard any modifications.

To indicate an Export Definition is temporarily inactive, clear the Active Flag column and click Save.

Delete an export configuration

Deletions can be made directly to the Export Definition and Export Columns grids. Deleting a Export Definition row automatically deletes the associated columns in the Export Columns grid.
