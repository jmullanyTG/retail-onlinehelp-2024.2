Schedule tree data mapping

The schedule tree data mapping section determines how the data returned in the schedule tree data selection method is used.

  • Caption – The label for the entity in the hierarchy tree. The caption needs to be mapped to a field in the stored procedure.
  • Resource ID – Defines the link between the schedule tree and the schedule data.
  • Parent ID – If the schedule tree object is hierarchal in nature, this defines the hierarchal parented relationship.
  • Color – Defines the color of the schedule tree object.
  • Image – If there is an associated image (icon) within the method, this value goes here (index number).
