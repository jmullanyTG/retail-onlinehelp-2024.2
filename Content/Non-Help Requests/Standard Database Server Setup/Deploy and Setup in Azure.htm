Deploy and Setup in Azure

Create the Virtual Machine

Create an instance in Azure using ARM templates to desired specification for Cores and Memory based on sizing as determined in sales cycle

https://portal.azure.com/#create/Microsoft.Template

Configure Disk Volumes

Disk volumes are configured in ARM template

Configure Networking

Networking (static IP, subnet, etc) are configured in ARM template

For reference:

  • For Database systems, use “resource” subnet

  • For Web application systems, use “dmz” subnet

  • For Compute nodes, use “compute” subnet
