Custom Prescription Definition

Optionality

If the Inventory Solution is used out of the box, there will be no need for any custom prescription because there will be no custom Inventory artifacts to act upon.

Core Prescription Retrieval Query

It is strongly recommended that the following query be executed, the results studied, and the capabilities for reuse of ID Lists noted. The StepObjective clearly states what each step will do, and the artifacts concerned, with the sole exception of the Procedural Delete, where the tables involved are too numerous, typically, to enumerate.

SELECT * FROM inventory.PurgeStep ps ORDER BY ps.SequenceNumber;
